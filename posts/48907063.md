# 구식 넷북 에어컨 리눅스 깔기
     2009년도에 산 에이수스 Eee PC 1000HE는 인텔 애플매토姆 N280 프로세서를 장착하고 있고 램은 1GB밖에 안됨 ㅋㅋㅋ. 
     이 프로세서는 인텔이 저렴한 저성능 cpu로 나온거임. 1.667GHZ의 클락 속도를 가지고 있고 l1 캐시는 56KB, l2 캐시는 512KB를 가지고 있는데 비교대상인 인텔 코어 i3 N305는 768KB의 l1 캐시와 4MB의 l2 캐시를 가지고 있고 3.8GHZ의 클락 속도를 가지고 있어서 구식 넷북의 성능이 얼마나 구리냐면 ㅋㅋㅋ 예전에 썼을 때에도 노트북에 비하면 너무 약했음.
     그래서 이거 쓰다보니 2012년도쯤에는 거의 쓰기 힘들 정도가 되버림. 프로그램도 점점무거워지고 웹사이트도 점점 인터랙티브 해지고 그러니 구식 넷북에 하드디스크도 작고 그러니까 쓰기가 힘들었음. 
     결국에는 울트라북 같은 걸로 업그레이드 해야함.
     # 새운 운용 체제
     10년이 지난 2023년에突然 구식 넷북이 생각나서 찾아봤음. 
     그러니 보니 그대로 가지고 있던 창고에서 꺼내자마자 그때처럼 윈도우 XP UI가 나오고 윈도우 엑스플로러도 몇초 걸려서 mở고 그러니까 기억이 가물가물 하지만 항상这么 느렸는지는 모르겠음 ㅋㅋㅋ. 
     결국에는 윈도우 XP는 지원이 2014년에 끝났으니깔아야 할 운용 체제를 찾아야 했고 윈도우 윈도우 7 이후 나오는 운영체제는 램이 1GB 이상이여야 하더라구. 
     근데 구식 넷북 램이 딱 1GB라서 1GB 램으로 쾌적하게 쓸 수 잇는 운영체제를 찾아야 했음. 
     우분투 리눅스도 램이 512MB以上면 돌아가지만 램이 적은 랩탑에 깔아본 적이 있는데 성능이 별로였음. 
     그래서 결국에는 가벼운 운영체제를 찾아야 했고 그 운영체제는 아치 리눅스였음 ㅋㅋㅋ. 
     아치 리눅스는 심플하고 모던하고 사용자 중심적인 운영체제라고 알고 있었고 아치 리눅스 깔면서 운영체제에 대해서도 많이 배울 수 있을 것 같아서 시작했음. 
     # 전 lắp đặt
     아치 리눅스는 정식으로 2017년 이후로 32비트CPU지원을 중단함. 
     그래서 community에서 유지하고 잇는 아치 리눅스 32를 깔았음.
     # 부팅 미디어 준비
     아치 리눅스 32 이미지 파일下载하고 usb에 설치했음. 
     아치 리눅스 32 설치하는데는 아치 리눅스 설치 가이드를 따라하면 됨. 
     # 인터넷 연결
     아치 리눅스 zsh 기본 쉘이라서 쉘에서 명령어 치면 돼. 
     인터넷 연결하려면 일단 네트워크 인터페이스 상태를 확인 해야함. 
     그다음에 iwctl이라는 명령어로 와이파이 네트워크 스캔하고 네트워크에 연결하면 돼.
     연결된 네트워크에 비밀번호를 입력하면 돼. 
     그다음에 ping이라는 명령어로 아치 리눅스 서버에 연결하는지 확인하면 돼.
     # 시스템 시계 설정
     ntp протокол으로 시스템 시계를 설정할 수 있음.
     # 디스크 분할
     fdisk라는 명령어로 디스크 목록을 확인할 수 있음.
     sda는 usb이고 sdb는하드디스크고 그안에 파티션을 만들면 돼
     MBR과 GPT 파티션 스키마가 있는데 MBR은 구식이고 GPT가 최신이라서 GPT를 쓰면 좋음.
     근데 아치 리눅스 32는 바이오스에서 부팅해야 하기 때문에 MBR을 사용해야 함.
     # 설치
     아치 리눅스 32를 설치하고swap 파티션을 만드려고했는데 램이 1GB로 작아서_swap 파티션의 크기를 많이 늘려서 하느니라_No space left on device 에러가 자주나서 짜증났음 ㅋㅋㅋ.
     RHEL에 따르면 램이 2GB 이하일 경우 swap 파티션은 램의 두배로 설정해야 함.
     # 설치 후
     아치 리눅스 32를 설치하고 아치 사용자 저장소에서 필요한 패키지를 깔아서 쓸 수 있음
