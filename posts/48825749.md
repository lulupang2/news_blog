# 텐다 장비_FW 인증백도어 취약점 발견  ㄹㅇ
   텐다 장비의_FW에는 인증백도어가 있어서 해커들이 원격으로 관리자 권한을 얻을 수 있음. 
   CVE-2026-11405로 등록된 이 취약점은 텐다 장비의 웹 관리 인터페이스에 접근할 때 사용되고, 
   텐다_US_FH1201V1.0BR_V1.2.0.14(408)_EN_TD부터 US_AC6V2.0RTL_V15.03.06.51_multi_TDE01 버전까지 영향을 받음. 
   텐다 장비를 쓰고 있다면FW를 업데이트 해야하고, FW 업데이트가 안된다면 원격 관리를 꺼야함. 
   또한기본 LAN IP 주소를 변경해 볼 수 잇지만, 이것으로는 모든 취약점을 해결할 수는 없음. 
   따라서 텐다 장비를 쓰고 있다면 조심해야하고, FW 업데이트를 기다려야함. 
   그리고 다른 네트워크 장비들도 주기적으로 업데이트 하라는 말도 같이 하고 싶어 ㅋㅋ
