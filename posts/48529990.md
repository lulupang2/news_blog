# 오프라인에서도 볼 수 있는 웹사이트 미러링 툴 'Kage' 공개! 
웹사이트를 미러링 해 오프라인에서 볼 수 있는 툴 'Kage'가 공개됐어. 
### Kage란?
Kage는 웹사이트를 하나의 바이너리로 저장해 오프라인에서 볼 수 있게 해주는 툴이야. 
기존의 'Save As'방식이랑 달리 자바스크립트는 모두 삭제하고, CSS, 이미지, 폰트만 가져와서 저장해. 
### Kage의 특징
* 웹사이트를 미러링 해 오프라인에서 볼 수 있음 
* 자바스크립트를 삭제하고 CSS, 이미지, 폰트만 가져와서 저장 
* 하나의 바이너리로 저장되어 용량이 작음 
### Kage의 사용법
1. Kage를 설치해 
2. 웹사이트를 클론해 
3. 클론한 웹사이트를 패킹해 
4. 패킹한 파일을 열어
