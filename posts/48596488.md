### 데이터 분석에 Plotnine 쓴 경험 있으면 개빡치네 ㅋㅋ
   요즘 데이터 분석하면서 Plotnine이라는 툴 쓰는 사람 많은 것 같아서 경험 공유해 봄.
   ### Plotnine 소개
   Plotnine은 파이썬에서 데이터 시각화를 위한 문법 기반 그래프 패키지임. 
   R 패키지인 ggplot2와 비슷한 문법을 사용해서 데이터 시각화하는 게 개쉽게 됨.
   ### Plotnine 사용법
   Plotnine으로 데이터 분석부터 시각화까지 개쉽게 해봄 ㄹㅇ.
   먼저 Anscombe의 Quartet이라는 데이터셋으로 설명해 볼게.
   이 데이터셋은 4개의 작은 데이터셋으로 구성되어 있는데, 서로 다른 분포를 가지고 있음.
   하지만 기술 통계치로는 거의 동일한 값을 가지고 잇음.
  그래서 이 데이터셋은 데이터 시각화의 중요성을 설명하는데 쓸만한 데이터셋이라고 할 수 있음.
   ### Plotnine 코드
   ```python
   from plotnine import *
   from plotnine.data import anscombe_quartet

   ggplot(anscombe_quartet, aes(x='x', y='y')) + geom_point()
   ```
   이 코드로 간단하게 점근 그래프를 만들 수 있음.
   하지만 아직은 별로 유용하지 않기 때문에 더 자세히 살펴보자.
   ```python
   ggplot(anscombe_quertet, aes(x='x', y='y', color='dataset')) + geom_point()
   ```
   이 코드로 각 데이터셋마다 다른 색상으로 표시할 수 있음.
   자동으로 범례도 생성되므로 편리함.
   ### 개별 패널로 시각화하기
   ```python
   ggplot(anscombe_quartet, aes(x='x', y='y', color='dataset')) + geom_point() + facet_wrap('dataset')
   ```
   이 코드로 각 데이터셋마다 개별 패널로 시각화할 수 있음.
  这样하면 색상 표시가 필요 없기 때문에 더 깔끔해 지는 것 같음.
   ### trend line 추가하기
   ```python
   ggplot(anscombe_quartet, aes(x='x', y='y', color='dataset')) + geom_point() + geom_smooth(method='lm', se=False, fullrange=True) + facet_wrap('dataset')
   ```
   이 코드로 각 패널마다 trend line을 추가할 수 있음.
   이 그래프는 Anscombe의 Quartet의 핵심을 잘 보여주고 있기 때문에 만족스럽다.
   ### 더 자세한 설정
   ```python
   ggplot(anscombe_quartet, aes(x='x', y='y')) + geom_point(color='sienna', fill='darkorange', size=3) + geom_smooth(method='lm', se=False, fullrange=True, color='steelblue', size=1) + facet_wrap('dataset') + scale_y_continuous(breaks=(4, 8, 12)) + coord_fixed(xlim=(3, 22), ylim=(2, 14)) + labs(title='Anscombe의 Quartet')
   ```
   이 코드로 더 자세한 설정을 할 수 있음.
   각 요소의 색상, 크기, 레이블 등을 설정할 수 있음.
   이런 설정들은 후에 다시 설명하도록 할게.
   ### 테마 설정
   ```python
   ggplot(anscombe_quartet, aes(x='x', y='y')) + geom_point(color='sienna', fill='orange', size=3) + geom_smooth(method='lm', se=False, fullrange=True, color='steelblue', size=1) + facet_wrap('dataset') + labs(title='Anscombe의 Quartet') + scale_y_continuous(breaks=(4, 8, 12)) + coord_fixed(xlim=(3, 22), ylim=(2, 14)) + theme_tufte(base_family='Futura', base_size=16) + theme(
       axis_line=element_line(color='#4d4d4d'),
       axis_ticks_major=element_line(color='#00000000'),
       axis_title=element_blank(),
       panel_spacing=0.09
   )
   ```
   이 코드로 테마 설정을 할 수 있음.
   기본적으로 제공되는 테마를 사용하거나, 사용자 정의 테마를 만들 수 있음.
   이런 테마 설정들은 후에 다시 설명하도록 할게.
   ### 결론
   Plotnine은 데이터 시각화를 위한 강력한 도구임.
   이 툴을 사용해서 데이터 분석부터 시각화까지 개쉽게 할 수 있음.
   이 글에서 소개한 내용들은 아주 기본적인 내용들인데, 후에 더 자세한 내용들을介绍하도록 할게.
