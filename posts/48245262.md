# 소개
루비로 만든 리눅스 셸 '루비쉬'가 나타났음. 이건 그냥 리눅스 명령어만 지원하는 게 아닌 루비 코드도 돌릴 수 있단 점이 특이점임.

# 특징
* 루비 코드도 리눅스 명령어도 같이 사용 가능. 결국 루비의 강력한 기능을 셸 스크립트에서도 사용할 수 있단 얘기임.
* 리눅스 명령어는 그냥 사용할 수 있고, 루비 코드는 `{}` 안에다가 작성하면됨. 예를 들어 `{ time = Time.now }`이런 식으로.
* 리눅스 명령어를 루비 메소드 스타일로 호출할 수 있음. 예를 들어 `ls('-la')`나 `cat('file.txt')` 이런 식으로.
* 리눅스 명령어를 체이닝할 수 있음. 예를 들어 `ls().sort()`나 `cat('file.txt').grep(/error/)` 이런 식으로.
* 루비로만 작성된 셸이라서 기본적으로 루비의 모든 기능을 사용할 수 있음. 결국 리눅스 명령어도 루비 코드도 그냥 섞어서 사용하는 거임.
* `lazy_load`라는 기능을 이용하면 셸 초기화를 지연시킬 수 있음. 결국 셸을 키면 바로 초기화가 되는데, 이 초기화를 지연시키는 거임. 이 초기화는 따로 쓰레드에서 돌아감.
* Restricted 모드라는 게 있어서 이 모드를 키면 루비 코드는 사용할 수 없음. 결국 일반적인 리눅스 셸만 사용하는 거임.
* zsh 호환성도 지원해줌. zsh 스타일의 명령어도 잘 돌아감.
* 설정 파일은 `~/.config/rubish/config`나 `~/.rubishrc` 이런 파일을 사용함. 예를 들어 `~/.rubishrc`에 `alias ll='ls -l'` 이런 식으로(alias 같은 명령어들)을 작성해 둘 수 있음.

# 사용법
 그냥 설치해서 쓰면 됨. `brew install --HEAD rubish` 이런 식으로 설치할 수 있음. 설치하면 바로 셸을 열어서 사용할 수 있음.
