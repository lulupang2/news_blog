# 소개
노트북 하나로 작가가 될 수 있을까? 오늘은 내가 만든 작가 전용 장비 'writerdeck'에 대해 이야기를 해 볼 거야.

# 작가 전용 장비 만들기
일단 작가 전용 장비를 만들기 위해서는 몇 가지 조건이 필요해. 첫 번째로, 작家的 방해 요소가 없어야 하고, 두 번째로, 쓸만한 키보드와 좋은 화면이 있어야 해.나는 이런 조건을 충족시키는 장비를 만들기 위해 오래된 노트북 하나를 이용했어. 이 노트북은 System76 Galago Pro야. 리눅스를 지원하니까 좋았어.

# Debian 설치
첫 번째로, 이 노트북에 데비안을 설치해야 했어. 데비안을 설치하는 과정은 매우 쉽지만,ちょっと 트릭이 있어. 우선, 데비안을 다운받기 위해 데비안 웹사이트에 가서 다운로드 해야 해. 그리고는 노트북에 설치해야 해. 설치 과정에서는 풀디스크 암호화를 스킵하고, 사용자 계정을 만들 때 루트 암호를 비워두면 sudo 사용자로 자동으로 설정되는 점을 이용할 수 있어.

# 네트워크 매니저 설치
데비안을 설치한 후, 네트워크 매니저를 설치했어. 네트워크 매니저를 설치하면, 와이파이 네트워크에 연결하기가 dễ려워. 네트워크 매니저를 설치하는 방법은 간단해. apt install network-manager만 해주면 돼.

# neovim과 kmscon 설치
네트워크 매니저 설치 후, 노트북에 neovim과 kmscon을 설치했어. neovim은 에디터고, kmscon은 터미널에서 사용하는 폰트와 색상을 변경할 수 있게 해 줘. neovim을 설치하는 방법은 apt install neovim이고, kmscon은 backports에서 받아와야 해.

# tmux 설치
그 다음, tmux를 설치했어. tmux는 터미널을 분할해서 사용할 수 있게 해 줘. tmux를 설치하는 방법은 apt install tmux야.

# acpi와 light 설치
또, 배터리 잔량과 화면 밝기를 변경할 수 있게 해 주는 acpi와 light를 설치했어. acpi와 light를 설치하는 방법은 apt install acpi light야.

# vimwiki 설치
마지막으로, vimwiki를 설치했어. vimwiki는 노트북에 기록을 남길 수 있게 해 줘. vimwiki를 설치하는 방법은 apt install vim-vimwiki야.

# Syncthing 설치
마지막으로, Syncthing을 설치했어. Syncthing은 노트북에 있는 파일을 다른 장치와 동기화시킬 수 있게 해 줘. Syncthing을 설치하는 방법은 문서를 따라하면 돼.
