Ü 프로그래밍 언어는 정적으로 타이핑된 컴파일 언어야. 코드를 작성할 때 안전하고 빠르라고 설계했어. 안전한 코드와 불안전한 코드 구분이 되고, 컴파일 타임에 올바른지 체크가 가능해. RAII, 템플릿, 캡슐화, 람다, 코루틴 등 다양한 기능을 지원해.
Ü는 메모리와 자원 관리를 위해 RAII를 사용하지만, 불안전한 코드에서는 수동으로 메모리를 관리할 수 있지. Ü는 불안전한 코드가 없는 한 메모리 안전과 스레드 안전을 보장해. C++에서 영감을 받았지만, C++의 단점은 없다고 합니다. 또한 Rust의 영향을 받았지만, 훨씬 사용하기 쉬워.
Ü 컴파일러는 LLVM 라이브러리를 기반으로 하고 있어, 많은 최적화와 코드 생성 기능을 지원해. Ü에는 컴파일러 외에도 표준 라이브러리, 빌드 시스템, 언어 서버, 문법 강조 파일 등 다양한 컴포넌트가 있지.
지원 하는 운영 체제는 윈도우, 리눅스, macOS 등이래. CPU 아키텍처는 x86, x86_64, AArch64 등이 destek해.
Ü를 선택하는 이유는 다른 언어에 비해 안전하고, 신뢰할 수 있고, 표현력이豊か하고, 많은 기능을 가지고 있다고 합니다. C, C++, Swift, Zig, Odin, Rust 등과 비교해 봐도 Ü가 최고래 ㅋㅋㅋㅋ
Ü를 빌드하기 위해서는 최신 C++ 컴파일러, CMake, Ninja 등이 필요해. LLVM 소스를 다운로드 해서 빌드할 수도 있고, LLVM 바이너리 라이브러리를 사용할 수도 있어.
Ü를 사용하기 위해서는 Ecode, QtCreator, Visual Studio 등 다양한 IDE를 사용할 수 있지.Ü에 대해서 더 자세한 정보는 공식 사이트에서 확인할 수 있지.
