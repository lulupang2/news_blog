이 템플릿은 GitHub에서公开되고 있고, 개발자들은 언제든지 이 템플릿을 사용해서 Flipper Zero 애플리케이션을 개발할 수 있음.Flipper Zero 개발자들은 이 템플릿을 사용해서 더 많은 Flipper Zero 애플리케이션을 개발할 수 있을 거라期待중임.

###.flipper-template 프로젝트 구조

_flipper-template 프로젝트는 다음과 같은 구조를 가졌음.

```markdown
flipper-template/
├── application.fam # Flipper 애플리케이션 매니페스트
├── build.zig # Zig 빌드 시스템 구성
├── build.zig.zon # Zig 패키지 매니페스트
├── icon.png # 애플리케이션 아이콘
├── setup.sh # 프로젝트 초기화 스크립트
├── src/ # 소스 코드 디렉토리
│ └── root.zig # 메인 애플리케이션 소스 코드
└── zig-out/ # 빌드 아티팩트 디렉토리
    └── bin/ # 컴파일된 오브젝트 파일 디렉토리
        └── app.o # 컴파일된 오브젝트 파일
```

#### 소스 코드구조

소스 코드는 `src/` 디렉토리에 위치한 `root.zig` 파일에 존재함. 이 파일은 Flipper Zero 애플리케이션의 메인 엔트리 포인트임.

#### 빌드 과정

빌드 과정을 다음과 같이 설명할 수 있음.

1. `zig build` 명령어를 사용해서 소스 코드를 컴파일함.
2. `zig build fap` 명령어를 사용해서 컴파일된 오브젝트 파일을 패키징함.
3. `zig build launch` 명령어를 사용해서 패키징된 애플리케이션을 Flipper Zero 디바이스에 배포하고 실행함.
