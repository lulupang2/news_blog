# 지오시티스 스타일 웹 사이트 만들기 
   파이썬으로 90년대 지오시티스 같은 웹 사이트를 만들 수 있는 방법이 나왔 븅신들아. 
   이 방법으로 몇 초 안에 자신만의 지오시티스 스타일 웹 사이트를 만들 수 있대. 
   근데 문제는 사이트의 일부가 로딩이 안 되면서 '사이트 일부 로딩 실패' 같은 메시지가 뜨는겨. 
   브라우저 확장 프로그램이나 네트워크 문제, 브라우저 설정 때문이라고 하네. 
   그래서 브라우저 확장 프로그램 끄라고 하거나 다른 브라우저 써봐라고 함. 
   암튼 이렇게하면 아주 간단하게 지오시티스 스타일 웹 사이트를 만들 수 있대.
