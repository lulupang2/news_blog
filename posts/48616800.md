# IPv6가 뭔데? 
   쉽게 말해서 현재 우리가 쓰고 있는 IPv4는 IP 주소가 부족해서 만들어진 새로운 시스템이야. 
   ## IPv6를 왜 써야 하나요? 
   IPv4 주소가 다 떨어져서 못 써서야. 
   # IPv6 사용률이 50% 달성 
   구글에서 측정한 결과 IPv6 사용률이 50%를 돌파했다고 발표했어. 
   하지만 APNIC이라는 기관에서 측정한 결과는 42%라고 하는데, 이게 왜 이런 차이가 나는 건지 좀 아래에서 설명하겠음. 
   ### IPv6 사용률 차이 
   APNIC에서는 광고를 통해 데이터를 수집하는데, 구글은 사용자가 직접 서비스를 사용할 때 데이터를 수집하기 때문에 차이가 나는 거야. 
   APNIC에서는 인터넷 사용량이 많은 곳에 데이터를 더 많이 수집하기 때문에 결과가 달라질 수 있음. 
   # IPv6를 쓸 때 장단점 
   ### 장점 
   *IPv4 주소 부족 문제 해결* 
   *새로운 기능 지원* 
   ### 단점 
   *하위 호환성 문제* 
   *기존 시스템과 연동이 어려울 수 있음* 
   ## IPv6로 바꾸면 편해질까? 
   사실 바꾼다고 해서 편해지거나 어려워지는 건 아님. 
   그냥 새로운 시스템을 익히기만 하면 돼. 
   # 결론 
   IPv6가 결국 50% 달성했어. 
   하지만 여전히IPv4와 함께 쓰고 있어서,雙方 都는 복잡하게 운영이 되고 있어. 
   그래도IPv6는 이제 막 시작된 거라고 생각하면, 
   앞으로 더 발전하고 개선될 거야. 
   IPv6에 대해 더 많이 배우고 사용해 봐. 
   그리고IPv6를 지원하는 서비스를 많이 이용해 주자!!
   개빡치는 점은 우리나라는IPv6를 사용하는 곳이 아직도 많이 부족해. 
  그래서인지 인터넷이 느릴 때가 많음. 
   그래도 IPv6를 쓴다고 해서 인터넷이 바로빨라지거나 하는 건 아니니까,
