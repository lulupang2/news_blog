### 새 관찰 시스템 만들기

난방 안 되는 집 앞마당에 새 관찰 시스템을 구축해 봤어. 아예 개인 프로젝트로 진행할 계획이었는데, 트위터에서 반응이 너무 좋았 ヾ(๑╹◡╹)ﾉ" så야 간단하게 설명할까 하다가 결국 시스템 구축 방법을 자세히 써 보기로 했음.

#### 시스템 구성

* Raspberry Pi (4B / 5 / Zero 2W) - 약 35~80 달러
* Micro SD Card (≥32 GB) - 약 10 달러
* USB 라바리어 마이크 - 약 16.95 달러
* Pi 전원 공급기 - 약 10 달러

총 비용은 약 80 달러임. 추가로 Gemini API 키와 eBird API 키가 필요해.

#### 설치 방법

1. Raspberry Pi Imager로 SD 카드를 초기화하고, Raspberry Pi OS Lite (64-bit)로 설치해.
2. USB 마이크를 Pi에 연결하고, 창문이나 외부에 설치해.
3. SSH로 Pi에 접속해서 설치 스크립트를 실행해.

```bash
ssh <사용자 이름>@birdnet.local
curl -s https://raw.githubusercontent.com/Twarner491/AvianVisitors/avian-visitors/newinstaller.sh | bash
```

설치 후에는systems의 birdnet.local에 접속해서 새 관찰 시스템을 사용할 수 있어.

#### 외부 접속 설정

기본적으로 시스템은 내부 네트워크에서만 접속할 수 있지만, 外部 접속을 설정할 수도 있어.

1. Cloudflare Tunnel을 이용해서 公共 HTTPS URL을 생성할 수 있어.
2. Home Assistant REST 센서를 이용해서 시스템과의 연동을 설정할 수 있어.
3. MQTT 브릿지를 이용해서 시스템과의 연동을 설정할 수 있어.

#### 새 도면 생성

시스템은 Gemini의 gemini-2.5-flash-image 모델을 이용해서 새 도면을 생성해.
새 도면은 450개의 일러스트레이션이 기본으로 포함되어 있고, 추가로 생성할 수도 있어.
```python
export GEMINI_API_KEY='당신의 키'
python3 ~/BirdNET-Pi/avian/scripts/pregen.py --labels ~/BirdNET-Pi/model/labels.txt --force
```
새 도면은 kachō-e 스타일로 생성될 거고, 실제 새 관찰 시스템에 사용될 거야.

#### 마무리

요약하자면 난방 안 되는 집 앞마당에 새 관찰 시스템을 구축해서 사용해 봤음.
시스템은 Raspberry Pi와 USB 마이크를 이용해서 새들을 관찰하고 구분하는 거야.
추가로 외부 접속 설정, 새 도면 생성 등 다양한 기능을 지원해.
