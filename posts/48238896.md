> **🔗 원문 보러가기:** [Microsoft starts canceling Claude Code licenses](https://www.theverge.com/tech/930447/microsoft-claude-code-discontinued-notepad)


마이크로소프트가 Claude Code 라이센스를 취소하기 시작했다고 함.
 Thousands 개의 마이크로소프트 개발자들이 GitHub Copilot CLI 사용할 거라고...
 
원래 마이크로소프트는 Claude Code를 사용해 개발자들이 코드를 더 쉽게 작성할 수 있도록 도와주고 있었음.
하지만 이제는 GitHub Copilot CLI를 사용하라고 하고 있음.
 
왜냐면 Claude Code가 마이크로소프트의 GitHub Copilot와竞爭이 되는 상황이 되었기 때문임.
마이크로소프트는 Claude Code를 사용하는 개발자들을 GitHub Copilot CLI로 옮겨야 한다고 생각하는 거 같음.
 
이제 마이크로소프트 개발자들은 Claude Code를 더 이상 사용하지 못하고, GitHub Copilot CLI를 사용해야 할 거야.
