코드 리뷰의 목적은 유지보수가 어려운 코드를 찾는 거임. 
   그래서 코드 리뷰때 가장 중요한 건 코드의 유지보수性을 높이는 거지. 
   유지보수性이 낮으면 مستق블에 코드 수정이나 추가가 어려워져서 코드의 퀄리티도 떨어짐. 
   코드 리뷰를 통해 코드의 구조나 로직을 개선하면 유지보수性을 높여줄 수 있음. 
   그리고 코드 리뷰는 디버깅에도 큰 도움을 줌. 
   코드 리뷰를 통해 코드의 에러나 버그를 찾을 수 있고, 코드의 안정성을 높일 수 있음. 
   그러니까 코드 리뷰는 정말 중요하다고 볼 수 있음. 
   kode를 작성할 때 항상 코드 리뷰를 고려해서 작성해야 함. 
   코드 리뷰를 통해 코드의 퀄리티를 높여야 함. 
   kode 리뷰가 제대로 이루어지면 mantenace 비용도 많이 절감할 수 있음. 
   그러니까 kode 리뷰는 꼭 해야하고, kode 리뷰를 통해 코드의 퀄리티를 높여야 함.
