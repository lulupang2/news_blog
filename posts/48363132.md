# 오픈 艾아이 구축 모델과 코덱스 이제 아마존 웹 서비스에서 사용 가능해 짐
오픈 艾아이의 구축 모델과 코덱스가 이제 아마존 웹 서비스에서 사용할 수 있게 되었음. 이거랑 관련해서 오픈 艾아이와 아마존 웹 서비스는 최근 협력을 강화하고 있고, 개발자들이 새로운 기술을 좀 더 쉽게 사용할 수 있게 하고 싶어 함. 오픈 艾아이의 구축 모델과 코덱스는 개발자들이 자연어 처리와 관련된 작업을 좀 더 쉽게 할 수 있게 해줄 거임. 예를 들어,開発者들은 이제 오픈 艾아이의 구축 모델과 코덱스를 사용해서 채팅 봇이나 번역기 등을 만들 수 있게 될 거야. 이거는 개발자들에게 획기적인 기술이라 봐도 될듯? ㅋㅋㅋ

이제 아마존 웹 서비스에서 오픈 艾아이의 구축 모델과 코덱스를 사용할 수 있게 되었으니, 개발자들은 이걸 사용해서 다양한 프로젝트를 진행할 수 있게 됨. 예를 들어, 개발자들은 오픈 艾아이의 구축 모델과 코덱스를 사용해서 자연어 처리와 관련된 작업을 할 수 있고, 이거를 사용해서 더 좋은 결과를 얻을 수 있게 될 거야. 개발자들이 오픈 艾아이의 구축 모델과 코덱스를 사용해서 무엇을 만들까? 그게 기대되고 재미져 보이네? ㄹㅇ
