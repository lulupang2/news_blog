### 상황 정리
요즘 AI 언어 모델(LLM)이 너무 많이 발전해서, 내가 10년 동안 쌓아왔던 직업이 다 사라지는 중이야. 일단 나는 소프트웨어 엔지니어야. 내 경력은 웹 프론트엔드에서 시작해서, 벡엔드로 넘어갔어. 그리고 금융, 회계, 결제 처리 같은 분야에서 일했어. 여기서 나는 PCI 준수, 이중원장 계산, 에스코우, 정산, 결제 라이프사이클, 은행 송금 멱등성을 다 học었다는 거임. 

### 도메인 지식이 사라지는 것
근데 últimamente 내가 다루던 도메인 지식이 다 사라지고 있어. ChatGPT, Claude Enterprise 이런 거 쓰면서, 내가 쓴 코드를 대체하고 있어. 처음에는 내가 작성한 코드를 AI가 완성하거나, 작성한 문서를 확인하는 용도로만 생각했는데, 어느새 내 역할을 다 대체하고 있어.

### 디버깅이 사라지는 것
내가 디버깅을 잘한다고 생각했는데, 이게 최근에 나온 MCPs, Claude 4.5 같은 도구들한테도 못 địch하고 있어. 이 도구들은 버그를 하나도 안 남기고 바로 고치고, 심지어 내가 풀기 어려웠던 버그들도 쉽게 해결해주고 있어. 그러다 보니, 내가 쌓아왔던 도메인 지식, 디버깅 능력, 코드 품질 이런 걸 하나도 안 쓰게 되고 있어.

### 코드 품질이 사라지는 것
마지막으로 내가 남은 코드 품질이나 소프트웨어 아키텍처에 대해서는, 여전히 내가 해야 하지만, 이것도 의미가 없어지고 있어. AI가 코드를 작성하는 데에는 그다지 실력이 없는데, 코드를 작성할 때는 인간이 도와줘야 하지만, 코드 품질에 대해서는 별로 신경 안 씀. 결국 코드 구조를 잘 갖추는 것, 디버깅을 잘 하는 것, 이 모든 게 사라져 가고 있고, 이걸 막을 방법도 отсутств하고 있음.

### 대책 마련
내가 이렇게 AI에게 다 털리고 있는 거 보니까, 진짜로 AI 기술을 깊이 공부하고, 연구자로서 일한다는 방향으로 공부할 생각임. 그런데, 이것을 공부하기 위해서는 학교에 다시 가거나, 연구실에 들어가야 하는데, 이것도 쉽진 않음.
