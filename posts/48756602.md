> **🔗 원문 보러가기:** [Kimi K2.7 Code is generally available in GitHub Copilot](https://github.blog/changelog/2026-07-01-kimi-k2-7-is-now-available-in-github-copilot/)


### 깃허브 코파일럿에서 키미 K2.7 코드 일반 제공 시작!
깃허브 코파일럿에서 **키미 K2.7 코드**라는 오픈웨이트 모델을 일반적으로 사용할 수 있게 되었음! 
이건 코파일럿에서 모델 선택할 때 사용할 수 있는 첫 번째 오픈웨이트 모델이라서 개발자들에게 더 많은 선택지도 제공함! 
그리고 비용도 낮춰서 코딩 워크플로우에 유용하다고 할 수 있음!
 
### 키미 K2.7 코드 특징
- **호스팅**: 깃허브가 마이크로소프트 애저에 호스팅해서 사용할 수 있게 해줌!
- **요금**: 제공자 가격 책정에 따라 사용량 기반의 비용 청구를함!
- **사용 가능 플랜**: 코파일럿 프로, 프로+, 맥스 플랜에서 사용 가능하고渐進적으로 확대할 예정임!
 
### 사용 방법
- **Visual Studio Code**: 버전 1.127.0 이상에서 사용할 수 있음!
- **Visual Studio**: 버전 17.14.6 이상에서 사용할 수 있음!
- **깃허브 코파일럿 클라우드 에이전트**: 사용할 수 있음!
- **깃허브 모바일 앱**: IOS와 안드로이드에서 사용할 수 있음!
- **JetBrains**: 버전 1.9.1-251 이상에서 사용할 수 있음!
 
### 기업용 사용
- **기업용 및 엔터프라이즈**: 기본적으로 키미 K2.7 코드가 비활성화되어있는데, 관리자가 설정을 변경해야 사용할 수 있음!
- **보안 및 준수**: 관리자가 자신의 보안, 규정 준수, 데이터 거버넌스 요구사항에 따라モデル을 활성화하는 것을 추천함!
 
### 모델 선택 가이드
- [깃허브 코파일럿에서 지원하는 모델](https://github.com/docs/codespaces/supported-models) 문서에서 더 많은 정보를 볼 수 있음! 
- [적합한 AI 모델을 선택하는 방법](https://github.com/docs/codespaces/choosing-the-right-ai-model)도 확인할 수 있음!
 
### 피드백 공유
- [깃허브 커뮤니티](https://github.com/community)에서 피드백을 공유해줘!
