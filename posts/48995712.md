# 유니티 cli 
     이제 터미널에서 유니티 프로젝트 관리가 가능해졌다. 툴 이름은 유니티 클라이언트라고 하고_cli_라고 부름. 
     그냥 unity로 실행하면 되기 때문에 외울일도無し ㅋ 
     ## 설치하기 
     ### 맥 OS, 리눅스 
     ```bash
curl -fsSL https://public-cdn.cloud.unity3d.com/hub/prod/cli/install.sh | UNITY_CLI_CHANNEL=beta bash
```
     ### 윈도우 
     ```powershell
$env:UNITY_CLI_CHANNEL='beta'; irm https://public-cdn.cloud.unity3d.com/hub/prod/cli/install.ps1 | iex
```
     이렇게 설치하면 되지. 
     # 사용해보기 
     ## 에디터 설치 
     ```bash
unity install 6000.2.10f1 -m android ios webgl
```
     ## 에디터 목록 보기 
     ```bash
unity editors
```
     ## 에디터 열기 
     ```bash
unity open
```
     # cli를 이용한 자동화 
     ## cli 명령어 
     ```bash
unity --help
```
     # 평가 
     ## cli 명령어의 결과를 json이나 tsv 포맷으로 얻기 
     ```bash
unity editors --format json
```
     ```bash
unity editors --format tsv
```
     # pipeline 
     ## 패키지 설치 
     ```bash
unity pipeline install
```
     ## 연결된 에디터 목록 확인 
     ```bash
unity pipeline list
```
     # cli 명령어 
     ## registered 명령어 실행하기 
     ```bash
unity command 
```
     ```bash
unity command <command-name>
```
     #_registered 명령어 
     ## static 메소드에 cli 명령어 등록하기 
     ```csharp
using Unity.Pipeline.Commands; 
using UnityEngine; 

public static class MyPipelineCommands { 
   [CliCommand("greet", "Log a greeting and return its length")] 
   public static int Greet( 
      [CliArg("name", "Who to greet", Required = true)] string name) { 
         Debug.Log($"Hello, {name}!"); 
         return name.Length; 
      } 
   }
```
     # eval 
     ## cli를 이용한 c# 코드 평가하기 
     ```bash
unity command eval "return Application.version;"
```
     ```bash
unity command eval "return UnityEditor.EditorApplication.isPlaying;"
```
     ```bash
unity command eval "var s = Application.dataPath; return s.Length;"
```
     ```bash
unity command eval_file "path/to/script.cs"
```
