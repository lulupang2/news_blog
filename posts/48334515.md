### Pandoc Templates: 마크다운 문서로 깔끔한 LaTeX PDF 만들기
   ### 개요
   - Pandoc Templates는 마크다운 문서를 다양한 포맷(e.g., PDF, LaTeX, HTML, DOCX)로 변환할 수 있게 해줌. 
   - Pandoc Templates를 쓰면, 개발자거나 문서 작성자가 아니라도 쉽게 마크다운 문서로 깔끔한 LaTeX PDF를 만들 수 있게 됨
   ### 주요 특징
   - **LaTeX PDF 생성**: 마크다운 문서를 LaTeX PDF로 변환할 수 있음. 
   - **다양한 문서 지원**: Artikel, Book, Dissertation/Thesis, Letter, CV/Résumé, Presentation/Slide, Paper, Website, Cookbook/Recipe 등 다양한 문서 지원
   - **Customizable**: 사용자가 템플릿을 수정하여 자신만의 스타일로 문서를 생성할 수 있음
   ### 사용 사례
   - 학술 논문이나 책을 작성할 때 마크다운 문서로 내용을 작성하고, LaTeX PDF로 제출할 수 있음
   - 개인 블로그나 웹사이트에 마크다운 문서를 업로드하여 깔끔한 LaTeX PDF로 변환할 수 있음
   ### 결론
   Pandoc Templates를 이용하면 마크다운 문서를 쉽게 깔끔한 LaTeX PDF로 변환할 수 있게 해줌. 개발자거나 문서 작성자가 아니더라도, Pandoc Templates를 사용하면 쉽게 깔끔한 문서를 만들 수 있게 됨.
