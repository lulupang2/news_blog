### 아마존 Bedrock에서Anthropic 모델 쓰려면 데이터 공유해야함 
이제는 Mythos 같은 모델 쓰려면 데이터 30일간 Anthropic랑 공유해야함 ㄹㅇ 개빡치는 점 
Fable 5, Mythos 5 이런 모델들은 Anthropic가 30일간 데이터 모두 저장해야됨 
이렇게 하면 Anthropic가 데이터 하나하나 보고는 안 보이는 악용 패턴을 찾을 수 있음 ㅋㅋㅋㅋ 
一度 데이터 저장하기 시작하면 AWS 데이터 보안 경계를 벗어나는 거야 
이후 30일过ぎ면 자동으로 데이터 삭제되는데, 안전 조사나 법적으로 요구되는 경우만 예외임 ㅇㅋ? 
 자세한 건 [Anthropic 홈페이지](https://support.claude.com/en/articles/15425996-data-retention-practices-for-mythos-class-models)에서 확인할 수 있음.
