# 오디오매스란?
   오디오매스는 웹 기반 멀티 트랙 오디오 에디터야. 무료로 사용할 수 있고 오픈소스인데, 여러 트랙을同時적으로 편집할 수 있어. 
   # 특징
   - 무료이면서도 프로급 편집기능 지원함
   - 오픈소스라서 개발자들이 직접 기여할 수 있음
   - 웹 기반으로 다양한 브라우저에서 사용 가능
   ㄹㅇ 편리하고 강력하지 않음? 웹 브라우저에서 사용할 수 있는 오디오 에디터 중에서 이런 건 또 없을 것 같단 생각이 든다. 
   # 사용 방법
   오디오매스 사용 방법은 간단해.首先 웹 사이트에 접속해서 오디오 파일을 업로드하고, редак트 화면에서 트랙을 편집하면 될 거임. 
   개인적인 생각이지만, 이 에디터는 정말 혁신적이야. 무료로 사용할 수 있고, 편리함과 기능적인 측면에서 다른 에디터를 앞서고 있음. 기대된다!!
