# 브라우저가 말하지 않은 것들

   요즘은 브라우저도 참 똑똑해졌어요. 우리가 브라우저에 별말도 없이 우리에 관한 정보를 다 알고 있대요 ㅎㄷㄷ. 
   
# 브라우저가 알고 있는 정보

   * 위치: 브라우저가 ip-api.com에 요청을 보내 위치를 알려줘요. 그리고 도시 이름과 인터넷 제공商 이름을 알고 있대요 ㄷㄷ.
   * 기기 정보: 브라우저 API를 이용해 기기의 화면 크기, 브라우저 타입, 언어, 그래픽 처리 장치 등 다양한 정보를 가져올 수 있대요.
   * 폰트 정보: 폰트를 가져와서 어떤 폰트가 설치되어 있는지 알고 있대요.
   
# 브라우저가 하는 일

   * 클립보드에 접근: 클릭이나 탭 등 유저 제스처를 이용해 클립보드에 접근할 수 있대요. 
   
# 우리가 해야 할 일

   * 브라우저 설정 변경: 브라우저 설정을 변경해 정보를 보호할 수 있대요.
   *EXTERNAL 링크를 조심해: 외부 링크를 클릭할 때 주의를 기울여야 해요. 
   
# 결론

   브라우저는 우리에 관한 정보를 많이 알고 있어요. 하지만 우리는 브라우저 설정을 변경하고 외부 링크를 조심하면 정보를 보호할 수 있대요. 그러므로 브라우저 설정을 잘 관리하고 외부 링크를 조심하면 좋을 것 같아요 ㅎㅎ
