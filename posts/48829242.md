### 지구멥티머스(Geosql): Claude/Codex의 지리적 데이터 분석 스킬
   Dekart-XYZ에서 만든 지구멥티머스(Geosql)는 데이터 과학자와 분석가를 위한 지리적 데이터 분석 스킬임. 포스트지아이에스(PostGIS), 빅쿼리(BigQuery), 스노플레이크(Snowflake), 웨어로봇스(Wherobots) 등에서 작동함.
   ### 특징
   * 지도와 함께 작동해서 지리적 데이터를 자동으로 분석할 수 있게 해줌
   * 로컬에서 작동하거나 셀프호스팅으로 사용할 수도 있음
   * 지도없이 작동하는 것보다 4배 이상 빠름
   * SQL쿼리 최적화로 비용을 절약할 수 있음
   ### 설치하기
   파이썬으로 설치하기: `pip install geosql && geosql`
   클라우드에서 설치하기: `geosql install claude`, `geosql install codex`, `geosql install copilot`
   ### 사용 예시
   * `/geosql`로 지리적 데이터를 분석하고 지도에 표시할 수 있음
   * `/geosql Show EV charger density along major roads and render a map`로 전기차 충전소 밀도를 지도에 표시할 수 있음
   * `/geosql Find the top 10 locations for Sporting Goods Store in Seattle based on POI co-location and distance to the nearest competitor`로 스포츠 용품점의 최적 위치를 찾을 수 있음
   ###작동 방식
   * 지리적 데이터를 분석하고 지도에 표시함
   * 사용자 인증을 로컬에서进行하므로 보안이 뛰어난 편
   ### 벤치마크
   * 포함된 벤치마크 테스트에서 100% 성공함
   * 지도없이 작동하는 것보다 4배 이상 빠름
   ### 기타
   * MIT 라이선스로 공개됨
   * 기여자를 모집중임
