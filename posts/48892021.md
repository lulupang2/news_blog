# 소개
솔라펑크 장르의 생존게임 스팀이 아닌 실제 야외 공간에서 진행하는 스트랜드폴에서 현실세계를 바꾸는게 가능함?

# 개발 배경
20년간 게임 개발을 하면서 디지털 세계와 실제 세계를 섞는 게임에 대해 관심을 가졌음. ARGS나 위치 기반 게임같은 거야. 결국에는 Nordic Larp 같은 게임에 관심을 갖게 되었고, 스트랜드폴을 개발하게 됨.

# 게임 소개
스트랜드폴은 에дин버러의 실제 공원에서 진행하는 3시간 동안의 야외 Larp 게임이야. 30명의 플레이어가 비상시대에 찾아온謎의 폭풍을 조사하고,missing 과학자를 따라가고, 장거리無線 네트워크를 연결하며, 중요한 결정을 내려야 해.

# 스토리
게임의 배경은 비상시대에 찾아온謎의 폭풍이 세상에 나타난 상황이야. 플레이어들은 이 폭풍을 조사하고, 해결해야 하는 거야.

# 시스템
플레이어들은 McNair-Feldman Device라는 장치를 사용해 폭풍을 조사하고, 네트워크를 연결할 수 있어. 이 장치는_mesh 네트워킹을 지원하며, GPS, 가속도계, 나침반, 작은 전자종이를 포함하는 장치야.

# 플레이어의 역할
플레이어들은 각자 다른 역할을 맡아야 해. 일부 플레이어는 폭풍을 조사하기 위해 네트워크를 연결하고, 일부 플레이어는 폭풍을 따라가서 이를 관찰할 수 있어.

# 게임의 특징
스트랜드폴은 실제 야외 공간에서 진행되므로, 플레이어들은 실제로 뛰어다니고, 폭풍을 따라가야 해. 또한, 플레이어들은 서로 협력하여 중요한 결정을 내릴 수 있어.

# 개발자
스트랜드폴은 Alex Macmillan과 함께 Experimental Social Scene이라는 팀에서 개발했어. Juhana Pettersson이라는 베테랑 Nordic Larp 디자이너의 도움을 받아 게임을 만들어냈어.
