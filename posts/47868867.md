# 애플 버그 패치 내역
 애플에서 아이폰과 아이패드용 소프트웨어 업데이트를 내놨음. 이 업데이트는 법집행기관이 삭제한 메시지나 자동으로 삭제된 메시지를 뽑아내는 버그를 수정한 거임. 
 왜냐하면 채팅 앱의 알림이 내용까지 캐시해놨거든? 그리고 이 캐시된 내용이 디바이스에 일주일에서 한 달까지 유지된다고 하노 ㅋㅋㅋ
 
 # 문제의 개요
 이 문제는 404 미디어에서 먼저 보도한 건데, FBI가 포렌식 툴을 써서 SIGNAL 앱에서 삭제된 메시지를 아이폰에서 뽑아낸거래. 
 SIGNAL의 알림이 채팅 앱의 메시지 내용을 담으면서, 알림이 캐시되어 디바이스에 저장됐던 거임. 그리고 SIGNAL 앱에서 메시지 삭제했어도, 알림 캐시가 남아있었던거지. 
 
 # SIGNAL 개발자의 요구
 이 문제가 알려지자 SIGNAL 개발자도 애플에 버그 수정을 요청했음. Meredith Whittaker라는 SIGNAL 개발자가 블루스카이 포스트에서 "삭제된 메시지의 알림은 OS에서 캐시될 때가 없는데, 이 버그를 해결해달라"고 말했어. 
 
 # 결과
 결국 버그가 수정되서, 이제는 채팅 앱에서 메시지 삭제하면 알림에서도 내용이 사라짐. 그리고 이 버그는 그냥 오류였던 듯?苹果도 버그 수정했으니 이제 법집행기관도 삭제된 메시지를 못 뽑아내겠노?ㅋㅋㅋ
