# id-agent : Token efficient UUID 대안
     id-agent는 AI 에이전트용 Token 효율적인 UUID 대체품이야. 기존 UUID는 23개의 토큰인데 반해 id-agent는 14개의 토큰으로 Collision Resistance가Equivalent해.
     ### UUID와의比較
     | Format | Example | Tokens | Collision Resistance |
     | --- | --- | --- | --- |
     | UUID v4 | 89b842d9-6df9-4cf4-8db0-9dc3aed3cfd7 | ~23 | 122 bits |
     | id-agent (default) | urd-antes-sorry-pac-dire-total-expire-going | ~14 | ~96 bits |
     ### 설치 및 사용법
     id-agent를 설치하기 위해서는 npm이나 pnpm을 사용해 설치할 수 있어.
     ```bash
     npm install id-agent
     pnpm add id-agent
     ```
     설치 후에는 id-agent를 import하고 사용할 수 있어.
     ```javascript
     import { idAgent } from 'id-agent';
     const id = idAgent();
     ```
     ### Options
     id-agent는 Options를 통해 설정을 변경할 수 있어.
     ```javascript
     const taskId = idAgent({ prefix: 'task' });
     const short = idAgent({ words: 3 });
     ```
     ### API Reference
     id-agent는 다음과 같은 API를 제공해.
     * `idAgent(opts?)`: 랜덤한 ID 생성
     * `idAgent.from(input, opts?)`: 입력 문자열에서 determistic ID 생성
     * `parse(id)`: ID를 파싱하고_components_ 반환
     * `validate(id)`: ID의 유효성을 검사
     * `createAliasMap(opts)`: token reduction을 위한 alias map 생성
     ### WORDLIST
     id-agent는 4096개의 워드를 포함하고 있는 워드 리스트를 제공해.
     ```javascript
     import { WORDLIST } from 'id-agent';
     ```
     ### Math Entropy
     id-agent의 엔트로피는 다음과 같이 계산될 수 있어.
     ```javascript
     Entropy per word = log2(4096) = 12 bits
     Total entropy = words * 12 bits
     ```
     ### Collision Probability
     id-agent의 충돌 확률은 다음과 같이 계산될 수 있어.
     ```javascript
     P(collision) ≈ n^2 / (2 * 2^b)
     ```
