### LLM이 되고보니 이런거라구요 ㅋㅋㅋㅋ

인공지능 모델이 어떻게 작동하는지 토큰화부터 예측까지의 전체과정을 설명하는 거임 ㅋㅋㅋ
#### 토큰화 
- **토큰화** : 문자열을 토큰으로 나누는 과정
- **토큰** : 어떤 텍스트를 나누었을 때 가장 작은 단위인 문자열
- **BPE** : 텍스트를 토큰화하는 알고리즘 중 하나로, 바이트 쌍 인코딩이라는 의미임
- **BPE 사용 이유** : 단어는 무한한 변형이 가능하여 토큰화하기 힘듬. 토큰화를 하기 위해 BPE를 사용하는 중

#### 언어 모델 학습
- **언어 모델** : 다음 단어를 예측하는 모델
- **학습 과정** : 토큰화된 문자열을 순차적으로 입력받아 다음 단어를 예측하는 방식으로 학습함
- **학습 목표** : 이전 단어에 대한 정보를 이용해 다음 단어를 정확하게 예측하는 것

#### 언어 모델 응용
- **대화** : 사용자와 대화하는 챗봇
- **문서 분류** : 문서의 주제를 분류하는 작업
- ** 기계 번역** : 다른 언어로 번역하는 작업

#### 모델 평가
- **모델 평가** : 모델의 성능을 평가하는 작업
- **평가 지표** : 모델의 성능을 측정하는 지표
- **평가 방법** : 개발된 모델에 대하여 테스트를 진행하고 성능을 확인

#### 모델 개선
- **모델 개선** : 모델의 성능을 높이는 작업
- **모델 개선 방법** : 모델의 구조를 변경하거나, 하이퍼파라미터를 조정하는 등 다양한 방법을 사용
- **모델 개선 목표** : 더 나은 성능을 가진 모델을 개발하는 것
