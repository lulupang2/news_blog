### 톨라리아란 뭘까?

톨라리아는 맥에서 마크다운 지식베이스를 관리하는 데스크탑 앱임. 사람들이 이것들을 다양한 용도로 사용하고 있어. SECOND BRAIN이나 개인 지식베이스를 관리하거나, 회사 문서를CONTEXT로 사용하거나, 개인적으로는 저도 이거 사용하고 있움. 엄청난 1만개가 넘는 노트를 가지고 있는데, 이건 제 리팩토링 작업과 개인적인 저널링, SECOND BRAIN 때문임.

### 주요 특징은?

* 파일 중심: 노트는 평범한 마크다운 파일들이니까, 편집기랑 별개로 동작하고, 내 데이터는 내 거임, 앱이랑은 상관없음.
* 깃허브 첫사랑: 모든 보트는 깃허브 레포지토리라서 버전 기록이 있고, 깃허브 리모트 사용 가능하고, 톨라리아 서버에 의존하지 않음.
* 오프라인 먼저: 계정이나 구독, 클라우드 의존 없이 완전 오프라인에서 작동하고, 톨라리아 사용 중지해도 데이터는 그대로 남아 있음.
* 오픈소스: 톨라리아는 무료이고 오픈소스임. 자기가 사용하려고 만들었지만 다른 사람과도 공유하고 싶음.
* 표준 기반: 노트는 마크다운 파일에 YAML 프론트머터 있음.专有 포맷 없이, 데이터가 잠기지 않게_standard tools랑 함께 동작함.

### 시작하는 법

최신 릴리즈를 다운받으면 되고, 처음 톨라리아 열면 가이드 볼트 클론하는 기회를 줌. 오픈소스고 로컬 설정도 지원함. 토리, 리액트, 타입스크립트로 만들어 짐. 로컬로 실행하거나 기여하려면 이 절차 따라하면 됨.

**설치 방법**

1. 노드js 20 이상 설치
2. pnpm 8 이상 설치
3. 러스트 안정 버전 설치
4. 맥OS에서 개발할 것

```bash
pnpm install
pnpm dev
```

그럼 브라우저 기반 모드 열면 되고, 네이티브 데스크탑 앱은 `pnpm tauri dev`로 실행하면 됨. 
기술文書는 [ARCHITECTURE.md](https://github.com/refactoringhq/tolaria/blob/main/ARCHITECTURE.md) 참고
