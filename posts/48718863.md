###요약 
   CUDA가 궁금하다? CUDA 프로그래밍을 해본 적 있냐? 
   CUDA 프로그래밍에서 커널이 런칭하면 어떤过程를 거쳐서 동작할까? 
   먼저, CUDA 프로그래밍 예시를 보고 넘어가도록 하장.
   ```cpp
   __global__ void vadd(const float* a, const float* b, float* c, int n) {
        int i = blockIdx.x * blockDim.x + threadIdx.x;
        if (i < n)
            c[i] = a[i] + b[i];
    }

    int main() {
        int n = 1 << 20; 
        size_t bytes = n * sizeof(float);
        float *a = (float*)malloc(bytes), *b = (float*)malloc(bytes), *c = (float*)malloc(bytes);
        for (int i = 0; i < n; i++) 
            a[i] = b[i] = 1.0f;
        float *da, *db, *dc;
        cudaMalloc(&da, bytes); 
        cudaMalloc(&db, bytes); 
        cudaMalloc(&dc, bytes);
        cudaMemcpy(da, a, bytes, cudaMemcpyHostToDevice);
        cudaMemcpy(db, b, bytes, cudaMemcpyHostToDevice);
        vadd<<<4096, 256>>>(da, db, dc, n); 
        cudaMemcpy(c, dc, bytes, cudaMemcpyDeviceToHost);
        printf("c[0]=%f c[n-1]=%f\n", c[0], c[n-1]);
    }
    ```
   CUDA 프로그래밍을 컴파일하는 과정, 커널의 런칭 과정, 커널이 동작하는 과정 등에 대해서 자세히 알려줌.
   그리고 CUDA 프로그래밍에서 사용하는 PTX, SASS, cubin 파일 등에 대해서도 소개함.
   따라서, CUDA 프로그래밍에 관심이 있거나, CUDA 프로그래밍을 đang انجام 중이라면 이 기사를 읽어보면 좋을 것 같음.
