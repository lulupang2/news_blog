### 마인크래프트 자바 에디션, GLFW 대신 SDL3 사용 시작
   마인크래프트 스냅샷 26.3 업데이트에서 라이브러리 업데이트가 이루어졌음. 
   GLFW 대신 SDL3를 사용하기 시작했대. 
   그리고 커스텀 연소 재료 관련 항목들을 추가했음. 
   또한 사인, 월드 생성, 전리품 테이블 등 다양한 기술적 변경 사항들이 적용됐대.
   그리고 개선된 항목들이 많음. 
   마인크래프트를 즐기는 사람들은 이 업데이트로 인해 게임이 더 안정적이고 다양한 기능을 사용할 수 있을 거 같다.
   ###Known Issues
   윈도우에서 풀스크린 모드를 사용하면 게임이 크래시가 나는 경우가 있다고 하니까 조심해야 할듯.
   ###New Features
   관전자 모드에서 포탈과 상호작용할 수 있게 됐대. 
   그리고 다양한 블록과 엔티티 관련 변경 사항들이 있대. 
   ###Changes
   마우스 입력이 상대적인 마우스 모드로 변경되고, 키 바인딩이 물리적 키를 사용하게 됐음.
   보더리스 풀스크린 모드가 기본으로 설정되고,전체 화면 모드를 전환할 때 게임을 재시작할 필요가 없게 됐대.
   디버그 오버레이 관련 사항도 개선된 거 같음.
   ###Technical Changes
   데이터 팩 버전과 리소스 팩 버전이 업데이트됨. 
   전리품 테이블 관련된 다양한 변경 사항들이 있대.
   윈도우와 입력 처리를 위한 라이브러리가 GLFW에서 SDL3로 변경된 거 같음.
   ###Environment Attributes
   새로운 환경 속성이 추가됨. 
   모브 스폰 관련 항목이 추가되고, 크리처 월드 제너레이션 관련 속성이 추가된 거 같음.
   ###Data Components
  新的 데이터 컴포넌트 추가됨. 
   마인크래프트에서 사용할 수 있는 연료와 다른 항목들의 컴포넌트들이 추가된 거 같음.
   ###Block Entity Data
   새로운 블록 엔티티 데이터 컴포넌트가 추가되어서 사인과 다른 블록 엔티티 관련 변경 사항들이 있대.
   마인크래프트를 하시는 모든 분들이 이번 업데이트로 인해 더 좋은 게임 경험을 하시기를 바래 ㅋㅋ
