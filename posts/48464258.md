### 개빡치는 점
앤드류 회사에서 새 modelo 나오면서 데이터 저장 정책 바뀜. 
Fable/Mythos 모델은 30일간 데이터 저장한대. 
원래 쓰던 모델은 그냥 걍 아무런 문제 없음.

### 데이터 저장되는 곳
이런 모델 쓰는 사람은 개발자, 기업체 등인데 
데이터는 클라우드에 저장되고 
기본적으로 30일간 저장된대. 
데이터가 자동으로 삭제되고 
제3자는 데이터에 절대 접근할 수 없다고 함.
안전하고 보안이 철저한 시스템이라고 nói해.

### 데이터 보호는 철저해
앤드류 회사 직원도 데이터 보는 거 금지야. 
데이터를 확인하려면 안전한 툴을 사용해서 
별다른 경우가 아니면 절대 데이터를 볼 수 없대. 
데이터 유출이나泄密 이런 거 절대 안 생기도록 
안전한 시스템을 유지하고 있다고 说해.

### 설정하는 방법
개발자 콘솔에서 설정하면 되고 
클라우드에도 설정이 가능하고 
별다른 설정이 필요 없는 경우도 있다고 함. 
설명서를 자세히 보면 어떻게 설정하는지 다 나와 있다고 nói해.

### 결론
요약하자면 개발자나 기업체 등이 
새 모델을 사용하려면 데이터 저장 정책을 변경해야 한대. 
안전하고 보안이 철처한 시스템이라고 说해. 
이제 개발자들이 새로운 모델을 사용해 볼 수 있겠네.
