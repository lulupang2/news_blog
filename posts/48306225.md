### 1994년 마이크로소프트 인턴 면접에서 받았던 4가지 프로그래밍 질문

컴퓨터, 업그레이드! 1994년 마이크로소프트 인턴 면접에서 받았던 질문을 소개하는데, 이중 두 개는 성능과 관련된 질문이었다는 점이 интерес롭다. 

8월 2일, 2023년에 작성된 이 글에서는 1994년에 마이크로소프트에서 인턴으로 근무할 때 받았던 4가지 프로그래밍 질문을 공유한다. 평소에는 성능에 Aware한 프로그래밍 수업을 하지만, 이번에는 다른 주제의 글을 게시할 계획이다. 

마이크로소프트 인턴으로 면접을 보았던 시절, 나는 4명의 사람과 면접을 보았는데, 각각이 나에게 클래식한 마이크로소프트 면접 질문을 던졌ㄷr. 그때インターネ트가 없었기 때문에, 이런 질문을 받을 줄은 전혀 몰랐구나. 

이 경험은 당시로서는 새로운 경험이었고, 지금 돌이켜보면 엄청난 재미였음ㅋㅋㅋ. 그 재미가 इतन해서, 아직까지도 기억이 남아있네. 

이번에는 이 4가지 질문과 당시 내가 구한 답, 그리고 오늘날에 와서 본다면 어떻게 달라질지에 대해 살펴보도록 하자. 각 질문을 일일이 소개하고, 해결책을 제시할 계획이다. 

### 질문 #1: Rectangle Copy

면접 과정은 질문이 점점 어려워지는 순서로 구성되었다는 점이 재미있네. 첫 번째 질문은 가장 쉬운 질문이었는데, C 코드로 하나의 버퍼에서 다른 버퍼로 사각형을 복사하는 코드를 작성하는 것이었다. 

사각형의 크기와 위치, 그리고 버퍼의 주소를 인자로 받는 함수를 하나 작성해야 했음. 예를 들어 다음과 같은 함수를 구현해야 했단 다. 
```c
void CopyRect(char *BufferA, int PitchA, char *BufferB, int PitchB, int FromMinX, int FromMinY, int FromMaxX, int FromMaxY, int ToMinX, int ToMinY) { /* 여기서 구현함 */ }
```
이 질문은 C/C++를 알고 있다면 정말简单하지만, 포인터를 다루는 것이 익숙하지 않다면 다소 어려울 수 있는 문제였을거임. 

### 질문 #2: String Copy

두 번째로 받았던 질문은 첫 번째 질문과 비슷했는데, 문자열을 복사하는 문제였다. 이 질문은 첫 번째 질문에 비해 더 쉬운 문제였지만, 첫 번째 질문을 풀었다면 이 또한 쉽게 풀 수 있었을텐데. 

이 문자열은 NULL로 끝나는 ASCII-Z 문자열이었음. 예를 들어 다음과 같은 함수를 구현해야 했단 다. 
```c
void CopyString(char *From, char *To) { /* 여기서 구현함 */ }
```
이 질문은 다소 이상하게 느껴졌는데, 특히 내가 성공적으로 코드를 작성한 후에 추가로 요구된 조건이 좀 이상했음ㅋㅋㅋ. 

### 질문 #3: Flood Fill Detection

세 번째 질문은 네 개의 질문 중에서 가장 интерес있는 문제였는데, 다소 어려운 문제였다. 이 문제는 마이크로소프트의 제품 중 하나에 구현된 플러드 필 알고리즘에서 나온 문제였다. 

플러드 필 알고리즘이 무엇인지 아는 사람은 이미 알고 있겠지만, 이 알고리즘은 이미지의 특정 좌표에서 시작하여 같은 색의 모든 픽셀을 다른 색으로 바꾸는 알고리즘이다. 

이 문제는 실제 플러드 필 알고리즘을 구현하는 것이 아니라, 특정한 바이트가 특정한 색과 일치하는지를检测하는 코드를 작성하는 것이었다. 예를 들어 다음과 같은 함수를 구현해야 했단 다. 
```c
int ContainsColor(char unsigned Pixel, char unsigned Color) { /* 여기서 구현함 */ }
```
이 문제는 다소 어려웠는데, 특히 각 바이트가 네 개의 픽셀을 포함하고 있었기 때문이다. 

### 질문 #4: Outlining A Circle

네 번째 질문은有点 Ridiculous한 질문이었다. 원을 그리기 위한 알고리즘을 이미 알고 있다면 쉽게 구현할 수 있었겠지만, 그렇지 않다면 구현하는 것은 거의 불가능에 가까웠다. 

이 문제는 원의 중심좌표와 반지름을 받고, 원의 외곽선을 따라 픽셀을 그리는 함수를 구현하는 것이었다. 예를 들어 다음과 같은 함수를 구현해야 했단 다. 
```c
void Plot(int X, int Y); /* 이미 구현된 플로트 함수 */
void OutlineCircle(int Cx, int Cy, int R) { /* 여기서 구현함 */ }
```
이 문제는 다소 어려웠지만, 특히 1994년당시에는 Floating Point를 사용하기에는 컴퓨터가 느렸기 때문에 정수로 계산해야만 했다는 점이 재미있네.
