# 신기한 신세계! 인공 뉴런 셀 자동으로 패턴 만들기
일단 이 기술은 인공 뉴런 셀 자동으로 패턴 만들기임. 예를 들어서 이미지나 3D 모델링 같은 걸 자동으로 만들 수 있음. 근데 문제点이 있는데, 이 기술은 해상도가 낮을 때는 잘 작동하는데, 해상도가 높아지면 너무 오래 걸리거나 컴퓨터 성능이 낮은 경우에는 작동을 안 함. 그래서 이 기술을 개선해서 높은 해상도에서도 잘 작동할 수 있게 만든 거야.

이 기술은 어떻게 작동하냐면, 일단 인공 뉴런 셀이랑 셀 자동으로 패턴 만들기를 사용하는데, 이때 셀은粗糙한 격자 위에서 작동함. 그리고 이 셀의 상태를 이용해서 이미지나 3D 모델링 같은 걸 만들 수 있음. 근데 이때 문제점이 있는데, 이 셀의 상태를 이용해서 이미지나 3D 모델링을 만들 때, слишком 많이 계산을 해야해서 컴퓨터 성능이 낮은 경우에는 작동을 안 함. 그래서 이 기술을 개선해서 높은 해상도에서도 잘 작동할 수 있게 만든 거야.

개선을 어떻게 했냐면, 일단 인공 뉴런 셀이랑 셀 자동으로 패턴 만들기를 사용하는데, 이때 셀의 상태를 이용해서 이미지를 만들 때, 이미지를 만들 때 필요한 정보만 계산을 하는 방식으로 만들었음. 그래서 컴퓨터 성능이 낮은 경우에서도 작동을 할 수 있음. 그리고 이 기술을 사용해서 높은 해상도의 이미지나 3D 모델링을 만들 수 있음. 그리고 이 기술은 다른 lĩnh vực에서도 쓸 수 있기 때문에, 아주 유용한 기술임. ㅋㅋㅋ

이 기술은 일단 EPFL과 구글 연구소에서 개발을 했음. 그리고 이 기술을 사용해서 높은 해상도의 이미지와 3D 모델링을 만들 수 있음. 그리고 이 기술은 다른 lĩnh vực에서도 쓸 수 있기 때문에, 아주 유용한 기술임. ㄹㅇ
