# 던전앤드래곤스 게임할 때 쓰는 thermal 프린터 유틸리티가 나오다니 ㅋㅋㅋㅋ
     
thermal 프린터는 대부분 작고 가벼워서 게임 테이블에 바로 두고 사용하기 좋음
     
주로 아이템 이름, 마법 이름, 몬스터 이름 등 게임에 필요한 텍스트를 출력하거나 플레이어의 캐릭터 시트를 바로 출력해서 사용할 수 있음
     
### 사용 예시
     
- 아이템 이름이나 마법 이름을 출력해서 게임에 사용
     
- 몬스터 이름이나 던전 이름을 출력해서 게임에 사용
     
- 캐릭터 시트를 출력해서 플레이어에게 배포
     
### 지원 환경
     
윈도우, 리눅스, 맥 등 대부분의 운영체제에서 작동함
     
ARM 기반이나 라즈베리파이에서도 작동함
     
### 사용 방법
     
1. thermal 프린터를 준비함
     
2. Sales & Dungeons 툴을 설치함
     
3. 템플릿을 만들거나 다운로드 받음
     
4. thermal 프린터로 출력함
     
### 템플릿 만들기
     
HTML과 CSS를 사용해서 원하는 템플릿을 만들 수 있음
     
Nunjucks 템플릿 엔진이나 자바스크립트를 사용해서 데이터를 삽입할 수 있음
     
### 기능
     
- 랜덤 생성기: 아이템 이름, 몬스터 이름, 던전 이름 등 랜덤으로 생성할 수 있음
     
- 게임 그리드: 게임을進行하면서 필요한 정보를 쉽게관리할 수 있음
     
- LLM 지원: AI를 사용해서 콘텐츠를 생성할 수 있음
     
### 오픈소스
     
Sales & Dungeons는 오픈소스 소프트웨어로 MIT 라이센스를 사용함
     
GitHub에서 소스코드를 확인하고 기여할 수 있음
