# 소개
   Grid2Poster는 전기 배분망 인프라를 OpenStreetMap 데이터에서 가져와 GeoPandas, OSMnx, Matplotlib를 이용해 프린트용 포스터를 생성해주는 도구임.
   국가나 대륙의 전송 선로를 다운로드해 렌더링하고, 다양한 테마와 형식을 지원함.
   
   # 기능
   * 각국 전기 배분망을 포스터로 만들 수 있음.
   * 국가, 주, 省, 대륙 단위로 지원함.
   * 사전 정의된 지역도 지원함.
   * 전송 선로와 분배 격자를 포함할 수 있음.
   * 다양한 테마와 형식 지원.
   * 로컬 GeoJSON 파일을 경계로 사용할 수 있음.
   
   # 사용법
   ```bash
   python create_grid_poster.py --country 한국 --tile-size-km 150
   ```
   
   # 옵션
   * --country: 국가 또는 지역 이름 (예: 한국, 미국, 유럽 등)
   * --boundary-geojson: 로컬 GeoJSON 파일 경로
   * --display-country: 포스터에 출력할 국가 이름
   * --subtitle: 포스터에 출력할 부제목
   * --padding: 경계에 대한 패딩 크기 (기본값: 0.10)
   * --theme: 테마 이름 (예: paper_grid, japanese_ink 등)
   * --list-themes: 사용 가능한 테마 목록 출력
   * --include-minor-lines: 부 전송 선로를 포함할지 여부 (기본값: Off)
   * --include-cables: 지하/해저 케이블을 포함할지 여부 (기본값: On)
   
   # 사전 정의된 지역
   * Continental Europe
   * Iberia
   * Latin America
   * Mediterranean
   * Middle East and North Africa
   * Southeast Asia
   * UK no Shetland
   * US Canada mainland
   * WAPP
   
   # 출처
   * OpenStreetMap 데이터 사용
   * MIT 라이선스 적용
