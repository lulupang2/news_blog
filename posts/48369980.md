시애틀에는 감시 카메라 같은 감시 기술들이 엄청 많음. 
그중에서 몇 개를 뽑아서 하나하나 설명해줄게.
 
### 감시 카메라
- **주소**: 시애틀 거의 대부분의 지역에 있음. 예를 들어, 523 Union St에도 있음.
- **외观**: 감시 카메라는 대부분 기둥이나 창고, 지붕 등 높은 곳에 설치되어 있음. 주차장, 문, 은행, 교차로, 정부 빌딩 등에 많이 설치되어 있음.indoors라면 대개 지붕이나 현금 등록기에 설치되어 있음. 
- **기능**: 감시 카메라는 비디오나 다른 데이터를 기록하고 저장할 수 있음. 또한 원격으로 제어할 수 있기 때문에 주변 환경을 관찰하기에도 좋음. 
- **작동 원리**: 카메라 녹화본은 분석을 통해 패턴을 찾아내고 공유할 수 있음. 인터넷이나 무선 주파수를 통해 네트워크에 연결되어 있기 때문에 비디오를 다른 곳으로 보낼 수 있고, 다른 사람들도 실시간으로 감시 카메라 영상을 볼 수 있음. 
- **사회적 중요성**: 감시 카메라는 사회적으로 규범을 강화하는 도구가 될 수 있음. 특정 행동이나 사람들을 '정상'으로 인식하는 방식으로 설계될 수 있는데, 이것이 네트워크를 통해 전파될 수 있음. 
- **토론**: 감시 카메라가 필요한지, 또는 감시가 전반적으로 필요한지 의문을 가질 수 있음. 감시가 없는 사회는 어떤 모습일까? 감시 카메라 대신 사람이 직접 감시한다면 어떻게 될까?
 
### 아마존 고
- **주소**: 2131 7th Ave
- **외观**: 普通의 편의점처럼 보이지만, 실제로는 그렇지 않음. 入口에서 아마존 앱을 스캔해야 들어갈 수 있음. 내부에는 현금 등록대가 없음. 아마존 계정의 QR 코드를 스캔해서 들어가야 함. 
- **기능**: 아마존 고는 고객이 매장에 얼마나 thời gian을 보내는지, 어떤 물건을 살펴보는지 등을 추적하고 기록할 수 있음. 
- **작동 원리**: 아마존 고는 고객의 구매 패턴을 분석하여 고객에 대한 정보를 수집할 수 있음. 예를 들어, 한카카스飾物을 산다면 유대인이 될 가능성이 높을 것이라고 추측할 수 있음. 또는 특정 건강 문제와 관련된 식품을 购買한 고객에게 특정 광고를 보여줄 수 있음. 
- **사회적 중요성**: 아마존 고의 패턴은有害할 수 있음. 예를 들어, 구글 포토가 흑인 사진을 '고릴라'로 표시한 적이 있음. 아마존의 데이터 수집과 사용에 대한監視와 투명성이 필요함. 
- **토론**: 아마존 고의 데이터 수집이 어떻게 사용될 지, 광고를 어떻게 표시할지에 대한 토론이 필요함. 아마존 고에 들어갈 때 실제로 무엇에 동의하는지, 어떻게 달라질 수 있을지에 대한 토론이 필요함.
