### 피루에타스(Piruetas) - 일기앱
개편한 일기앱을 소개함. 
기능은 아래와 같음.
* 하루一页
* 이미지 및 포맷 지원
* 기본적으로 비공개
* telemetry 지원 안함
* 일기 공유 지원
* 일기 내보내기 지원
* 이미지 드래그로 업로드 지원
* 라이트 모드 및 다크 모드 지원
* 테마 지원

### 셀프 호스팅
 셀프 호스팅 지원함
docker-compose.yml 아래와 같음
```yml
version: '3'
services:
  piruetas:
    image: forgejo.patilla.es/patillacode/piruetas:latest
    ports:
      - '8000:8000'
    volumes:
      - ./data:/data
    environment:
      # generate with: openssl rand -hex 32
      SECRET_KEY: change-me-to-a-random-string
      ADMIN_USERNAME: admin
      ADMIN_PASSWORD: changeme
      SECURE_COOKIES: 'false' # set to 'true' if serving over HTTPS
      REGISTRATION_OPEN: 'false' # set to 'true' to allow self-registration
      # TRUST_PROXY: 'true' # uncomment if behind a reverse proxy
    restart: unless-stopped
```
### 가격
* 무제한 일기 작성
* 이미지 업로드
* 데이터 내보내기
* 공유 링크
무료로 제공함. 
그리고 오픈소스임. 
기부 가능. 
개별 후원으로 지속 가능. ㅋㅋㅋ
