파이썬 3.15가 출시되기 전에 새로운 기능을 미리 살펴봄.
이번 버전에서 추가된 기능 중에_asyncio TaskGroup Cancellation_이란 게 있는데, 
이게 무슨 말인지 알아 보면 이 TaskGroup이란 파이썬에서 여러 개의 비동기 태스크를 쉽게 관리하게 해주는 도구임.
그런데 이 TaskGroup에서 태스크를 중단하기는 실제로 그렇게 간단하지가 않음. 
예를 들면interrupt라는 예외를 만들어서 TaskGroup에서 발생시키면 그 안의 태스크들은 자동으로 중단되는데, 
이 예외가 다른 예외들하고 엉키면서 복잡해질 가능성이 있음.
근데 파이썬 3.15부터는 TaskGroup.cancel이란 메소드를 새로 추가해서 태스크들이 쉽게 중단되도록 해줌.
이렇게하면 예외처리할 필요도 없이 태스크그룹을 쉽게 중단시킬 수 있음.

또 _Context Manager Improvements_가 있는데, 이것은 파이썬의 컨텍스트 관리자가 더 강력해졌단 뜻임.
파이썬에서 컨텍스트 관리자는 특정 코드블록의 실행을 제어하는 역할을 함.
컨텍스트 관리자는 기존에 decorator 처럼도 사용할 수 있었는데, 
이제는 async 함수, 제너레이터 함수에서도 정상적으로 동작하도록 개선되었음.

 그리고 _Thread Safe Iterators_를 추가했는데, 
이것은 멀티스레딩 환경에서 반복자(iterators)를 안전하게 사용할 수 있게 해줌.
기존에는 반복자들이 스레드에 안전하지 않았는데, 
이제는 threading.serialize_iterator를 사용하면 반복자를 안전하게 사용할 수 있게 되었음.

그리고 _Counter xor Operation_을 지원하게되었는데, 
이것은 collections.Counter 객체에 새로운 연산을 추가한 것으로,
이제 xor 연산을 지원해서 두개의 카운터를 xor 하여 새로운 카운터를 생성할 수 있게 되었음.

마지막으로 _Immutable JSON Objects_를 지원하게 되었는데, 
이것은 json.load 함수에 새로운 인자를 추가해서 immutable객체로 json을 로드할 수 있게 되었음.
이젠 frozendict를 사용해서 json객체를 불변객체로 만들 수 있게 되었고, 
이로써 파이썬의 json 지원이 더 강력해졌음.
