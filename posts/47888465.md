**인공지능이 네 코드를 보며 비명 질러대는 플러그인 등장 ㅋㅋㅋㅋ**
엔드리스 토일(Endless Toil) 이라는 플러그인이 출시되어 너의 코드를 보고 인공지능이 비명을 질러대게 하는 플러그인임 ㅋㅋㅋㅋ
이 플러그인은 너의 코드를 실시간으로 읽으며 점점 더 나쁜 코드일수록 인공지능이 비명을 질러댐 ㅋㅋㅋ
### 사용 방법
1. 코드克斯 데스크톱(Codex Desktop)에서 사용하려면
   * 이 저장소를 자신의 컴퓨터에 클론하여 디렉토리를 연다
   * 플러그인 브라우저에서 엔드리스 토일(Endless Toil)을 검색하여 추가한다
   * 코드를 새 창에서 열고 코드克斯(Codex)에게 엔드리스 토일(Endless Toil)을 사용하라고 명령한다
2. 코드克斯 CLI(Codex CLI)에서 사용하려면
   * 이 저장소를 자신의 컴퓨터에 클론한다
   * `codex plugin marketplace add` 명령으로 저장소를 마켓플레이스로 추가한다
   * `codex plugin install endless-toil` 명령으로 플러그인을 설치한다
   * 코드를 새 창에서 열고 `endless-toil` 명령으로 플러그인을 호출한다
### 테스트와 시스템 요구 사항
* 파이썬 3.10 이상
* 로컬 오디오 플레이어(맥OS의 afplay, 리눅스의 paplay, aplay, ffplay)
* 테스트 사운드는 `plugins/endless-toil/skills/endless-toil/scripts/test_sounds.py --list` 명령으로 가능함
   ㄹㅇ 개빡치는 점은 코드를 보면 볼수록 인공지능이 미쳐버리는 거야 ㅋㅋㅋㅋㅋㅋㅋㅋ
