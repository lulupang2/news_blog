# Bun-sqlgen 라이브러리
     라이브러리는 Bun.sql에서 사용하는 쿼리 자동완성 및 타입 검사를 제공해줌. 
     그냥 쿼리 이름만 적으면 자동으로 완성되는 거 보니까 개꿀임. 
     ## 예시
     ```javascript
     import { withTypes } from '@ilbertt/bun-sqlgen';
     import { SQL } from 'bun';
     ```
     쿼리 이름만 적으면 자동으로 완성되니까 쿼리 작성하기 너무 편리할 것 같음. 
     ### 사용법
     1. 라이브러리 설치: `bun add @ilbertt/bun-sqlgen`
     2. 쿼리 작성: ` sql.GetUser`SELECT id, email, display_name FROM users WHERE id = ${id}`; `
     3. 타입 검사: 그냥 tsc로 컴파일하면 됨
     ### 장점
     * 자동완성: 쿼리 이름만 적으면 자동으로 완성됨
     * 타입 검사: tsc로 컴파일하면 자동으로 타입검사 되서 컴파일 에러 발생함
     * 노.orm: orm 안써도 됨
