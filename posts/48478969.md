안트로피스새끼들이 새로운 AI 모델 패블을 내놓았는데, 이건 일반인들이 쓸 수 있는 버전이라고 함. 그런데 문제는 이 modèle에다가 감시 카메라도 장착했음. 패블이라는 AI에게 사이버 보안 관련된 요청을 하면, 이건 사이버 공격에 쓸 수 있을까봐 자동으로 막아버림. 
   사이버 보안 전문가들이 이게 진짜 문제라는 거임. 왜냐면, 패블이 사이버 보안 관련된 요청을 하면 자동으로 방어 메커니즘을 작동시켜서, 사용자에게 사이버 보안이나 바이오 관련된 주제라는 안내 메시지를 보냄. 
   이거 때문에, 사이버 보안 전문가들은 진짜 빡친 거임. 예를 들어, 패블한테 소프트웨어 엔지니어링 관련된 코드를 작성해달라고 요청해도, 이건 사이버 보안 관련된 요청이라고 판단해서 막아버림. 
   안트로피스새끼들은 이 모델을 더 좋은 방향으로 발전시킬 수 있을거라고 생각하는데, 지금은 그냥 감시 카메라가 되어버린 거임. 
   그리고 이 모델을 쓰려면, 사이버 보안 전문가들이 따로 승인받아야 하는 programu도 있음. 이 프로그램에 신청해서 승인받으면, 사이버 보안 관련된 작업을 할 수 있게 됨. 
   이런식으로 안트로피스새끼들이 패블 모델을 운영하고 있지만, 사이버 보안 전문가들은 이게 진짜 개빡치는 점이라는 거임.
