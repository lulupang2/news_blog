#軟體開發團隊的人類工程學 
   요즘 소프트웨어 개발팀 지원하는 새로운 방법 필요함. 이 책은 개발자와 소프트웨어 팀, 조직을 위한 비밀 무기를 제공할 것임. 
   최고의 엔지니어 조직과 세계各地의 수천명의 개발자로부터 수집된 엄격한 경험적 증거를 통해 강력한 원리들을 배울 수 있을 것임.
   SOFTWARE 개발을 위해 죽음의 행進을 해야만 하는 것은 아니야. 이 책은 SOFTWARE 팀이 기술 사업을 앞으로 밀어붙일 수 있는 행동과학의 변환 힘을 이해하는 인간적인 대안을 제시할 것임.
   SOFTWARE 개발팀에서 실패를 방어하고 협력과 문제 해결 문화를 구축하는 방법에 대한 강력한 원칙을 배우게 될 것임.
   이 책은 개발자와 리더들에게 SOFTWARE 공학의 'black box'를 이해하는 科學的 접근법을 제공할 것임.
   SOFTWARE 개발의 未來를 위한 새로운 방법을 제공할 것임.
   이 책은 SOFTWARE 팀을 위한 心理學的 모델을 제공할 것임.
   SOFTWARE 팀이 SOFTWARE 개발에서 人間性을 다시 부여하고 行動과학의力を 이용하여 더 즐거운, 혁신적인, 번영하는 환경을 창조하는 방법을 배울 수 있을 것임.
