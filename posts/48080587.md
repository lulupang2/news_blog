# ymawky란?
/arm 아셈블리 언어로 만들어진 웹서버임. 
macOSに 최적화되어 있지만, 리눅스나 유닉스에서 사용할 수 있도록 수정할 수 있음. 

## 주요 기능
요약하자면 이 웹서버는 다음과 같은 기능을 지원함:
* 1기가바이트까지의 파일 업로드를 지원
* HTTP 메소드(GET, PUT, DELETE, OPTIONS, HEAD) 지원
* MIME 타입 지원
* 에러 핸들링
* 보안 기능(예: slowloris 공격 대응)

## 설치 및 사용법
일단 Xcode Command Line Tools를 설치해야함. 
그리고 `make` 명령으로 빌드를 하고, `./ymawky` 명령으로 웹서버를 실행할 수 있음. 
`www/` 디렉토리에 html 파일을 넣으면, 웹서버에서 제공할 수 있음.

## 예시
`./ymawky` 명령으로 웹서버를 실행하고, `curl` 명령이나 웹 브라우저를 사용해서 테스트할 수 있음.
예를 들어, `curl http://127.0.0.1:8080/` 명령으로首页를 불러올 수 있음.

## MIME 타입
ymawky는 다양한 MIME 타입을 지원함. 
예를 들어, html 파일은 `text/html; charset=utf-8` 타입으로 제공됨.

## 에러 핸들링
ymawky는 커스텀 에러 페이지를 지원함. 
예를 들어, 404 에러 페이지는 `err/404.html` 파일로 제공될 수 있음.

## 보안
ymawky는 slowloris 공격과 같은 보안 위협에 대응하고 있음.
예를 들어, 요청이 10초 이상 걸리면 연결을 끊음.

## 결론
ymawky는 ARM 아셈블리 언어로 만들어진 웹서버로, macOS에 최적화되어 있지만 리눅스나 유닉스에서 사용할 수 있도록 수정할 수 있음. 
다양한 기능을 지원하고, 보안 기능을 포함하고 있기 때문에 개발자들에게 interessant함.
