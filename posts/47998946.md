# 투어 소개
Southwest 항공 본사에서 투어를 진행했어. 
LEAD Center(Leadership, Education, and Aircrew Development) 에서 시작했는데, 
여기서 Southwest 항공 승무원들이 교육을 받는다고 함. 
승무원들은 해상 구난, 비상 훈련, 소화기 사용, 자위 등을 연습하는데, 
매년 짧은 기간의 재교육을 받아서 실제 상황에 대응할 수 있게 되어 있음.

# 승무원 훈련
승무원들은 비상 시에 대처하는 방법을 배우고, 
100kg 짜리 구명함을 사용하는 훈련도 받음. 
승무원들은 수영을 잘할 필요가 없는데, 
구명함이 자동으로 확장되어 승무원을 구해주니까. 
소화기 훈련도 받고, 비상구문 데모도見는데, 
승무원들은 비상 시에 빠르게 대처할 수 있어야 해. 
승무원 유니폼도 보았는데, 
과거의 유니폼도 볼 수 있고, 
여러분도 볼 수 있음.

# 조종사 훈련
Southwest 항공의 조종사들은 산소 마스크를 8초 안에 하나씩 끼울 수 있어야 하는데, 
입마개는 할 수 있지만 Beard는 안된대 ㅋㅋㅋㅋ. 
여러분도 한번试해 봐. 
조종사들은 시뮬레이터에서 비행 훈련을 받는데, 
23개의 시뮬레이터가 있어. 
한개당 1백만 달러임. 
조종사들은 시뮬레이터에서 실제와 같은 훈련을 받으며, 
비상 시에 대처할 수 있는 способ을 배우고, 
여러분도 경험해 볼 수 있음.

# 점심과 Southwest Store
점심을 먹은 후에 Southwest Store에 갔는데, 
여기서 기념품을 살 수 있고, 
성인용 원피스를 покуп할 수 있음. 
여러분도 구매해 볼 수 있음.

# 사회 미디어 행운
투어가 끝난 후에, 
사회 미디어 전문가와 대화할 수 있었는데, 
여러분도 Southwest 항공의 따뜻한 이야기들을 볼 수 있음.

# 네트워크 작전 센터
네트워크 운영 센터에 갔는데, 
여기서 Southwest 항공의 모든 비행을 관리하는데, 
여러분도 이곳에서 어떻게 비행을 관리하는지 볼 수 있음.

# TechOps 헝거
TechOps 헝거에 갔는데, 
여기서 항공기의 유지 보수를 하는데, 
여러분도 이곳에서 항공기의 내부를 볼 수 있음.

# 헤르브와 콜린 오피스
투어가 끝난 후에, 
헤르브 켈러허와 콜린 버렛의 오피스를 방문했는데, 
여기서 두人の 업무를 볼 수 있고, 
여러분도 이곳에서 두人的 업무를 볼 수 있음.

# 리스닝 센터
리스닝 센터에 갔는데, 
여기서 社會 미디어를 관리하는데, 
여러분도 이곳에서 社會 미디어를 어떻게 관리하는지 볼 수 있음.
