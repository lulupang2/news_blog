### Sun Ray 서버 깔기
Sun Ray 서버 설정하기는 쉽지 않지만, 이글에서 알려드리겠습니다 ㅋㅋㅋ
1. **VM 설정** 
   -먼저 Proxmox VM에 OpenIndiana 2025.10을 설치해야 합니다.
   -VM 설정은 아래와 같습니다:
     * Install media: OpenIndiana Hipster 2025.10 Live DVD (64-bit x86)
     * Guest OS: Solaris Kernel
     * Machine: type q35, firmware as SeaBIOS
     * Graphic card: Standard VGA
     * Disks: 60GB, bus set to VirtIO Block, cache set to write back, discard turned on
     * CPU: model host, 1 socket, 4 cores
     * Memory: 8GB
     * Network: set model to VirtIO (paravirtualized)

2. **OI 설치** 
   -OpenIndiana를 설치합니다. 
   -GUI 설치 프로그램을 사용하여 설치할 수 있으며, 설치 중에 에러가 발생할 수 있으나, 무시하고 설치를続行할 수 있습니다.

3. **SRSS 패키지 설치** 
   -OI에 Sun Ray Server Software (SRSS)를 설치합니다. 
   -`pkg install sunray-essential` 명령으로 SRSS를 설치할 수 있습니다. 

4. **Sun Ray 서버 설정** 
   -Sun Ray 서버를 설정합니다.
   -`/opt/SUNWut/sbin/utconfig` 명령으로 초기 설정을 수행할 수 있습니다.

5. **SRSS를 실행** 
   -Sun Ray 서버를 시작합니다.
   -`/opt/SUNWut/sbin/utstart` 명령으로 SRSS를 실행할 수 있습니다.
 
### 결론
Sun Ray 서버를 설치하고 설정하는 것이 쉽지 않지만, 이글에서 알려드린步驟를 따르면 설치할 수 있습니다. 
또한, 이글에서 알려드린 설정을 통해 Sun Ray 서버를 성공적으로 설치하고 작동시킬 수 있을 것입니다. 
Sun Ray 서버 설정은 쉽지 않지만, 노력하면 할 수 있습니다.  ㄹㅇ
