# 고급 컴파일러 만들기, 혼자서도 할 수 있는 온라인 강좌
고급 컴파일러 만들기 강좌가 열렸음. 이건 코넬대 CS 교수인 어드리안 샘프슨이 하는 건데, 프로그래밍 언어 구현에 관한 거임. 중간 언어, 데이터 플로우, 클래식 최적화 같은 걸 배우고, 병렬화나 就時 컴파일, 가비지 컬렉션 같은 분야도 공부할 수 있음.

# 강의 내용
- 중간 언어랑 데이터 플로우, 최적화에 대해서 배움
- 병렬화, 就時 컴파일, 가비지 컬렉션 같은 최신 기술도 있음
- 강의는 영상과 노트, 코딩 과제로 구성되었음
- 코딩 과제는 모두 오픈소스고,_LLVM_이나 교육용 IR로 작업할 수 있음
- 각 강의는 비디오와 노트, 코딩 과제가 있고, 과제는 모두 오픈 엔딩임. 즉, 알아서 해보라는 뜻임 ㅋ

# 강의 목차
1. 소개 및 개요
2. 프로그램 표현
3. 지역 분석 및 최적화
4. 데이터 플로우
5. 글로벌 분석
6. 정적 단일 할당
7. _LLVM_ 소개
8. 루프 최적화
9. 인터프로시져 분석
10. 에일리어스 분석
11. 메모리 관리
12. 다이나믹 컴파일러
13. 동시성 및 병렬성
14. 빠른 컴파일러

# 강의 특징
- 모든 강의 자료는 오픈소스임
- 모든 과제는 오픈소스로 작업해야 함
- 강의는 비디오와 노트로 구성되어 있음
- 과제는 모두 오픈 엔딩임
- 강의는 自主학습이므로, 원할 때 언제든지 수강할 수 있음. 결국, 컴파일러 만들기는 혼자서도 할 수 있는 거임. 어д里안 샘프슨 교수가 하는 고급 컴파일러 만들기 강좌, 추천함!
