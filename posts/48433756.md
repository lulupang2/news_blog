# Lathe: LLM으로 튜토리얼을 생성해 학습하세요
Lathe는 LLM을 사용해 새로운 도메인을 학습할 수 있는 툴입니다.
튜토리얼을 생성하고, مباش히手を動かして 학습할 수 있어요.
Lathe를 사용하면,doctype 없이 튜토리얼을 생성하고, 관리하고, 학습할 수 있습니다.

## 설치하기
Lathe를 설치하기 위해서는 다음의 명령어를 실행하세요.
Homebrew를 사용하는 경우:
```bash
brew install devenjarvis/tap/lathe
```
Linux를 사용하는 경우:
```bash
curl -sSf https://raw.githubusercontent.com/devenjarvis/lathe/main/install.sh | sh\n```
Go를 사용하는 경우:
```go
go install github.com/devenjarvis/lathe@latest\n```

## 사용법
Lathe를 사용하기 위해서는 다음과 같은 명령어를 실행하세요.
```bash
lathe serve
```
그 다음, 브라우저를 열고, 튜토리얼을 생성하고, 관리하고, 학습할 수 있습니다.

## 튜토리얼 생성하기
Lathe를 사용하여 튜토리얼을 생성하기 위해서는 다음과 같은 명령어를 실행하세요.
```bash
/lathe build a 3D Slicer in Erlang
```
그 다음, 브라우저를 열고, 생성된 튜토리얼을 보실 수 있습니다.
