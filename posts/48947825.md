# ai 오픈소스 현재 상황
    모질라 기술 이사가 보낸 편지에 따르면 뉴질랜드 북쪽에서 마오리 방송국이 te reo 언어로 스피치 모델을 학습시켰다고 함. 이 언어는 시장에서 너무 작아서 아무도 건드리지 못하는데 오픈소스로 풀었대. pwc라는 회사는 오픈소스 모델을 기반으로 금융 언어 모델을 만들어서 현재 수백명의客户한테서 쓴다고 함. 스위스에는 공개 컨소시엄이 국가 모델을 만들어서 공개했데. 
    NONE이 이것들을 가능하게 했음. 이건 25년 전에도 일렀음. 모질라는 웹의 프론트 도어를 하나의 회사에서 가지고 있으려던 시도를 막기 위해 만들어졌고 지금도 똑같은 상황이 재현되고 있단 말임. 따라서 경쟁과 인터옵터빌리티에 베팅할거라 함.
    [오픈소스 ai 현재 상황]
    * 오픈소스 모델은 더 이상 양적 열세가 아님. 오픈소스 모델이 더 많아졌고, 상위 5개 모델은 모두 오픈소스 모델임. 
    * 오픈소스 모델은 코딩이나 지시 따르는 것, 일반적인 지식에서는 다른 모델과 비슷한 성능을 내지만, 推론이나 장기적 텍스트 처리 등에서는 조금 뒤쳐짐. 
    * 오픈소스 모델의 채택률은 79%로 높으며, 대부분의 개발자는 오픈소스 모델과 클로즈드 모델을 함께 사용하고 있음. 하지만 프로덕션에서는 오픈소스 모델이 약간 뒤처지는 편임.
    * 개발자들이 오픈소스 모델을 채택하는 이유는 비용, 성능, 유연성 등을 이유로 들어서 채택하는 편임.
    * 하지만 오픈소스 모델도 결국에는 운영하기가 어려운 편이고, 유지 관리비용이 많이 듦. 
    # ai 오픈소스 스택
    * ai 오픈소스는 9개의 레이어와 48개의 컴포넌트로 구성되어 있고, 각각의 컴포넌트는 10개의 기준으로 평가하고 있음.
    * 오픈소스 스택은 기능 면에서는 силь하지만, 운영면에서는 약함. 
    * 운영적인 면에서 약함을 보완하기 위해서는 표준화와 기업 준비가 필요함. 
    # 누가 오픈소스에 베팅할까?
    * 오픈소스는 비즈니스 모델로 가능함. 
    * 여러 회사가 오픈소스에 베팅하고 있고, 수익을 내고 있음. 
    * 펀딩도 많이 받았고, 기업 가치도 높음. 
    * 대표적인 회사로는 데이터브릭스, 미스트랄, 딥시크 등이 있음. 
    따라서 ai 오픈소스 현재 상황은 점점 더 강력해지고 있고, 많은 사람이 오픈소스에 베팅하고 있단 말이야. 개좋은 상황임.
