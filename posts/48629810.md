### SQL로 통계 내는 신박한 방법
DuckDB 확장자 the-stats-duck v0.6.0이 출시됨. 이름은 i-m-not-dead( 죽지않았어 ) ㅋㅋㅋ. the-stats-duck은 Bedevere와 KoliLang에서 사용하는 통계 엔진임. 
DuckDB에서 직접 통계를 내는 것을 가능하게 해주고, MIT 라이센스로 오픈소스임. 브라우저에서 사용할 수도 있음. Palmer Penguins 데이터셋을 사용해 본 결과는 다음과 같음:
```sql
SELECT * FROM 'penguins';
```
데이터셋을 프로파일링하기 위해 `meta()` 함수를 사용할 수 있음:
```sql
SELECT column_name, kind, n_missing, n_distinct, mean, median, stddev, top 
FROM meta('penguins');
```
이 함수는 각 열에 대한 정보를 반환함. 
회귀 분석을 수행할 수도 있음:
```sql
SELECT * 
FROM lm_summary('penguins', formula := 'body_mass_g ~ flipper_length_mm + bill_length_mm');
```
bootstrap() 함수를 통해 부트스트랩 샘플링을 수행할 수도 있음:
```sql
WITH b AS (
  SELECT bootstrap(body_mass_g, 'mean', 2000, 42) AS samples 
  FROM penguins
) 
SELECT list_aggregate(samples, 'quantile_cont', 0.025) AS lo, 
       list_aggregate(samples, 'quantile_cont', 0.975) AS hi 
FROM b;
```
VISUALIZE문을 통해 다양한 그래프를 생성할 수도 있음:
```sql
VISUALIZE species AS x, body_mass_g AS y 
FROM penguins 
DRAW violin 
TITLE 'Body Mass (g) by species';
```
v0.6.0에서는 violin marks, 2D facets, per-layer STAT smooth가 추가됨. 
예를 들어, scatter plot에 fitted line을 추가할 수 있음:
```sql
VISUALIZE bill_depth_mm AS x, bill_length_mm AS y, species AS color 
FROM penguins 
DRAW point 
DRAW line 
STAT smooth 
SCALE x ZERO false 
SCALE x LABEL 'Bill Depth (mm)' 
SCALE y ZERO false 
SCALE y LABEL 'Bill Length (mm)' 
TITLE 'Penguins Bill Depth VS Bill Length (Smooth)';
```
개별 열에 대한 분포도 생성할 수 있음:
```sql
WITH pois AS (
  SELECT k, dpois(k, 3) AS pmf 
  FROM range(0, 11) AS t(k)
) 
VISUALIZE k AS x, pmf AS y 
FROM pois 
DRAW bar;
```
v0.6.0에서 read_stat의 성능이 개선됨. 기존에는 O(N²)이었지만, 이제는 파일을 한 번만 파싱하고, 버퍼링되는 열 저장소에서 데이터를 스트리밍함. 
예를 들어, 200k 행, 7MB의 XPT 파일을 읽는 시간은 67초에서 1.3초로 줄어듦. 
the-stats-duck는 오픈소스이며, github.com/KoliStat/the-stats-duck에서 확인할 수 있음.
