# 오픈노리지
오픈노리지란, 로컬 우선의 아름다운 마크다운 에디터이자 llm 위키에 클라우드, 코덱스 등의 통합 지원이 가능한 에디터임. 맥앱이나 웹앱/클라이언트로 리눅스, 윈도우, 인텔 맥에서 사용할 수 있당.
## 기능
* fullWidth wysiwyg 지원으로 마크다운 파일을编辑할 때 구글 독스나 노션 페이지 처럼 편집할 수 있게 해줌.
* 클라우드, 코덱스, 커서 데스크탑 앱을통해 협업 ai 편집 지원
* mcp/클라이언트를 통해任何 Harness/AGENT 와 사용 가능
* llm 위키, 에이전트 두뇌, spec 주도 개발을 위한 mcp, 스킬, 에이전트 검색 지원
* 깃/Github 을통해 노코드 팀シェア, 오토싱크 지원 
* [공식문서](https://openknowledge.ai/docs) 참고
## 설치
* 맥 : 데스크탑 앱 다운로드 후 dmz 열고 오픈노리즈를 어플리케이션으로 이동 후 런치 ㄱㄱ
* 최신 릴리즈 확인
* 리눅스, 윈도우, 인텔 맥 : Node.js 24 이상에서 로컬 웹앱으로 실행 가능 (npm install -g @inkeep/open-knowledge)
* cd 프로젝트폴더
* ok init : 프로젝트 초기화 및 설정
* ok start --open : 웹 에디터 실행 및 브라우저 열기
