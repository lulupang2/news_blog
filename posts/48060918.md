### 지오json이 뭔지 알아봄
지오json은 지도에 다양한 데이터 구조를 담을 수 있는 형식이야. 예를 들면
```json
{
  "type": "Feature",
  "geometry": {
    "type": "Point",
    "coordinates": [125.6, 10.1]
  },
  "properties": {
    "name": "디나가트 제도"
  }
}
```
이런식으로 데이터를 담을 수 있고, 지오json에서는 포인트, 라인 스트링, 폴리곤, 멀티 포인트, 멀티 라인 스트링, 멀티 폴리곤 등의 기하학적인 타입을 지원함. 그리고 기하학적인 객체에 추가적인 속성을 가진 것을 피쳐 객체라고 하고, 피쳐를 모아놓은 것을 피쳐 컬렉션 객체라고 함. 지오json은 2015년에 인터넷 엔지니어링 태스크 포스(IETF)에서 표준화했음. 그 결과 2016년 8월에 RFC 7946이라는 새로운 표준을 발표했지. 이제는 이 표준이 기존의 2008년 지오json 규격을 대체하는 중임. 개빡치는 점은 이제 지오json을 사용하면 지도에 대한 데이터를 쉽게 다룰 수 있다고 하는거임. ㅋㅋㅋ
