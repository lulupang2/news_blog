# Bootimus: PXE 부팅 서버 구축 꿀팁!

부트시스템 구축이 너무 귀찮았지? 하나하나 설정하고, 구성하는게 thật로 개빡치는 점임. 근데Bootimus라는 툴을 사용하면 그런 귀찮은 부분들을 모두 해결해 줌. 
Bootimus는 하나의 바이너리 파일로 PXE 부팅 서버를 쉽게 구축할 수 있는 툴임. 설정 파일 없이, 설정 변경 بدون 런타임 종속성으로 쉽게 부팅서버를 만들 수 있음. 
Bootimus는 자동으로 50개 이상의 리눅스 배포판을 감지하고, 프록시 DHCP를 지원해 줌. 또한 REST API를 통해 부팅 할당, 스캔, WOL 트리거 등을 스크립트로 쉽게 제어할 수 있음. 

## 주요 기능

*   하나의 바이너리로 PXE 부팅 서버 구축 가능
*   설정 파일 없이, 런타임 종속성 없이 작동
*   자동으로 50개 이상의 리눅스 배포판 감지
*   프록시 DHCP 지원
*   MAC 기반 ACL 지원
*   JWT 및 LDAP 토큰 인증 지원
*   REST API 지원

## 활용 예

*   PXE 부팅 서버 구축
*   부팅 할당 및 스캔 자동화
*   WOL 트리거 및自動화
*   부팅 서버 모니터링 및 로그 기록

## 사용법

*   Docker를 사용하여 Bootimus를 실행
*   Bootimus를 사용하여 PXE 부팅 서버 구축
*   REST API를 사용하여 부팅 할당, 스캔, WOL 트리거 등을 제어
