**최신 인공지능 모델을 현실세계에서 돌리기**
==============================================

요즘 클라우드 서비스 이용하다 보니 돈도 많이 나가고 속도도 느리고 불안정한 경우가 많음. 그래서 직접 컴퓨터를 조립해서 최신 인공지능 모델을 돌려보겠다는 생각이 듦. 

### 하드웨어 구성

*   **CPU:** AMD EPYC Milan 7313P
*   **RAM:** 128GB DDR4 ECC RDIMM
*   **GPU:** 4* NVIDIA RTX PRO 6000
*   **PCIe Switch:** c-payne Microchip Switchtec PM40100
*   **-boot NVMe:** 4TB M.2

이런 구성으로 최신 인공지능 모델을 돌릴 수 있다고 함. 

### 소프트웨어 구성

*   **Docker:** 각 모델별로 Docker 컨테이너를 만들어서 실행
*   ** модели:** 다양한 모델들을_cached_ 해놓고 사용

이렇게 해서 최신 인텔리전스를 현실세계에서 이용할 수 있음. 

### 후기

*   직접 컴퓨터를 조립해서 사용하는 것보다 클라우드 서비스를 이용하는 것이 편리함.
*   클라우드 서비스 이용하다 보니 비용도 많이 나가고 속도도 느리고 불안정한 경우가 많음.
*   직접 컴퓨터를 조립해서 사용하는 것은 편리하지는 않지만, 비용도 적게 들고 속도도 빠르고 안정적인 경우가 많음.

### 결론

 직접 컴퓨터를 조립해서 최신 인텔리전스를 현실세계에서 이용하는 것은 좋은 선택지가 될 수 있음. 다만, 클라우드 서비스를 이용하는 것보다 더 많은 시간과 돈을 투자해야 하지만, 더 많은 편리함과 안정성을 제공할 수 있음.
