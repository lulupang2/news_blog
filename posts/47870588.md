# 데이터 센터? 못 쓴다고? Monterey Park에서 일어난 일 
 Monterey Park 동네에データ 센터를 짓려는 개발자들이 있었습니다. 250,000 平方 フィート 규모의 데이터 센터를 500 피트 거리에다가 만들어 보려고 했는데요. 근데 동네 주민들이 이걸 알게 되면서 바로 행동을 시작했어요. 

 ## 먼저 데이터 센터는 왜 안되냐고? 
SGV Progressive Action이라는 단체에서 организ해 주었는데요. 그들은 '데이터 센터는 환경에 영향을 준다는 거 임. 특히Noise랑 공기 오염이 심해'라고 설명했어요..backup용 발전기가 14개인데 1년에 디젤 200,000 갤런을 태우면서 60db 이상의 소음이 나고, 지진이나 재해 시에 비상용 발전기가 동시에 가동되면서 더 심해질거라고 했는데요.Residents들은 이를 막기 위해 노력했어요. 

 ## 데이터 센터 반대 캠페인을 시작 
 'No Data Center MPK'라는 사이트를 만들어서 데이터 센터를 반대하는 운동을 시작했어요. 지역 주민들을 대상으로 door-to-door 캠페인과 전화, 전단지를 통해 알리며, 150명의 주민이 참여한 teach-in도 열었고 3D프린터로 데이터 센터 모델을 만들어 보여주며 주민들의 참여를 이끌어냈어요. 
 결국 4,500명의 주민이 서명하여 데이터 센터를 반대하는 서명을 모았고, 개발자가 데이터 센터计划을 철회한 걸로 알려졌습니다. 

 ## 데이터 센터 반대 캠페인 결과 
 개발자가 데이터 센터计划을 철회하고, 시의회에서 데이터 센터를 금지하는 결의안을 채택했습니다. 시의회는 데이터 센터에 반대하는 주민들의 목소리에 응답하여 데이터 센터를建設하지 않겠다고 선언했어요. 

 ## 그 후에는? 
주민들은 계속해서 데이터 센터 반대 운동을 진행하고 있으며, 주변 지역에서도 데이터 센터를 반대하는运动이 펼쳐지고 있어요. 주민들은 데이터 센터가 환경과 지역 사회에 미치는 부정적인 영향을막기 위해 끊임없이 노력하고 있呢
