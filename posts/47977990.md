# AI 이미지 생성에 새로운 기술 등장!
   AI가 이미지 안의 글자와 숫자를 제대로 넣어주는 기술이 등장했음. 이 기술을 사용하면 AI가 이미지 안에 정확한 텍스트와 숫자를 넣을 수 있게 됨.
   
   ## 문제점
   기존의 AI 이미지 생성 기술은 이미지 안의 글자와 숫자를 제대로 넣지 못하는 경우가 많았음. 특히, 복잡한 이미지 안의 텍스트와 숫자를 제대로 넣는 것이 어려웠음.
   
   ## 새로운 기술
  새로운 기술은 '언더드로잉'이라고 불리는데, 이미지 안의 텍스트와 숫자를 먼저 정밀하게 위치시키고, 이후에 이미지 생성 모델이 이를 완성하는 방식임. 이 기술을 사용하면 이미지 안의 텍스트와 숫자를 정확하게 생성할 수 있게 됨.
   
   ### 예시
   예를 들어, 다음과 같은 이미지를 생성하고 싶다고 가정해 봄. 클레이 애니메이션 형태의 보드게임에서 1부터 50까지의 숫자가 적힌 50개의 돌이 있는 이미지를 생성하고 싶어. 기존의 AI 이미지 생성 기술로는 이러한 이미지를 생성하기 어려웠지만, 언더드로잉 기술을 사용하면 쉽게 생성할 수 있게 됨.
   
   ### 과정
   1. 먼저, SVG 형태의 언더드로잉을 생성하여 이미지 안의 텍스트와 숫자를 위치시킴.
   2. 이후, 이미지 생성 모델에 언더드로잉과 함께 이미지 생성 요청을 함.
   3. 이미지 생성 모델이 언더드로잉을 바탕으로 이미지 생성을 완료함.
   
   ### 결과
   언더드로잉 기술을 사용하면 이미지 안의 텍스트와 숫자를 정확하게 생성할 수 있게 됨. 이는 다양한 분야에서 이미지 생성을 사용하는 경우에 유용하게 적용할 수 있음.
   
   ### 마무리
   요약하자면, AI 이미지 생성에 새로운 기술이 등장했는데, 이미지 안의 텍스트와 숫자를 제대로 넣는 기술이름. 이 기술을 사용하면 다양한 분야에서 이미지 생성을 더 쉽고 효율적으로 사용할 수 있게 될 것임. ㅋㅋㅋ
