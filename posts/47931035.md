### 소개
GTFOBins는 리눅스에서 사용할 수 있는 명령어를 모아놓은 목록으로, 이것을 이용해서 다양한 작업을 할 수 있음. 예를 들어 쉘 탈출, 권한 상승, 파일 전송, 셸 생성 등이 가능함.

### 기능
GTFOBins에는 다양한 명령어가 포함되어 있는데, 여기에는 7z, aa-exec, ab, acr, agetty, alpine 등이 포함되어 있음. 각 명령어는 특정 작업을 수행할 수 있도록 설계되어 있으며, 이를 이용해서 다양한 목적을 달성할 수 있음.

### 사용 예시
예를 들어, 7z 명령어를 사용해서 파일을 압축할 수 있음. 또는 aa-exec 명령어를 사용해서 쉘을 생성할 수 있음. 또한, ab 명령어를 사용해서 파일을 업로드하거나 다운로드할 수 있음.
