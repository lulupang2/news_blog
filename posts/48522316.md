# 펄린 노이즈 필드를 활용한 창의적인 그래픽 제작

펄린 노이즈 필드는 2D 포스 필드에 기반한 알고리즘으로, 이를 통해 멋있는 그래픽을 만들 수 있음. 필자는 이 알고리즘을 사용해 25가지 서로 다른 그래픽을 만들어봄.

## 펄린 노이즈 필드란?
펄린 노이즈 필드는 각 점이 방향과 크기를 갖는 2D 포스 필드. 이를 통해 움직이는 오브젝트를 시뮬레이션할 수 있음. 예를 들어 랜덤한 방향과 크기를 갖는 포스 필드를 생성하면 다음과 같은 그래픽을 만들 수 있음.

## 구현
펄린 노이즈 필드를 구현하기 위해 몇 가지 클래스를 생성했음. 이를 통해 빠르게 아이디어를 테스트하고, 그래픽을 생성할 수 있음.

## 결과
25가지 서로 다른 그래픽을 만들어봄. 이를 통해 펄린 노이즈 필드의 다양한 가능성을 확인할 수 있었음.
