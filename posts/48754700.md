### 크롬에서 <usermedia> 태그 떴다! 
   이제는 사용자 미디어 접근 쉽게 해결할 수 있어. 
   크롬 개발자 블로그에서 새로운 <usermedia> 태그에 대한 글을 포스팅했어. 
   이 태그는 사용자 미디어 접근을 쉽게 해주고, 보안을 강화하며, 권한 거부로 인한 문제를 해결해줘. 
   ### <usermedia> 태그의 기능 
   * 사용자 미디어 접근을 쉽게 해줘 
   * 보안을 강화해줘 
   * 권한 거부로 인한 문제를 해결해줘 
   ### 사용 방법 
   * HTML에 <usermedia> 태그를 추가해 
   * 자바스크립트에서 setConstraints() 메서드를 사용해 하드웨어 요구 사항을 설정해 
   * 스트림 이벤트를监听해 
   ### <usermedia> 태그의 속성 
   * stream: 사용자 미디어 스트림을 제공해 
   * setConstraints(): 하드웨어 요구 사항을 설정해 
   * error: 오류 발생 시 오류 정보를 제공해 
   ### 사용 예 
   ```html
   <usermedia id="media-ctrl">
       <button>카메라와 마이크 활성화</button>
   </usermedia>
   ```
   ```javascript
   const el = document.getElementById('media-ctrl'); 
   el.setConstraints({ 
       video: { 
           width: 1280, 
           height: 720 
       }, 
       audio: { 
           echoCancellation: true 
       } 
   }); 
   el.addEventListener('stream', () => { 
       videoPreview.srcObject = el.stream; 
   }); 
   ```
   ### 결론 
   <usermedia> 태그는 사용자 미디어 접근을 쉽게 해주고, 보안을 강화하며, 권한 거부로 인한 문제를 해결해주는 태그야. 
   이제는 이 태그를 사용해서 더 좋은 사용자 경험을 제공할 수 있을거 같아.
