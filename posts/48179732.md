# 프로젝트 글래스윙: MITOS 프리뷰는 뭘 보여줬나
MITOS 프리뷰를 이용한 시험 결과를 공유하려고 함. 이번에는 우리에게서 실제 보안 취약점을 찾는 데 도움이 될 만한 다양한 보안 관련 LLM을 테스트해봄. 이 LLM은 우리 시스템에 잠재적인 취약점을 식별해주고, 어떻게 고칠 수 있는지도 알려줌. 그리고 공격자가 최신 모델을 이용해서 어떻게 공격할 수 있는지도 보여줌.
MITOS 프리뷰는 Anthropic에서 만든 거고,(Project Glasswing) 프로젝트에서 이용한 모델임. 이 모델은 기존의 일반적인 보안 스캐너보다 훨씬 더 좋은 결과를 보여줌.
MITOS 프리뷰는 실제로 우리 코드에서 몇 가지严重한 취약점을 찾아줬고, 이를 이용해서 어떻게 공격할 수 있는지도 보여줌.
근데 여기서 문제가 하나 생기더라. 모델이 어떤 요청에 대해서는 거부를 하더라는 거임. 예를 들어 어떤 취약점을 研究하려고 하면 모델이 거부하거나, 다른 요청이랑 같은 요청을 했는데 다른 결과가 나온다는 거임.
이거 때문에 모델이 실제로 안전한지 확인하기가 힘들어. 모델이 안전하기 위해서는 추가적인 보안 기능을搭載해야 한다는 거임.
그리고 모델이 실제로 취약점을 찾았을 때, 이게 실제로 공격 가능한지 확인하는 과정이 필요해. 이 과정을 자동화하기 위해서는 모델이 더 발전해야 한다는 거임.
현재는 모델이 찾아준 취약점을 실제로 공격할 수 있는지 확인하기 위해 사람이 개입해야 하지만, 모델이 더 발전하면 자동으로 공격 가능한지 확인할 수 있을거임.
