# 오픈와프 소개
   오픈와프는 내가 원하는 AI 모델을自由롭게 쓸 수 있게 해주는 오픈소스 프로젝트야. 워프 위에 BYOP(自带提供商) 능력을 추가해서 genai 적응계층 원生으로 6种 API協議를 지원하고, 사용자 정의 모델 및 시스템 힌트 단어, 원生 다중 언어를 지원해줘.
   # 핵심 기능
   * 사용자 정의 제공업체: 6种 API協議를 원生으로 지원해서 OpenAI, Anthropic, Gemini, Ollama 등과 같은 모델을 내가 원하는대로 쓸 수 있게 해줘.
   * 다중 언어 지원: 원生으로 다중 언어를 지원해서 내가 원하는 언어로 AI 모델을 쓸 수 있게 해줘.
   * 시스템 힌트 단어 지원: 사용자 정의 시스템 힌트 단어를 지원해서 내가 원하는 시스템 힌트 단어로 AI 모델을 쓸 수 있게 해줘.
   * 로컬 저장소: 모든 사용자 데이터를 로컬에 저장해서 내가 원하는대로 데이터를 관리할 수 있게 해줘.
   # 사용 방법
   1. git clone -b openWarp https://github.com/zerx-lab/warp
   2. cargo build --release
   3. ./target/release/openwarp
   #_FAQ
   * 오픈와프와 워프 공식의 관계: 오픈와프는 워프의 오픈소스 코드를 기반으로 하는 커뮤니티 분기점으로, 워프 공식 회사와는 아무런 관련이 없으며, 위의Dual AGPL/MIT 라이센스를 따르도록 해.
   * API 키 업로드: 업로드 안 돼. 사용자 정의 제공업체 자격 증명은 로컬 구성 파일에만 저장되고, OpenWarp에서 직접 바탕 URL과 통신하니까, 중간에 아무것도 안 지나감.
   * 지원 모델 제공업체: OpenWarp는 genai 다중 프로토콜 적응계층을 지원해서, OpenAI / OpenAI 응답 / Anthropic / Gemini / Ollama / DeepSeek 등 6개의 프로토콜을 원生으로 지원해줘. openAI 호환 포인트(Qwen / Groq / together / openrouter / SiliconFlow / LM Studio 등)를 openAI 프로토콜로 선택하고 바탕 URL을 입력해서 접속할 수 있게 해줘.
   * 워프 업스트림 업데이트 계속 받을 수 있나?: 워프 업스트림 메인 라인을 계속병합해서, 사용자 경험을 유지하면서, BYOP와 다중 언어를 강화할 수 있게 해줘.
