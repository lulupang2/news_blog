> **🔗 원문 보러가기:** [Don't put aria-label on generic elements like divs](https://www.matuzo.at/blog/2026/aria-label-generic-elements)

웹 접근성에 관해서 얘기좀 하자. 요즘 웹 개발하면서 접근성 고려하기 진짜 중요함. 
   그래서 오늘은 `Don't put aria-label on generic elements like divs` 이라는 글이랑 관련된 얘기를 좀 풀어보겠음. 
   이 글은 웹 접근성에 관한 몇 가지 실수와 그에 관한 해결 방법에 관해서 다루고 있음. 
   제목에서 이미 많이 나왔지만 div나 span 같은 요소에 aria-label을 사용하지 말라는 이야기임. 
   ```html
   <div aria-label="News"> … </div>
   ``` 
   이런식으로 사용하지 말라는거임. 왜냐하면 ARIA 규격을 보면 generic 요소에 이름을 붙일 수 없다는规則이 있기 때문임. 
   generic 요소에는 div, span 같은거가 포함되어있음. 
   그리고 실제로 여러 브라우저와 스크린 리더에서 테스트해봤는데 결과가 다 달라서 좀 놀람. 
   ```html
   <div aria-label="News">Content</div>
   <p> Some inline <span aria-label="News">content</span> </p>
   <ge-neric aria-label="News"> Content </ge-neric>
   ``` 
   이런식으로 테스트했는데 voiceover, talkback, narrator 등에서 announcement이 다르게 나옴. 
   | 스크린 리더 | 브라우저 | announcement |
   | --- | --- | --- |
   | VoiceOver | macOS Safari 26.3.1 | News, group |
   | VoiceOver | iOS Safari 26.3 | Content |
   | Talkback | Android Chrome 148 | News |
   | Talkback | Android Firefox 150 | Content |
   | Jaws | Windows 11 Chrome 148 | Content |
   | NVDA | Windows 11 Chrome 148 | Content |
   | NVDA | Windows 11 Firefox 150 | Content |
   | Narrator | Windows 11 Chrome 148 | News, group, content |
   이렇게 결과가 다르는데 Empty 요소로 테스트해도 결과가 다름. 
   ```html
   <div aria-label="News" style="width:100px; height:100px"></div>
   <div aria-label="News"></div>
   ``` 
   이런식으로 테스트했는데 voiceover, jaws, narrator, talkback 등에서 다르게 annoncement함. 
   | 스크린 리더 | 브라우저 | announcement |
   | --- | --- | --- |
   | VoiceOver | macOS Safari 26.3.1 | News, empty group |
   | VoiceOver | iOS Safari 26.3 | None |
   | Talkback | Android Chrome 148 | News |
   | Talkback | Android Firefox 150 | None |
   | Jaws | Windows 11 Chrome 148 | None |
   | NVDA | Windows 11 Chrome 148 | News |
   | NVDA | Windows 11 Firefox 150 | News |
   | Narrator | Windows 11 Chrome 148 | None |
   개빡치는 점은 이렇게 결과가 다르다는 점이고, 요약하자면 div에 aria-label 사용하지 말라는 얘기임. 
   하지만 section 같은 경우는 예외적으로_generic 요소에 aria-label 사용해도됨. 
   ```html
   <section aria-label="News"> … </section>
   ``` 
   이런식으로 사용해도됨. 
   그리고 popover 요소도 예외적으로 generic 요소에 aria-label 사용해도됨. 
   ```html
   <div popover aria-label="I'm not sure how useful a label here is"> </div>
   ``` 
   결론은 div에 aria-label 사용하지 말라는 이야기임. 진짜 웹 접근성을 고려해서 개발해야함.
