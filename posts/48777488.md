### 비대칭 펜스란?
비대칭 펜스는 동시성 프로그래밍에서 사용되는 하나의 기술임. 비대칭 펜스를 사용하면 동시성 프로그래밍에서 발생하는 문제를 해결할 수 있고, 성능 또한 개선할 수 있움. 
비대칭 펜스는 두 가지 종류가 있으니, 하나는 비대칭 라이트 펜스고, 다른 하나는 비대칭 헤비 펜스임. 
비대칭 라이트 펜스는 컴파일러 최적화를 방지하는 것이 주 목적이고, 비대칭 헤비 펜스는 메모리 장벽을 생성하는 것이 주 목적임. 
### 비대칭 라이트 펜스
비대칭 라이트 펜스는 컴파일러 최적화를 방지하는 것이 주 목적임. 비대칭 라이트 펜스는 컴파일러가 코드를 재배열하여 성능을 최적화하지 못하도록 방지함. 
비대칭 라이트 펜스는 `asm volatile`를 사용하여 구현되는데, `asm volatile`는 컴파일러가 코드를 재배열하지 못하도록 방지하는 어셈블리 명령어임. 
비대칭 라이트 펜스는 다음과 같이 구현될 수 있음:
```c
asm volatile("" : : : "memory");
```
이 코드는 컴파일러가 코드를 재배열하지 못하도록 방지하고, 메모리 장벽을 생성하지는 않음. 
### 비대칭 헤비 펜스
비대칭 헤비 펜스는 메모리 장벽을 생성하는 것이 주 목적임. 비대칭 헤비 펜스는 `membarrier` 시스템콜을 사용하여 구현되는데, `membarrier` 시스템콜은 메모리 장벽을 생성하는 시스템콜 임. 
비대칭 헤비 펜스는 다음과 같이 구현될 수 있음:
```c
int syscall(SYS_membarrier, int cmd, unsigned int flags, int cpu_id);
```
이 코드는 메모리 장벽을 생성하고, 비대칭 라이트 펜스와 함께 사용하여 비대칭 펜스를 구현할 수 있음. 
### 결론
비대칭 펜스는 동시성 프로그래밍에서 사용되는 하나의 기술이며, 비대칭 라이트 펜스와 비대칭 헤비 펜스가 있음. 
비대칭 라이트 펜스는 컴파일러 최적화를 방지하는 것이 주 목적이고, 비대칭 헤비 펜스는 메모리 장벽을 생성하는 것이 주 목적임. 
비대칭 펜스를 사용하면 동시성 프로그래밍에서 발생하는 문제를 해결할 수 있고, 성능 또한 개선할 수 있기 때문에, 개발자들이 사용하여 보다 효율적인 코드를 작성할 수 있을거임.
