# Introduction 
.whatcable 앱은 usb-c 케이블 성능 查看하는 macOS 앱 임
 macOS 앱이니까 ios 쓰는 븅딱들은 안된다는거 암 ㅋㅋ
 
 ## 기능 
 * usb-c 케이블 연결했을 때 무슨 기능 있는지 바로 알려줌
 * Charging 속도도 측정 가능함
 * 연결된 기기의 정보도 보여줌
 *_PD Discover Identity_ 지원함
 
 ## 사용 방법 
 1. Releases 페이지에서 최신 버전을 다운받고
 2. 압축 해제 후 WhatCable.app를 Applications 폴더로 이동하면 끝임
 
 ## 코드 분석 
 * 앱은 IOKit 서비스를 사용해 usb-c 케이블의 정보를 가져옴
 * IOKit 서비스는 macOS에서 제공하는 프로그래밍 인터페이스임
 * WhatCable 앱은 이 프로그래밍 인터페이스를 사용해 케이블 정보를 가져옴
 
 ## 기타 
 * WhatCable 앱은 Swift로 작성되어 있음
 * macOS 14(Sonoma) 이상에서 사용 가능함
 * Apple silicone + Intel 모두 지원함
 *MIT 라이센스로 개방되어 있음
 
 ### Caveats 
 * cable e-marker 정보는 cable에 따라 다를 수 있다함
 * PD spec coverage는 PD 3.0 / 3.1 까지 지원함
 * Vendor name lookup은 일부 제조사만 지원함
