### 浮動小수점 깡똘하게 이해하기
   ###### 개요
   일상생활에서 쓰이는 浮動小数점은 어떻게 작동할까? 
   사실 浮動小수점의 작동 원리는 간단함.
   일단 시작부터 핵심 원리는 2진법과 과학적 표기법을 이해하는 것임.
   ### 2진법과 과학적 표기법
   2진법은 0과 1로만 구성되기 때문에 너무 간단하다는 점에서 착안할 수 있음.
   하지만 이는 컴퓨터의 기본 단위이기도 하기 때문에 매우 중요함.
   먼저 10진법에서 쓰이는 과학적 표기법을 살펴봄.
   327.849를 예로 들어보면 
   3 * 10^2 + 2 * 10^1 + 7 * 10^0 + 8 * 10^-1 + 4 * 10^-2 + 9 * 10^-3
   이렇게 표현할 수 있음.
   하지만 이렇게 표현하면 여러가지 문제점이 있기 때문에 과학적 표기법을 씀.
   3.27849 * 10^2로 표현하면 위에 말한 문제점들이 사라짐.
   10진법과 마찬가지로 2진법도 과학적 표기법을 사용할 수 있음.
   1001.0101을 예로 들어보면 
   1 * 2^3 + 0 * 2^2 + 0 * 2^1 + 1 * 2^0 + 0 * 2^-1 + 1 * 2^-2 + 0 * 2^-3 + 1 * 2^-4
   이렇게 표현할 수 있음.
   과학적 표기법을 쓰면 
   1.0010101 * 2^3으로 표현할 수 있음.
   ### 浮動小数점의 원리
   浮動小수점은 위에서 설명한 2진법의 과학적 표기법에 몇가지 제약을 가한 것임.
   significand와 exponent로 구성되며 
   cada tipo de 浮動小数점마다 significand와 exponent의 크기가 다름.
   예를 들어 float의 경우 significand는 24bit, exponent는 8bit임.
   이러한 제약으로 인해有些 decimal 숫자들은 정확하게 표현할 수 없음.
   ### 浮動小수점의 encoding
   이제 浮動小수점을 binary로 encoding하는 방법을 알아봄.
   encoding 과정은 간단함.
   1. sign bit를 추가함.
   2. exponent를 bias를 가해 encoding함.
   3. significand를 encoding함.
   마지막으로 encoding된 binary를 확인하는 방법도 알아봄.
   ### special value
   float에서 쓰이는 spécial한 value들에 대해 알아봄.
   float의 exponent range는 254개이기 때문에 2개의 biased exponent 값이 남음.
   이러한 biased exponent 값을 special value로 사용함.
   ### 결론
   浮動小수점의 원리는 간단하다는 점에서 착안할 수 있음.
   2진법과 과학적 표기법을 이해하고 몇가지 제약을 가하면 浮動小수점을 이해할 수 있음.
   이러한 understanding으로 인해 浮動小수점이 어떻게 작동하는지 알 수 있음.
