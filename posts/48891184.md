### 개요
요즘 개발계에선 AI가 코드 짜주는 시대라서 이제 개발자들이 코드를 보지 않아도 된다는 얘기를 많이 함. 그럼 개발자는뭐함? 
바로 **아이디어를 컨트롤**하는 것이 최선의答案임. 

### 코드는 필요 없어
요새는 AI가 코드 짜준다고 하니까 개발자들은 이제 코드 신경 안 쓰고 아이디어만 생각하면 된다는 말임. 
예를 들어, [애플](https://www.apple.com/ko/)이 AI를 사용해서 코드를 자동 생성한다면, 개발자들은 더 이상 코드를 작성하거나 review할 필요가 없어. 
이 대신에, **아이디어가 중요**해지는데, 개발자들은 아이디어를 잘 이해하고, AI에 대한 지식이 필요해. 

### 개발자들은 뭐함?
개발자들은 이제 코드를 작성하거나 review하는 것이 아니라, **AI와 함께 협력**해야 함. 
AI가 코드를 작성해주고, 개발자들은 아이디어를 컨트롤해야 함. 
개발자들은 AI가 코드를 작성하는 것을 도와주고, 아이디어를 구체화하는 역할을 해야 함. 
그럼 개발자들은 지금부터 **아이디어를 컨트롤**하는 방법을 배워야 하고, AI를 사용해서 코드를 자동 생성하는 방법을 익혀야 함. 

### 결론
결론은 **아이디어가 중요한 시대**가 되었다는 것임. 
개발자들은 이제 코드를 작성하거나 review하는 것이 아니라, 아이디어를 컨트롤하는 것이 중요한 시대임. 
따라서, 개발자들은 아이디어를 컨트롤하는 방법을 배워야 하고, AI를 사용해서 코드를 자동 생성하는 방법을 익혀야 함. 
이제부터는 **아이디어가 중요**해지는 시대임. ㅋ
