# 키키란 무엇인가요?

   키키는 [том토마](http://tomotama.com/) 디자인 철학으로 만들어진 홈피 구축 키트야.looks wrong, 직접 고칠 수 있어서 튜토리얼 보면서 고치거나 하는 일 없음 ㅋ
 
   데모 사이트에서 테스트 해볼 수도 있고, 여기 [tomotama.com](http://tomotama.com/)도 키키로 구축했어! 下载하고 구매는 [itch.io](https://itch.io/)에서 가능하다구! 

   # 키키가 khác다고?

   소스 코드가わず 1500행으로 되어있으며(<50kb)人間が 읽어서 이해할수도 있을 정도로 작다는 점이 핵심임. 그럼 무슨 기능이 있을까? ㅋㅋ

   * 내장 5가지 테마 (responsvie 하다고 함)

   * 공개 위키 모드

   * 라이브/다이나믹 사이트 모드

   * 정적 사이트 생성 모드

   * 고퍼 홀에 대한 예비 지원

   * 간단한 마크업 언어 '버그'

   * 마크다운 지원

   * 보조/화면리더 친화적 출력

   * RSS, HTML 생성

   * 풍부한 코멘트와 수정가능한 소스

   * 인터랙티브 도움말과 문서

   * 다이나믹, 스테틱 페이지 생성

   * 100% 핸드메이드 PHP, HTML, CSS

   # 키키에 없는것

   자바스크립트도 없고, 외부 라이브러리/의존성도 없고 쿠키/tracking data도 없고 100킬로바이트도 안됨. ML생성 쓰레기 코드도 없고 데이타베이스 구조도 없고 설치해야 할게 없고 소셜미디어 쓰레기도 없고.self-업데이트도 없다고함 ㅋㅋ

   # 라이센스

   키키는 쉐어웨어야! 비영리 용途에는 무료고 만 15불이야. 풀 버젼은 5가지 모든 테마, 내장 마크다운 지원, 퍼블릭 위키 모드 지원, 그리고 푸터에있는 쉐어웨어 링크를 없애도 된다고함. 돈 안넣으면 푸터에 쉐어웨어 링크를 놔두면 된대 ㅋ

   소스코드는원본 수정없이 공유 가능하고, 비즈니스용으로 게임 회사가서 사이트 빌드해도 사용은 가능하지만 판매/배포는 안된다고함. 요약하면, 키키는 아주 작지만 강력하고, 수정하기 쉽고, 비용도 저렴해서ใคร나 쉽게 홈페이지를 만들 수 있는 툴임.
