# 라이브러리 소개
lib0xc는 C 언어의 표준 라이브러리를 보완하여 더 안전한 시스템 프로그래밍을 가능하게 하는 라이브러리입니다.

## 목표
- C 언어를 더 안전하게 사용하는 것이 목표입니다. 하지만 C 언어 자체의 제약으로 인하여 완전히 안전한 언어로 만들 수는 없습니다.
- 그래서 lib0xc는 C 언어의 사용 빈도 수를 고려하여 사용 빈도가 높은 부분을 안전하게 만들었습니다.

## 주요機能
- **familiar, Easy to Adopt**: lib0xc의 API는_standard_라이브러리의 함수 이름과 동일하게 설계되어 있습니다. 그래서 기존 코드에서 lib0xc의 함수로 쉽게 바꾸어 쓸 수 있습니다.
- **Embrace Static Bounds**: lib0xc의 함수 대부분은 메모리 할당을 필요로 하지 않습니다. 대신에 고정 크기의 구조체 또는 배열을 사용하여 코드를 더 안전하게 만듭니다.
- **Support -fbounds-safety**: clang 컴파일러의 bounds safety 확장 기능을 지원합니다. 이 기능을 사용하면 메모리 상의 데이터 접근 RANGE를 확인하여 런타임 에러를 줄일 수 있습니다.
- **Codify, Document, Test**: lib0xc는 업계에서 사용하는 패턴을 정리하여 문서화하고 테스트 코드를 함께 제공합니다. 그래서 사용자들에게 안정적인 코드 작성 환경을 제공합니다.

## 주요 구성 요소
- **standard library extensions**: C 언어의 표준 라이브러리를 확장한 함수들입니다.
- **Systems Programming Utilities**: 시스템 프로그래밍에 사용되는 유틸리티 함수들입니다.

## 사용 방법
- 빌드 방법: `make lib` 명령어를 사용하여 라이브러리를 빌드할 수 있습니다.
- 테스트 방법: `make test` 명령어를 사용하여 테스트를 실행할 수 있습니다.
