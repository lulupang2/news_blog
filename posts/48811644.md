## Ternlight 이란 모델
   Ternlight 이라는 모델이 떴는데, 브라우저에서 돌아가는 7MB 임베딩 모델이라고 함. 
   얘는 WASM로 동작하는데, cpu에서 0초만에 동작한다고 함 ㅋ 
   ## 모델 성능
   * 임베딩 latency가(ms) 단위라고하는데, single embed() 호출시 @ternlight/mini가 있다고함 
   * Linear weights ternary (BitLinear) Transformer layers도 지원되고, 
   * Attention heads, d_model, Output, Max input 이런것도 있음 
   * 모델은 MIT라이센스로 공개되었음. 
   ## 특징
   * 첫 로딩이 끝나고 캐시하면 네트워크 사용 안 함 
   * Runs on @ternlight/mini라고함. 
   ## 참고
   이 모델은 정말로 브라우저에서 돌아감. 브라우저에서 돌리는 거자나, 모델 크기도 7MB 밖에 않아서 용량 걱정도 없고, cpu에서도 잘 돌아감. 
   ㄹㅇ 재밌네. 이렇게 브라우저에서 돌아가는 모델은 많지 않은데, 이런것도 나왔군.
