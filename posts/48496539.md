# 인공지능이 프로젝트 만들기
개쩌는 프로젝트 아이디어가 있는데, 혼자서 하기엔 돈이 많이 들거나 너무 어려울 때가 있음. 이럴 때 쓰는 게 FablePool이라는 거 ㅋㅋㅋ

## 어떻게 동작하냐면
- 먼저, 큰 프로젝트 아이디어를 내놓음. 예를 들어, 애플 같은 거랑 경쟁할 수 있는 새로운 운영체제를 만들거나, 완전히 오픈ソース인 아마존 웹 서비스를 만드는 거야. 이때, 프로젝트당 최소 모금 목표가 100달러야 함.
- 사람들이 0.25달러 이상任意 금액으로 기부 가능. 기부하면, 프로젝트 진행 상황을 실시간으로 볼 수 있음.
- 프로젝트마다 미화 100달러 이상의 목표금액이 있고, 사람들은 이 목표에 동-money를 맡길 수 있음.  

## 현재 진행중인 프로젝트
- 펠리벤치-자전거 만들기(14.50달러 모금중, 185달러 목표): 야쉬 타플리얄이 만드는 프로젝트
- C#에서 쓰레기 수집 개선하기(14달러 모금중, 200달러 목표): 基思가 만드는 프로젝트
- 오픈소스 PID 튜닝 파이썬 라이브러리(5달러 모금중, 152달러 목표): 매튜 배러스가 만드는 프로젝트
- 기술 메시아(4달러 모금중, 740달러 목표): 그즈가 만드는 프로젝트
- 완전히 오픈소스인 AWS 만들기(1.25달러 모금중, 516달러 목표): 데이비드 호프가 만드는 프로젝트
- 에어버스(1달러 모금중, 670달러 목표): 다니엘 주이딩가가 만드는 프로젝트
- 영국 차량 튜닝 데이터베이스(1달러 모금중, 702달러 목표): 케빈 피에로니가 만드는 프로젝트
- 오픈소스 тур보퍼퍼 스타일 오브젝트 스토리지 네이티브 검색 데이터베이스(0.50달러 모금중): 크리스 스톤즈가 만드는 거임

## 지원대기 중인 프로젝트
- 오픈소스 제이콥 스콜리(2004년 비디오 게임) 구현(0.25달러 모금중): E M이 만드는 프로젝트
- 그랜드 테프트 오토 7 만들기(모금 0달러): 브라이언 베스트가 만드는 프로젝트
- 페이블 6 만들기(모금 0달러): 찰스 레잉이 만드는 프로젝트
- 구식 옷가게에 새로운 조명을 설치하기(모금 0달러): 저스틴 개스킨이 만드는 프로젝트
- 미래의 부는 공유되는 부다(모금 0달러): api 본즈독이 만드는 프로젝트
- vscode git 히트맵 확장(모금 0달러): 매드프롭스가 만드는 프로젝트
- 지식이 없는 나쁜 회사들이 모두를 속이는 게임 만들기(모금 0달러): 카터 스미스가 만드는 프로젝트
- 영국 대중 투표 시스템(모금 0달러): 크리스 스톤즈가 만드는 프로젝트

이런 프로젝트들이 있고, 지원하면 인공지능이 직접 만들어나가는 걸 볼 수 있음. 정말 재밌고 신기한 경험임 ㅋㅋ
