### 클라우드플레어에서 에이전트 계좌 만들고 도메인 구매까지 자동화
에이전트들이 클라우드플레어에서 계좌 만들고 도메인 구매하는 기능이 추가됨. 개발자들이 클라우드플레어에 코드 배포할 때 더 편해질듯

 Stripe과 클라우드플레어가 손잡고 에이전트가 사용자 대신 계좌 만들고 도메인 구매하는 기능을 지원하기 시작했음. 이제 에이전트가 클라우드플레어에서 계좌 만들고 도메인 구매하고 코드 배포까지 다 자동화 할 수 있게 되었음.

 에이전트가 사용자 대신 계좌 만들고 도메인 구매하고 코드 배포할 수 있는 기능은 클라우드플레어와 스트라이프가共同으로 설계한 새로운 프로토콜을 기반으로 작동함. 이 프로토콜은 에이전트가 사용자 대신 계좌 만들고 도메인 구매하고 코드 배포할 수 있게 하면서도, 사용자의 승인을 받는 동시에 보안을 유지하는 방식으로 설계됨.

 이제 에이전트가 클라우드플레어에서 계좌 만들고 도메인 구매하고 코드 배포할 수 있는功能이 추가되어, 개발자들이 코드 배포할 때 더 편해질 것으로 기대됨
