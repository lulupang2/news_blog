### NVIDIA 그래픽 카드 VRAM을 리눅스 스왑 공간으로 쓰는 꿀팁

리눅스를 쓰는 사람들이라면 다 아는 사실인데, 램이 모자라면 스왑 공간을 이용해야 해.근데 스왑 공간은 보통 HDD나 SSD를 이용하는데, 이 방법은 엄청 느려. 
그래서 오늘 소개할 소프트웨어는 NVIDIA 그래픽 카드의 VRAM을 리눅스 스왑 공간으로 이용할 수 있게 해주는 **nbd-vram**임. 
이 소프트웨어는 특히 램 업그레이드가 불가능한 노트북을 위한 솔루션으로, NVIDIA RTX 카드와 같은 VRAM容量이 큰 그래픽 카드를 가진 사용자를 위함.

### 사용 방법

우선 [nbd-vram의 깃허브 페이지](https://github.com/c0dejedi/nbd-vram)에서 소프트웨어를 다운받아야 해. 
그다음에는 터미널에서 아래 명령어를 이용해서 소프트웨어를 설치하고启動하면 됨.

```bash
git clone https://github.com/c0dejedi/nbd-vram
cd nbd-vram
sudo ./install.sh
sudo systemctl start vram-swap-nbd
```

그리고 아래 명령어로 스왑 공간이 제대로 설정되었는지 확인할 수 있음.

```bash
swapon --show
```

### 설정 변경

설정을 변경하려면 `/etc/systemd/system/vram-swap-nbd.service` 파일을 수정하고, 아래 명령어를 이용해서 시스템을 재시작해야 해.

```bash
sudo systemctl daemon-reload
sudo systemctl restart vram-swap-nbd
```

### 성능 비교

성능은 RTX 3070 LAPTOP을 이용해서 테스트했는데,sequential throughput는 **1.3 GB/s**였음. 
이는 NVMe보다 더 빠른속도인데, 그 이유는 데이터가 스왑 공간으로 이동할 때 PCIe를 이용하기 때문임.

### 결론
요약하자면, nbd-vram은 NVIDIA 그래픽 카드의 VRAM을 리눅스 스왑 공간으로 이용할 수 있게 해주는 소프트웨어임. 
 램 업그레이드가 불가능한 노트북을 쓰는 사용자들에게 좋은 솔루션이 될 것 같음.
