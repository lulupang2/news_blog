# HP 16C 리바이벌판 나왔다!
HP 16C 리바이벌판은 35년 전의 전설적인 프로그래머 계산기를 리메이크한 것임. 
원래 모델의 스마트한 디자인과 파워를 유지하면서 현대 사용자를 위한 업그레이드된 기능을 추가한 거임.
엔지니어, 프로그래머, 열정가는 사람들은 똑같은直관적인 레이아웃과 고급 기능을 좋아할 거야. 
新增된 기능으로는 빠른 성능과 업그레이드된 프로그래밍 기능이 있지.
**구매하기**: 
* 유럽 구매는 [여기](링크)를 点击해
* 미국 구매는 [여기](링크)를 点击해
* 국제 구매는 [여기](링크)를 点击해
**기본 정보**:
* 용도: 프로그래밍, 컴퓨터 과학, 논리 설계, 엔지니어링
* 엔트리 시스템: RPN(Reverse Polish Notation)
* 키보드:_numeric, 헥스, 십진수, 8진수, 이진수 전용 키 있음
* 고급 기능: 정수 연산, 비트 연산, 논리 테스트, 기수 변환, 워드 크기 제어, 부동 소수점 연산, 키 스트로크 프로그래밍, 조건 분기, 서브루틴, 플래그
* 메모리 레지스터: 99개
* 프로그램 메모리: 203바이트
* 프로그램 저장/불러오기 기능 새로 추가됨
* 전원 공급: 2 x CR2032 배터리
* 자동 전원 끄기: 5분
* 디스플레이: 1줄 x 10문자, 7세그먼트 LCD
* 크기: 129 x 82 x 15mm
* 무게: 116그램(배터리 포함)
이 리바이벌판은 원래 모델의 디자인과 기능을 유지하면서 사용자에게 더 편리한 사용 환경을 제공하는 거임. 
이제는 헥스, 십진수, 8진수, 이진수를 자유롭게 전환해서 비트 연산, 논리 테스트, 정수 연산 등을 할 수 있지.
프로그래밍에도 강화된 기능들이 새로 추가되어서conditional branching, subroutines, labels, flags를 지원해줌.
프로그램 메모리는 203바이트까지 지원이 가능하고 데이터 레지스터는 99개나 있어 활용도도 상당히 높아졌음.
게다가 이제는 프로그램을 저장하고 불러올 수도 있으니 프로용으로도 쓸만하다 이거지.
