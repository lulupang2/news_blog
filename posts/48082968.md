## 스페이스 캐뎃 핀볼의 리턴
와 이런 게임이 리눅스에서 돌아간다는 거จริง이냐? ㅋㅋㅋ
나는 윈도우 XP 쓰던 시절에 이 게임 엄청 재미있게 했음. 솔리테어나 프리셀, 하츠, 미스스위퍼는 재미없었지만 핀볼은 너무나 매료시킴. 
이 게임은 윈도우 XP에 기본으로 포함되어 있었고, 나도 많이 玩했었어. 
이 게임을 다시 리눅스에서 돌려보는 방법이 있길래 소개해 봄.

## 스페이스 キャ덜 핀볼을 리눅스에서 설치하는 방법
리눅스에서 스페이스 캐뎃 핀볼을 돌리려면 Flatpak을 사용하면 편함. 
Flatpak을 사용하면 게임을 쉽게 설치하고 실행할 수 있음. 
터미널에 아래 명령어를 입력해서 설치하면 됨: 
```bash
flatpak install com.github.k4zmu2a.spacecadetpinball
```
## 높은 해상도와 풀 틸트 데이터 그래픽
이 게임은 예전 윈도우 XP 시절에 나온 게임이라 그래픽이좀 구림. 
하지만 풀 틸트! 핀볼이라고 하는 게임의 데이터를 사용하면 1024x768 해상도로 게임을 즐길 수 있음. 
이 데이터를 사용하려면 archive.org에서 풀 틸트! 핀볼 게임 데이터를 다운받아야 함. 
그리고 데이터를 다운 받은 후에 데이터 디렉토리에 압축을 풀어주면 됨:
```bash
cd ~/.var/app/com.github.k4zmu2a.spacecadetpinball/data/SpaceCadetPinball
unzip ~/다운로드/CADET.ZIP
```
그리고 나서 기존의 데이터 디렉토리를 삭제해주면 됨:
```bash
sudo rm -r $(flatpak info --show-location com.github.k4zmu2a.spacecadetpinball)/files/extra/Pinball
```
이렇게 해주면 높은 해상도와 풀 틸트 데이터 그래픽으로 게임을 즐길 수 있음.

그리고 풀 틸트! 핀볼 게임 데이터를 사용하면 게임의 규칙이 조금 달라지는데, 레이턴 레인을 만들기 더 쉬워짐.
