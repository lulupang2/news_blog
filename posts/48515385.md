### 파카(Paca) 소개
파카는 인간과 인공지능이 협력하여 프로젝트를 관리할 수 있는 플랫폼임. 지라, 트렐로, 클릭업, 먼데이 같은 도구들은 있지만, 파카는 인간과 인공지능이 함께 일할 수 있는 가볍고 오픈소스인 점이 달림. 
파카의 핵심은 인공지능이 단순히 작업을 자동화하는 것이 아니라, 인간과 함께 협력하여 새로운 가치를 창출하는 것임. 
#### 파카의 주요 기능
*   **유니파이드 스크럼반 보드**:人类와 인공지능이 동일한 보드를 공유하여 협력할 수 있음.
*   **인앱 인공지능 채팅**: 프로젝트 레벨에서 인공지능 에이전트와 채팅하여 작업을 계획할 수 있음.
*   **액티비티 디프 및 리버트**: 모든 필드 변경 내용을 시각적으로 확인하고 이전 버전으로 되돌릴 수 있음.
*   **BDD colaboration**: 인간과 인공지능이 함께 BDD 시나리오를 작성할 수 있음.
*   **시스템 디자인 문서**: 인간과 인공지능이 함께 시스템 디자인 문서를 작성하고 유지할 수 있음.
*   **MCP 서버**: 인공지능 에이전트를 연결하여 구조화된 데이터에 접근할 수 있음.

#### 시작 방법
파卡를 시작하는 방법은 세 가지가 있음:
1.  **인터랙티브 설치 스크립트**: 생산环境을 위한 추천 방법임. 리눅스 서버에서 도커를 사용하여 시작할 수 있음.
2.  **도커 컴포즈**: 수동으로 설정할 수 있음. 환경 파일을 생성하고 도커 컴포즈를 사용하여 시작할 수 있음.
3.  **로컬 개발**: 로컬 개발환경을 위한 방법임. 저장소를 클론하고 도커 컴포즈를 사용하여 시작할 수 있음.

#### 커스텀마이징
파카는 가볍고 확장 가능함. 플러그인 시스템을 사용하여 기능을 추가하거나 수정할 수 있음. 플러그인은 백엔드와 프런트엔드에서 작동하며, 샌드박스 환경에서 실행되어 보안을 제공함.
#### MCP 서버
MCP 서버를 사용하여 인공지능 에이전트를 연결할 수 있음. Claude desktop과 같은 클라이언트를 사용하여 파카의 도구와 상호 작용할 수 있음.
#### 환경 변수
Environment 변수를 사용하여 MCP 서버를 설정할 수 있음. PACA_API_KEY, PACA_API_URL 등이 있음.
#### 도움말
파카의 문서와 가이드를 참고하여 자세한 정보를 확인할 수 있음. 로컬 개발, 도커 컴포즈, MCP 서버 등에 대한 정보가 있음.
