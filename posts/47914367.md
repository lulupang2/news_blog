### AI도 기억력을 상실할 수 있다?! 

요즘 AI 기술은 발전했지만, 기억력이 부족한 편임. 왜냐면 세션이 매번 초기화되어서 기존에 배웠던 정보를 기억하지 못하거든. 하지만 최근에 'YourMemory'라는 기술이 등장해서 AI의 기억력을 강화할 수 있게 됨. 

### YourMemory란?

YourMemory는 인공지능이 인간처럼 기억력을 가질 수 있게 해주는 기술임. 에빙하우스 망각곡선을 적용해서 중요도에 따라 기억력이 상실되거나 유지되는 원리를 이용함. 예를 들어, AI가 배운 정보 중에서 중요한 정보는 기억력이 길게 유지되고, 중요하지 않은 정보는 기억력이 짧게 유지되어서 자동으로 잊혀지는 방식이야. 

### 작동 방식 

YourMemory는 Python 3.11 이상에서 동작하며, Docker나 데이터베이스 설정 없이 쉽게 설치할 수 있음. 사용자들은 Claude, Cline, Cursor 등 다양한 클라이언트에서 YourMemory를 사용할 수 있으며, 기존의 MCP-compatible 클라이언트에서도 호환됨. 

### 사용 방법 

1. YourMemory를 설치한다: `pip install yourmemory`
2. 설정을 초기화한다: `yourmemory-setup`
3. config 경로를 얻는다: `yourmemory-path`
4. Claude 등의 클라이언트에서 YourMemory를 설정한다: `.claude/settings.json` 등의 파일에 YourMemory 설정을 추가함 
5. 메모리 워크플ロー 설정을 추가한다: `CLAUDE.md` 파일에 메모리 워크플ロー 설정을 추가함 

### 예시 

```python
store_memory("Sachit prefers tabs over spaces in Python", importance=0.9, category="fact")
recall_memory("Python formatting")
# {"
-content": "Sachit prefers tabs over spaces in Python", 
# "strength": 0.87}
```
