# 로컬에서 개인정보 보호하는 최고퀄리티 TTS, 코코로
 mấy 년 전만 해도 현실적인 로컬 음성 생성은 상상할 수도 없을 정도였음. 근데，现在은 퀄리티가 엄청나고, 중요한 건 개인 정보를 지키면서 이런 결과를 내준다는 거야. 
 위에 있는 영상은 샘플 텍스트에서 생성된 오디오를 보여주고 있고, 이건 이전에 GTX 1080 Ti에서 사용한 로컬 머신에서 돌린 거야. 이 머신에는 전용 GPU가 있지만, GPU는 모두 LLM 추론을 위해 예약되어 있고, 음성 생성은 CPU로 完全하게 동작함. 
 사용된 모델은 Kokoro야. Kokoro는 82M 파라미터밖에 안 되는 모델이지만, 영어, 표준 중국어, 힌디 등 여러 언어에서 현실적인 음성을 생성해 줌. 특히 영어는 50개가 넘는 다양한 음성을 제공하니까, 다양한 선택지를 가질 수 있겠음. 
 Kokoro를 설정하는 방법은 여러 가지가 있지만, 가장 간단한 방법은 Kokoro-FastAPI라는 프리메이드 컨테이너 이미지를 사용하는 거야. 이 컨테이너 이미지에는 사전에 다운로드된 음성 모델이 포함되어 있기 때문에, 약 5GB 크기로 꽤 큰 편이야. 
 도커나 포드맨을 사용해서 컨테이너를 실행하려면, 다음 명령어를 사용하면 됨:
```bash
podman run -p 8880:8880 ghcr.io/remsky/kokoro-fastapi-cpu
```
 컨테이너가 正しく 동작하는지 간단하게 확인하려면, `localhost:8880/web`으로 접근해서 웹 UI를 사용할 수 있으면 된당. 여기서 텍스트를 주면 자동으로 음성을 생성하고 재생시켜 줌. 
 컨테이너는 단순한 웹 UI뿐만 아니라, 오픈艾 TTS 인터페이스를 지원하니까, 이미 오픈艾 TTS API를 사용하는 프로그램은 쉽게 Kokoro로 전환할 수 있을 거임. 
 테스트를 위해, 자바스크립트와 파이썬 예제 코드가 github.com/remotebrowser/speak에 있다. 이 레포ジ토리를 클론하면, 다음 명령어로 음성을 생성하고 재생할 수 있음:
자바스크립트:

export TTS_API_BASE_URL=http://127.0.0.1:8880/v1
