## 1. 개요

안녕 꼬맹이들아, 오늘은 셰이더 코딩하느라 눈 실핏줄 터지는 너희들을 위해 기가 막힌 물건 하나 가져왔다. 이름하여 **Shader Lab**. 

한마디로 요약하면 **"셰이더용 포토샵"**이야. 예전처럼 수학 공식 뚜드려 박으면서 머리 싸매지 않아도, 포토샵 레이어 쌓듯이 슥슥 만지면 셰이더가 튀어나온다는 거지. 라떼는 말이야... C++에 HLSL 다 직접 짰는데 세상 참 좋아졌다, 그치?

## 2. 주요 기능 및 파라미터 (뭐뭐 있나 보자)

본문 보니까 설정값이 아주 그냥 빼곡해. 주로 CRT 감성이나 레트로 효과 주는 데 특화된 거 같더라고. 레이어 구조로 되어 있어서 쓰기 편해 보여.

### 2.1 기본 레이어 및 색상 설정
* **불투명도(Opacity):** 레이어 겹칠 때 필수지.
* **색조/채도(Hue/Saturation):** 색감 조절은 기본 중의 기본이야.
* **블렌딩 모드(Blend Mode):** 포토샵처럼 레이어끼리 어떻게 섞을지 정하는 기능이야.

### 2.2 CRT 효과 (레트로 감성 미쳤음)
* **스캔라인(Scanline):** 옛날 TV 줄 가는 거 강도 조절 가능해.
* **배럴 왜곡(Barrel Distortion):** 브라운관 볼록하게 튀어나온 느낌 알지? 그거야.
* **비네트(Vignette):** 테두리 어둡게 만드는 거.
* **잔상(Persistence):** 화면 움직일 때 묘하게 끌리는 느낌까지 구현했네.

### 2.3 특수 효과 및 노이즈
* **글리치(Glitch):** 지지직거리는 연출. 속도랑 강도 다 조절된다.
* **블룸(Bloom):** 뽀샤시하게 빛 번지는 거. 셰이더의 꽃이지.
* **플리커(Flicker):** 화면 깜빡깜빡하는 거.

## 3. 할배의 한줄평

이거 보니까 게임 UI나 레트로풍 인디 게임 만드는 애들한테는 진짜 신세계일 것 같다. 매번 노드 짜고 코드 복붙하느라 시간 다 보냈을 텐데, 이건 그냥 포토샵 레이어 만지듯이 하면 되니까 말이야. 

근데 툴이 아무리 좋아져도 기본 원리는 알아야 하는 거 알지? 공부 게을리하지 마라. 자, 가서 한번 써보고 후기 남겨봐라!
