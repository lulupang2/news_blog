### TurboQuant 개요
TurboQuant는 벡터를 압축하기 위한 기술임. 기존의 DRIVE, EDEN 기술을 발전시킨 버전임. 
본격적으로 이야기를 시작하기 앞서, Vector란 무엇인지부터 설명해 보겠다. 
#### Vector란?
- Vector란ordered list of numbers라고 생각해도 되고, 
- 공간 상의 arrow라고 생각해도됨. 
- 예를들어 [0.3, -1.2]를 2차원 공간상의 arrow로 표현할 수 있지. 
- 3차원 이상의 차원에서는 사람이 직접 그려보는 것보다는, 
- ordered list의 개념으로 이해하는게 편할 수 있 음. 
- 예를 들어, [1, 2, 3]은 3차원 공간상의 한 점을 의미할 수 있어.
 
#### Quantization 이란?
- Quantization이란, 숫자를 특정한 범위로 줄이는 것임. 
- 예를들어, 0부터 10까지의 숫자를, 
- 0, 2, 4, 6, 8, 10의 6개 값으로 Quantization 할 수 있지.
- 이렇게 Quantization한 값을 이용해 원래의 값을 근사할 수 있음. 
   ### DRIVE 기술 
   - DRIVE는 Random rotation, 
   - post-rotation distribution analysis를 수행하는 기술임.
   - DRIVE 기술은, 
   - input vector를 random하게 rotate한 후, 
   - rotate된 vector의 sign과 scale을 전송하는 방식임. 
   - RECEIVER는 이 전송된 정보를 이용해, 
   - 원래의 vector를 recover함.
 
### EDEN 기술 
- EDEN은 DRIVE 기술을 발전시킨 버전임. 
- EDEN은, 
- DRIVE에서 사용하는 random rotation을, 
- Randomized Hadamard Transform으로 대체하였음. 
- 이 덕분에, 
- computation 시간을 줄일 수 있었음.
 
### TurboQuant 기술 
- TurboQuant는 EDEN 기술을 발전시킨 버전임. 
- TurboQuant는, 
- EDEN에서 사용하는 scale을, 
- 고정된 값으로 대체하였음. 
- 이 덕분에, 
- computation 시간을 줄일 수 있었음.
 
   ### 결론 
   - TurboQuant는, 
   - DRIVE, EDEN 기술을 발전시킨 버전임. 
   - TurboQuant는, 
   - 벡터를 압축하기 위한 기술임. 
   - 벡터를 압축함으로써, 
   - 저장공간을 절약할 수 있음. 
   - 또한, 
   - computation 시간을 줄일 수 있음.
