### Beam Spring B104 키보드, 사기냐?
딴죽 걸린 Beam Spring 키보드를 구입한 사용자 리뷰입니다. 
사실 Beam Spring 키보드는 IBM의 메인프레임 터미널용으로 만들어진 키보드인데, 이베이에서 원형을 1000달러에서 2000달러 사이의 가격으로 거래되고 있다고 합니다. 
현재 beam spring 키보드는 50%가량 싸게 나온다고 하네요. 

#### Beam Spring B104 키보드 구성
- Beam Spring B104 키보드: 399달러
- ANSI (정규 US) 레이아웃 또는 ISO 레이아웃 선택 가능
- 케이스 색상: 블랙, 오프화이트/베이지, 인더스트리얼 그레이, 다크 블루, 라이트 블루, 트루 레드, 정규 그레이 중 선택 가능
- 키셋: 
  - 모두 검은색 키에 하얀색 글자, 39달러
  - 검은색 수정키에 하얀색 글자, 나머지 키는 하얀색에 검은색 글자, 49달러
  - 추가 키셋, 모두 검은색 키에 하얀색 글자, 49달러
  - 추가 키셋, 검은색 수정키에 하얀색 글자, 나머지 키는 하얀색에 검은색 글자, 59달러
