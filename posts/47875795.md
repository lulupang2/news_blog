# 프로젝트介绍
.flipdisc-display를 이용해서 기술적인 내용을 넣은 프로젝트에요.

## Flipdisc-Display란?
.flipdisc-display는 전자기펄스를 이용해서 작은 디스크를 두 가지 색상으로 변경할 수 있는 디스플레이 类型이에요.
아주 오래전부터 만들어졌지만, 여전히 같은 기술을 사용하고 있어요.

## 프로젝트 목적
저는 큰 인터랙티브 월 아트를 만들고 싶었는데, 일반적인 스크린의 LED 빛이 싫어요.
.flipdisc-display는 대안으로 고려해 볼 수 있는 선택이에요.
높은 가독성, 긴 수명 그리고 25-60프레임을 구현할 수 있어요.
또한, 디스크가 플립될 때 우산에 비가 떨어지는 소리와 같은 달콤한 소리가 나요 ㅋㅋㅋ

## 프로젝트 진행
9개의 Alfazeta 패널을 사용해서 3x3 그리드를 만들었어요.
각각의 ボード에는 28x7 패널이 두 개씩 있어요.
설계의 크기는 다이어그램을 참고하세요.

## 하드웨어 소개
ATMEGA128 마이크로컨트롤러, 수백개의 MELF 다이오드 그리고 두 개의 DIP 스위치로 구성되어 있어요.
한 개의 DIP 스위치는 주소를 설정하고, 다른 하나는 보드 레이티를 설정해요.

## 전원 공급
24V 10A의 Meanwell 전원 공급 장치를 사용해요.

## 프레임 만들기
80/20 알루미늄 프레임을 사용해서 만들어졌어요.
경고: 보드와 디스크는 매우 약해서 주의해서 다루세요.

## 케이블링
각각의 칼럼을 시리즈로 연결하고, 데이터 라인은 RS485 블록 터미널의 +/-를 사용해요.
18AWG 와이어를 전력 라인에, 22 AWG 스탠드 케이블을 데이터 라인에 사용해요.

## 소프트웨어
Node.js 라이브러리를 만들어서 .flipdisc 스크린을 사용할 수 있게 했어요.
Github에 업로드 되어 있어요.

## 예시
```javascript
import { createDisplay } from 'flipdisc'
const path = '/dev/tty0'
const display = createDisplay([[1], [2]], path)
display.send(frameData)
```

## 인터페이스
Expo 앱을 만들어서 사용자 인터페이스를 제공해요.
앱에서는 시ην을 일시정지, 시작, 건너뛰기, 이전으로 할 수 있어요.
 또한, Scene 변수를 설정해서 경험을 개인화할 수 있어요.
 화면에 직접 그림을 그릴 수도 있어요.

## 디자인
Strict한 픽셀 제한이 있기 때문에 créativity를 발휘할 수 있어요.
간단한 정보만 전달하면 되서 의도적인 결정이 필요하대요.
