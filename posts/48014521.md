### 티아이 83 플러스 프로그래밍 튜토리얼: 초보자 가이드

티아이 83 플러스를 프로그래밍하기 전에,先에 티 그래프 링크 83+ 애플리케이션을(pc에) 다운 받아라. 그러면 프로그래밍을 하는데 편리할 거야.

 티아이 그래프 링크 83+, 83i 뷰어, 티 커넥트, 그리고 에뮬레이터를 다운로드 받아라. 그리고 티아이와 컴퓨터를 연결하기 위해 블랙 시리얼 케이블이나 티 커넥트 케이블도 준비해놔야 해.

이 튜토리얼에 포함된 프로그램들은 자유롭게 수정해서 사용하고, inet에 올リ거나 판賣도 가능해. 단, 아래 조건들을 지켜야 함.
* 프로그램을 조금이라도 수정해야 해.
* 제작자 보리스 체르니(Boris Cherny)에 대한 크레딧을 제공해야 해.

튜토리얼을 만들면서 많은 도움을 준분들 teşekkür하고 싶음: ~일리야 에스~, ~톰 티~.

초보자라면, 튜토리얼을 처음부터 끝까지 읽으며, 모든 샘플 프로그램들을 직접 작성해보고, 모든 도전 과제들을 시도해봐.そして楽しノㅑ.

#### 시작하기

 DISP 함수는 간단하게 텍스트를 출력하는 법을 가르쳐줌. OUTPUT 함수는 좌표를 지정해서 텍스트를 출력하는 법을 가르쳐줌.

```markdown
: Disp "TUTORIAL"
: Output(3,2,"TUTORIAL")
```

 CLRHOME 함수는 현재 화면을 초기화하는 법을 가르쳐줌.

```markdown
: ClrHome
: Disp "HELLO WORLD"
```

 LBL/GoOTO 함수는 레이블을 지정하고, 특정 레이블로 이동하는 법을 가르쳐줌.

```markdown
: Lbl A
: ClrHome
: Output(4,2,"HELLO WORLD")
: Goto A
```

 END/PAUSE 함수는 루프를 종료하거나, 일시 정지하는 법을 가르쳐줌.

```markdown
: Lbl A
: ClrHome
: Output(4,2,"HELLO WORLD")
: Pause
: ClrHome
```

 MENU 함수는 메뉴를 만들고, 사용자 입력을 처리하는 법을 가르쳐줌.

```markdown
: Menu("--MENU--","BANANA",1,"ORANGE",2,"PEACH",3,"QUIT",X)
```

 INPUT/PROMPT 함수는 사용자 입력을 받아서 처리하는 법을 가르쳐줌.

```markdown
: ClrHome
: Prompt X
: Disp X
: Pause
: ClrHome
: Input X
: Disp X
: Pause
: Stop
: End
```

 VARIABLES는 변수를 선언하고, 값을 할당하는 법을 가르쳐줌.

```markdown
: ClrHome
: 250A
: 500B
: Output(2,1,A)
: Pause
: ClrHome
: Output(2,1,B)
: Pause
```

### 모어 어드밴스드 토픽

FOR 함수는 반복문을 사용하는 법을 가르쳐줌.

```markdown
: For(A,0,50)
: Output(3,1,"THIS")
: End
```

WHILE 함수는 조건이 만족될 때까지 반복문을 사용하는 법을 가르쳐줌.

```markdown
: ClrHome
: 1Y
: While Y<16
: 1X
: While X<8
: Output(X,Y,"+")
: X+1X
: End
: Y+1Y
: End
```
