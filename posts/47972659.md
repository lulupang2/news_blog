### AutoRound란 무엇임?
AutoRound는 LLM과 Vision-Language Models을 위해 설계된 전통의 양자화 툴킷이야. Sign-Gradient Descent를活用해서 2~4비트의超초반 비트 너비에서도 높은 정확성을 보여주고, 하드웨어 兼容性도 широк게 지원해줌.

### 주요 특징
* **수준급의 정확성**: 2~3비트에서 좋은 성능을 보이고, 4비트에서는 최고 성과를 거두었음 ㅋㅋㅋ
* **생태계 통합**: Transformers, vLLM, SGLang 등과의 원활한 호환성을 지원함.
* **다중 형식 내보내기 지원**: AutoRound, AutoAWQ, AutoGPTQ, GGUF와 같은 여러 형식을 지원해서 호환성을 최대화함.
* **빠른 混合 비트/데이터형식 스키마 생성**: vài 분 만에 자동으로 구성할 수 있고, 모델의 BF16 RAM 크기의 1.1~1.5배의 오버헤드만 가지고 있으면 됨.
* **Round-to-Nearest 모드 최적화**: --iters 0을 사용해서 4비트에서 약간의 정확도 저하와 함께 빠른 양자화를 진행할 수 있음.
* **양자화 비용 저렴**: 단일 GPU에서 10분만에 7B 모델을 양자화할 수 있고, 자세한 내용은 양자화 비용 문서를参考吧.
* **10+ Vision-Language 모델 지원**: Out-of-the-box로 10개의 비전언어 모델에 대한 양자화를 지원하고, 지원 matrix도 제공해줌.
* **여러 가지 레시피 선택**: 사용자에 따라 auto-round-best, auto-round, auto-round-light를 선택해서 쓸 수 있음. 자세한 문서는 양자화 레시피를 참고해.
* **고급 유틸리티 포함**: 다중 GPU 양자화, 다중 캘리브레이션 데이터셋, 10+ 런타임 백엔드 지원 등이 포함된 고급 유틸리티를 제공하고, weight만을 위한 양자화 뿐만 아니라 MXFP, NVFP, W8A8 등 추가 데이터형식의 지원도 활발하게 진행 중임.

### 설치 방법
pip install auto-round

### CLI 사용법
auto-round --model Qwen/Qwen3-0.6B --scheme 'W4A16' --format 'auto_round' --output_dir ./tmp_autoround

### API 사용법
from auto_round import AutoRound

# 모델 불러오기 (FP8, BF16, FP16, FP32 지원)
model_name_or_path = 'Qwen/Qwen3-0.6B'

# 사용 가능한 스키마: 'W2A16', 'W3A16', 'W4A16', 'W8A16', 'NVFP4', 'MXFP4' (실제 커널 없음), 'GGUF:Q4_K_M' 등
ar = AutoRound(model_name_or_path, scheme='W4A16')

# 지원되는 형식: 'auto_round' (기본값), 'auto_gptq', 'auto_awq', 'llm_compressor', 'gguf:q4_k_m' 등
ar.quantize_and_save(output_dir='./qmodel', format='auto_round')

### 주요 하이퍼파라미터
scheme (str|dict|AutoScheme): 미리 정의된 양자화 키 (예: W4A16, MXFP4, NVFP4, GGUF:Q4_K_M)
bits (int): 양자화에 사용할 비트 수 (기본값: None)
