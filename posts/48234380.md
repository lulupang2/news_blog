### Deno 2.8 나왔 븅신들아!
Deno 2.8이终于 나왔 븅신들아! 이번 버전은 Deno의 minor release 중에서 가장大き다고 하니까 기대해도 될 것 같아. 
업그레이드 하려면 터미널에서 `deno upgrade` 입력하면 되고, Deno가 설치 안 된 븅신들은 설치하고 나서 업그레이드 하세요.

#### 새로운 기능
* `deno audit fix`: ahora npm 패키지의 취약점 자동으로 고칠 수 있어요!
 + `deno audit`는 npm 패키지의 취약점 확인하는 툴이었는데, 이제는 `deno audit fix`로 자동으로 고칠 수 있어요.
* `deno bump-version`: 프로젝트 버전 업그레이드 할 때 편해졌어요!
 + 이제 `deno bump-version`로 프로젝트 버전 업그레이드 할 수 있어요.
* `deno ci`: CI 환경에서 Deno 설치 할 때 편해졌어요!
 + 이제 `deno ci`로 Deno 설치할 수 있어요.
* `deno pack`: Deno 프로젝트를 npm 패키지로 빌드할 수 있어요!
 + `deno pack`로 Deno 프로젝트를 npm 패키지로 빌드할 수 있어요.
* `deno transpile`: TypeScript 코드를 JavaScript로 바꿀 수 있어요!
 + `deno transpile`로 TypeScript 코드를 JavaScript로 바꿀 수 있어요.
* `deno why`: 패키지 의존성 확인할 때 편해졌어요!
 + `deno why`로 패키지 의존성을 확인할 수 있어요.

#### 성능 개선
* npm 설치 속도 개선 됐어요!
 + npm 패키지 설치 속도가 3.66배 빠르다고 하니까 좋네요.
* Node.js 호환성 개선 됐어요!
 + Node.js 테스트套件 통과율이 76.4%로 개선됐다고 하니까 좋네요.
