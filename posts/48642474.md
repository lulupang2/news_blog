## 게임 개요
게임의 이름은 'You're the OS'래 ㅋㅋㅋㅋคอม퓨터의 운영체제가 된다는 느낌으로 해보는 건 어떨까嗯? 
프로세스랑 메모리랑 입출력 이벤트들을 관리해 줘야 한단다. 
유저가 프로세스가 너무 오래 걸리면 기대 이상으로 기다리게 되는데 이때 리부팅 당할 수 있단다. 
개빡치는 점은 [이 링크](https://plbrault.github.io/youre-the-os)에서 직접 해볼 수 있단다. 
itch.io에서도 다운 받을 수 있대. 

## 시스템 요구 사항
파이썬 버전은 3.14 이상이야. 
그 아래 버전은 못 쓴단다. 
그래서 pyenv를 써서 필요한 버전을 깔아야 해. 
pipenv도 설치해야 해. 
프로젝트 루트에 비어 있는 .venv 폴더가 있어야 하고, 
사용法은 pipenv sync --dev로 의존성을 설치하고, 
데스크톱 앱으로 실행은 pipenv run desktop, 
웹 버전은 pipenv run web이야. 

## 샌드박스 모드
샌드박스 모드는 메뉴를.skip하고 바로 사용자 지정 스테이지를 실행하는 거임. 
개발목적으로 씀. 
샌드박스 설정 파일을 만들어야 하는데 예제는 src/sandbox/sample.py에 있대. 
이 파일은 src/sandbox 디렉토리에 저장하는게 좋고, 
이 디렉토리에 추가된 파일은 Git에 올라가지 않음. 
그리고 pipenv run sandbox sandbox.sample을 실행하면 된단다. 
자동 스크립트로 실행도 가능하대. 

## 빌드 및 배포
웹 버전을 실행하지 않고 빌드하는 명령어는 pipenv run web build랜다. 
itch.io용 zip 파일을 만들려면 pipenv run web archive래. 
린터를 실행하려면 pipenv run pylint, 
유닛 테스트는 pipenv run pytest임.
