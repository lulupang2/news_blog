Godot 게임 엔진으로 유체 시ミュ레이션을 구현하는 방법을 배우는 곳입니다. 먼저, Navier-Stokes 방정식이라는게 있는데, 이 방정식을 이용해서 유체의 흐름을 시뮬레이션할 수 있습니다. 하지만, 이 방정식은 계산하기가 매우 복잡해서, 게임에서 사용하기에는 부적절합니다. 그래서, 근사화된 방정식을 사용해서 유체의 흐름을 시뮬레이션합니다. 
   
   ### 유체 시ミュ레이션의 기본 개념
 
   유체 시ミュ레이션은 유체의 밀도와 속도를 계산해서, 유체의 흐름을 시뮬레이션하는 것입니다. 유체의 밀도는 유체의 각 셀에 담긴 물질의 양을 뜻하고, 속도는 유체의 흐름의 방향과 속도를 뜻합니다. 
 
   ### Godot에서 유체 시ミュ레이션 구현하기
 
   Godot에서 유체 시ミュ레이션을 구현하기 위해서는, 먼저 Grid를 생성해야 합니다. Grid는 유체의 흐름을 계산하는데 사용됩니다. Grid의 크기는 N x N으로 설정하고, 각 셀의 크기는 cell_size로 설정합니다. 
   
   ```gdscript
   @export var N := 16
   @export var cell_size := 32
   var size := 0
   ```
   
   다음으로, 유체의 밀도와 속도를 계산하는데 사용할 배열을 생성해야 합니다. 
   
   ```gdscript
   var density: PackedFloat32Array
   var density_prev: PackedFloat32Array
   var u: PackedFloat32Array
   var u_prev: PackedFloat32Array
   var v: PackedFloat32Array
   var v_prev: PackedFloat32Array
   ```
   
   ### Grid를 그리는법
   
   Grid를 그리는 방법은 다음과 같습니다. 
   
   ```gdscript
   func _draw():
       for j in range(0, N + 2):
           for i in range(0, N + 2):
               var x := i * cell_size
               var y := j * cell_size
               var rect := Rect2(x, y, cell_size, cell_size)
               var is_boundary := i == 0 or j == 0 or i == N + 1 or j == N + 1
               var fill := Color(0.16, 0.08, 0.08) if is_boundary else Color(0.08, 0.08, 0.08)
               draw_rect(rect, fill, true)
               draw_rect(rect, Color(0.35, 0.35, 0.35), false)
   ```
   
   ### 밀도 계산하기
   
   밀도를 계산하는 방법은 다음과 같습니다. 
   
   ```gdscript
   func fade_density(delta: float) -> void:
       for j in range(1, N + 1):
           for i in range(1, N + 1):
               var idx := IX(i, j)
               density[idx] *= 0.1 * delta
   ```
   
   ### 마우스 클릭으로 밀도 추가하기
   
   마우스 클릭으로 밀도를 추가하는 방법은 다음과 같습니다. 
   
   ```gdscript
   func _input(event):
       if event is InputEventMouseButton and event.pressed:
           var cell := cell_from_mouse(to_local(event.position))
           var i := cell.x
           var j := cell.y
           if i >= 1 and i <= N and j >= 1 and j <= N:
               density[IX(i, j)] += 1.0
               queue_redraw()
   ```
