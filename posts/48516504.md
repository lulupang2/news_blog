### AI 오픈소스 툴 tensorzero(repoarchive 처리당함)
tensorzero는 오픈소스 llm오퍼레이션 플랫폼으로 게이트웨이, 관측 가능성, 평가, 최적화, 실험 등 기능들을 제공하며, 다양한 llm 제공업체와 통합 가능함.

#### 특징 및 기능
- **게이트웨이**: 다양한 llm 제공업체에 접근할 수 있는 통합 api 제공
- **관측 가능성**: 추론 결과와 피드백을 데이터베이스에 저장하고, ui 또는 프로그래밍 방식으로 접근 가능
- **평가**: 추론 결과 및 워크플로우를 벤치마크하는 기능
- **최적화**:生产 метрик과 인간 피드백으로_prompt, 모델, 추론 전략 등을 최적화함
- **실험**: a/b 테스트, 라우팅, 폴백, 재시도 등 기능들을 제공해서_ship with confidence하게 해줌.

#### 사용 예시
tensorzero를 사용해서 오픈ai sdk와 통합해서 사용해 봅시다.
```python
from openai import OpenAI

# tensorzero 게이트웨이에 접근
client = OpenAI(base_url='http://localhost:3000/openai/v1', api_key='not-used')

# 추론 결과 얻기
response = client.chat.completions.create(
    model='tensorzero::model_name::anthropic::claude-sonnet-4-6',
    messages=[{'role': 'user', 'content': 'tensorzero에 대해 알려줘'}]
)
```
