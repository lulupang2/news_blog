### 리눅스에서 브라우저를 컨테이너로 돌리면 보안이 걱정이 되는 사람들 들어와 보셈

#### LXC를 사용하여 브라우저 컨테이너 만들기

LXC는 리눅스를 위한 컨테이너 기술이야
LXC는 리눅스 시스템의 프로세스를 격리하여 컨테이너로 만드는 기술인 것처럼
네트워크를 격리하여 컨테이너로 만들 수 있지
LXC를 사용하여 브라우저를 컨테이너로 만들 수 있어
다음은 LXC를 사용하여 브라우저 컨테이너를 만드는 과정이야

1. **LXC 설치하기**
   - LXC를 설치하려면 다음 명령어를 실행해
     ```bash
     sudo pacman -S lxc lxcfs
     ```
   - LXC를 설치하면 LXC를 사용하여 컨테이너를 만들 수 있어

2. **LXC 컨테이너 네트워킹 설정하기**
   - LXC 컨테이너를 네트워킹 설정하려면 다음 명령어를 실행해
     ```bash
     sudo systemctl enable lxc-net.service --now
     ```
   - LXC 컨테이너를 네트워킹 설정하면 컨테이너 안의 브라우저가 외부 네트워크에 접근할 수 있어

3. **LXC 컨테이너 만들기**
   - LXC 컨테이너를 만들려면 다음 명령어를 실행해
     ```bash
     sudo lxc-create --config /etc/lxc/www.conf --name www -t download --- -d debian -r trixie -a amd64
     ```
   - LXC 컨테이너를 만들면 컨테이너 안에서 브라우저를 실행할 수 있어

4. **컨테이너 안의 브라우저 실행하기**
   - 컨테이너 안의 브라우저를 실행하려면 다음 명령어를 실행해
     ```bash
     sudo lxc-start www
     sudo lxc-attach www -- su www -c firefox
     ```
   - 컨테이너 안의 브라우저를 실행하면 컨테이너 밖의 시스템 파일에는 접근할 수 없어
