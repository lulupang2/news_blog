# 마이크로소프트의 플린트는 인공지능 에이전트의 시각화 언어임
인공지능 에이전트가 데이터를 시각화하는 건 사용자와 데이터를 연결하는 다리같은 건데, 문제는 그런 에이전트를 만들어내는 게 개빡チ음
- 간단한 차트 스펙은 신뢰성이 있는데, 만들어진 차트가 다 그런대로 형편없어보임ㅋㅋㅋ 왜냐면 시스템 기본값에만 의존해서 그런 거임
- 복잡한 차트 스펙이라면 그래도 좋은 차트는 만들어내지만, 에이전트가 그런 스펙 이해하기 개빡チ고 신뢰성 떨어짐
플린트 개발진은 언어 자체의 한계가 문제라는 걸 알아챈 거임.현재 시각화 언어는 인공지능 에이전트에게 너무 низレイヤ의 명령어를 요구하기 때문에, 에이전트가 사람처럼compiler가 알아서 처리해줘야 할 시각적 결정도 따로 해줘야 하니까 그런 문제가 생기는 거임
플린트는 그런 문제를 해결하기 위한 시각화 중간 언어임.인공지능 에이전트가 간단한 의미 기반 명령어만으로 좋은 차트를 만들어낼 수 있게 해주는 거임. 레이아웃 최적화 엔진이 있어서 높은 수준의 명령어만으로도低レイヤ의 세부사항을 자동으로 채워넣어 좋은 차트 만들어주니까 사람도 이해하기 쉬움
마이크로소프트의 데이터 포뮬레이터 프로젝트에도 사용된دهای 플린트임. 오픈소스로 공개되어서 마음만 먹으면 바로 다운받아 에이전트 앱에다가 플러그인으로 끼워 넣어볼 수 있음
