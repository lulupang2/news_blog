# 개요

      요즘 AI 모델들이 진짜 많이 늘엇다고는 하지만, 그런 애들이 그림 그리면 어떻게 나올까 궁금헀음. 그래서 일단 GPT-5.6, Claude, Gemini, Grok 이런거 다 가지고 와서 모나리자랑 스타리 나이트를 그려봄.

      

      # 설정

      일단 환경 설정은 그냥 같게 해줌. 컬러 펜슬툴셋 같은건 모든 모델이同じ 걸 쓰게 해줬고, 타겟 이미지를 보거나 자신이 그린거 보는 기능 같은건 다 똑같이 줌. 그려는 도구도 같게 해줫음. 컬러랑 펜 사이즈, 압력 이런거 다 설정할 수 잇음. 그리고 그린거 보면서 고칠 수 잇음.

      

      # 결과

      일단 모나리자 그리고 나서 결과를 보니까 GPT-5.6가 젤 잘그림. 그다음이 Claude Fable 5였음. 근데 Grok 4.5는 진짜 쓰레기였음. Gemini 3.6 Flash도 그런대로 괜찮았음.

      

      # 비용

      비용까지 계산해봤는데 Claude Fable 5가 젤 많이 들었음. 거의 20배나 더 들었음. Grok 4.5는 비용은 적게 들엇지만 결과가 개망했음. Gemini 3.6 Flash도 비용은 적게 들엇지만 결과는 괜찮았음.
