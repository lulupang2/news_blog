   # 유에스비시.max 만세!
   작년 휴가 때 처남 와이프가 가져간 거 보니까 유에스비시 케이블이랑 포트가 너무 많더라. 나는 유에스비시.max 만 쓴다. 
   어때보이지? 그냥 유에스비시 케이블 하나면 다 된다는 뜻이야.  
   # 유에스비시에 대한 나의 생각
   유에스비시.max가 최고인 이유는 케이블 하나면 모든 장치를 충전할 수 있다는 거야. 폰, 노트북, 시계, 이어폰 등등 다 유에스비시로 충전함. 
   유에스비시.max 때문에 여행할 때 가방에 너무 많은 거 들고 다닐 필요가 없더라. 
   # 유에스비시.max의 장점
   * 하나의 케이블로 모든 장치를 충전할 수 있음
   * 가방에 太多한 거 들고 다닐 필요가 없음
   * 유에스비시.max 때문에 모든 유에스비시 포트가 동일하기 때문에 케이블을 잃어버렸을 때 바로 구할수 있음 
   # 유에스비시.max의 단점
   * 아직 모든 장치가 유에스비시.max를 지원하지 않음
   * 유에스비시.max 케이블이 항상 동일한 성능을 보장하지는 않음
   하지만 유에스비시.max의 장점이 너무 많아서 나는 유에스비시.max만 쓸 거야. 
   # 결론
   유에스비시.max 만세! 다른 유에스비시는 쓰지 마. ㅋ
