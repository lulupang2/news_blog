# 퀄컴 리눅스 2.0 떳다 ㅋㅋㅋㅋ
 
     퀄컴에서 새로운 리눅스 버전 2.0을 발표했음. 
     이제부터 퀄컴 칩을 사용하는 기기는 이 리눅스로 업데이트될 거 임. 
     퀄컴 리눅스 2.0에는 새로운 보안 기능과 성능 개선이 포함되어 잼. 
     특히, 새로운 보안 패치로 인해 기존의 보안 취약점들이 모두 해결됨. 
     이제 퀄컴 칩을 쓰는 기기들은 더 이상 보안 걱정 안해도 되겠네? ㅋㄷㅋㄷ 
     퀄컴은 앞으로도 지속적으로 리눅스 업데이트할 계획이라고 함. 
     퀄컴 리눅스 2.0은_already_ 많은 기기에서 사용 가능하고, 앞으로 더 많은 기기로 확대될 거 임.
