# RK3562 데비안 리눅스 워크스테이션 만들기
RK3562라는 chip을 쓰는 80달러짜리 안드로이드 태블릿을 데비안 리눅스로 돌리는 방법을 알아보려고 함.
### 하드웨어 스펙
* SoC: Rockchip RK3562 (4x Cortex-A53 @ 2.0 GHz)
* NPU: 1x Rockchip NPU core (RKLLM)
* RAM: 4 GB LPDDR4
* 스토리지: 128 GB eMMC (안드로이드) + SD 카드 (데비안)
* 디스플레이: 10.1" DSI 패널, 1280x800

### 소프트웨어 지원
* 디스플레이: 완전 지원
* 터치스크린: 완전 지원 (10점 멀티 터치)
* 와이파이: 완전 지원
* 블루투스: 완전 지원
* 오디오 출력: 완전 지원
* 마이크: 완전 지원
* 가속기: 부분 지원 (Panfrost, OpenGL ES 지원)
* NPU (RKLLM): 활성
* 자이로스코프: 완전 지원
* 플래시: 완전 지원
* 전원 버튼: 완전 지원
* 잠금 화면 방향: 완전 지원

### 알려진 문제점
* 배터리 LEVEL 이 초기화 안 됨
* 카메라 색상 조정이 필요함

### 빌드 방법
1. 호스트 머신에 необходим한 패키지를 설치
2. 빌드 스크립트를 실행
3. SD 카드에 이미지 쓰기

### 빌드 옵션
* --ui-session: Phosh 세션 프로파일
* --gpu-stack: GPU 스택 선택 (mali, panfrost)
* --display-server: 디스플레이 서버 선택 (wayland, x11)
* --cpu-governor: CPU 고버너 선택
* --force-clean-rootfs: 루트 파일 시스템 재생성

### 결과
RK3562 태블릿에서 데비안 리눅스가 작동함. NPU도 지원함. 참고로 RK3562는 4x Cortex-A53 @ 2.0 GHz를 쓰는 칩이라고 함. 요약하면 80불짜리 안卓 태블릿을 개조해서 데비안 리눅스로 돌리는 방법임.ㅋㅋㅋ
