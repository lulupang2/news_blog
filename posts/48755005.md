# 개쩌는 DIY 로봇 청소기 '오무우' - 직접 만들고 커스텀해봐 ㅋㅋㅋㅋ
     오무우는 직접 만들 수 있는 오픈소스 로봇 청소기야. 이름은 旋回라는 뜻의 영단어에서 따옴. 
     ## 주요 특징
     - 오픈소스 하드웨어, 소프트웨어, 펌웨어를 기반으로 만들어짐
     - 직접 3D 프린팅과 라즈베리파이, ESP32 등으로 만들 수 있기 때문에 커스텀 하기 존나 쉬움
     - 홈 어시스턴트와 통합되어 있고, 클라우드 기능 없이도 작동하는 로컬 우선 방식으로 설계됨
     ## 현재 프로젝트 상태
     최초 버전(v0)이 나온 상태이고, 지금은 바닥에서 작동하는 기본적인 버전을 만들고 있는 중임. 
     ## 커뮤니티 참여
     - 깃헙에서 진행 중이며, 누구나 참여하여 모듈을 개발하고 PR을 보낼 수 있음
     - 여러 명이同じ 모듈을 작업할 수 있으며, 최선의 솔루션이 나올 때까지 계속 개선됨
     ## 기타
     - 현재는 라즈베리파이 5를 사용하고 있지만, 추후 라즈베리파이 3B+ тоже 작동하게 할 예정
     - 기존의 로봇 청소기를 업그레이드 하거나, 직접 부품을 조달하여 만들 수 있음
