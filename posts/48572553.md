# ai cad란게 나오네요 ㅋㅋㅋㅋ 

   * ai로 cad 모델링이 가능해졌다고 하는 거 같은데 

   * 자연어 입력하면 3D 모델로 만들 수 있다고 하고 

   *-parametric 컨트롤도 가능하다고 하니까 
 
   # cad 모델링 예시 
 
   * v8 엔진도 만들었고 
 
   * 9실린더 레이더 엔진도 만들었고 
 
   * 터보팬 제트 엔진도 만들었고 
 
   * 각종 기계부품들도 만든다네 

   # 개발 환경 
 
   * 노드js 20.19.0 이상이거나 22.12.0 이상 필요 
 
   * npm 10 이상 필요 
 
   * supabase cli도 필요하고 
 
   # ngrok 설정 
 
   * ngrok 설정해서 로컬 개발 환경에서 외부로 Callback하는 환경 설정도 해야 한다고 하네 
 
   # 사용된 라이브러리 
 
   * react 19 
 
   * typescript 
 
   * tanstack start 
 
   * vite 
 
   # 기여 
 
   * 기여하고 싶으면 Issues나 Pull Requests 보내라고 하니까 
 
   * CONTRIBUTING.md 보라고 하는데 
 
   # 라이센스 
 
   * gpl v3 라이센스를 사용 중이라고 하네
