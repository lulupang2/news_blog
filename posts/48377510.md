### 데이터보우 살펴보기
데이터보우는 ADBC 드라이버 지원하는 DB들을 하나의 인터페이스로 관리할 수 있게 해주는 Rust로 만든 CLI 도구임.
uv나 카고로 설치 가능.
### 데이터보우의 특징
- 30개가 넘는 DB 지원
- 대화형 SQL 쉘 제공
- 쿼리 결과 다양한 포맷으로 내보내기 가능(예: CSV, JSON, 애로우 IPC)
- 프로파일 기능으로 연결 설정 저장하고 불러오기 가능
### 데이터보우 사용 예시
```bash
# 대화형 쉘로 쿼리하기
databow --driver duckdb
# 쿼리 결과 CSV로 내보내기
databow --driver postgresql --uri "postgres://localhost/analytics" --query "SELECT * FROM monthly_report" --output report.csv
# 프로파일로 연결 설정 저장하고 사용하기
databow --profile production-warehouse --query "SELECT * FROM orders"
```
### 데이터보우의 미래
- 추가 기능 개발 중(예: 닷 커맨드, 더 많은 내보내기 포맷 지원 등)
- ADBC 커뮤니티와 함께 성장중임.
