# 10기가 유심 어댑터 등장!
   예전에는 10기가 네트워킹을 쓰려면 비싸고 큰 썬더볼트 어댑터를 사야했음. 그런데 이제는 RTL8159 기반의 10G 유심 3.2 어댑터가 나오니까, 예전 썬더볼트 어댑터는 이제 그만 쓰면 된다고 보면 되겠음.
   # 사이즈 비교
   [이미지](사이즈 비교 이미지 URL)
   이새끼는 내가 시험해 본 WisdPi 에서 나온 80달러짜리 모델이야. 이놈은 5G/2.5G 어댑터보다 2배 더 비싼데, 예전 내가 산 썬더볼트 10G 어댑터보다 반값이야.
   # 결론
   개빡치는 점은, 이새끼는 컴퓨터에 따라서 10Gbps를 못 쓸 수도 있어. 나는 네 개의 컴퓨터에서 시험해 봤는데,
   * Framework 13: USB 4 / USB 3.2 Gen 2
   * MacBook Neo: USB 3.1, USB 2.0
   * M4 MacBook Air: USB 4 / USB 3.1 Gen 2
   * 데스크탑: AMD Ryzen 7900x, B650 메인보드, USB 3.2 Gen 2x2
   근데 USB 포트 정보를 찾는건 개 빡세임. 일부 웹사이트는 USB 3.2 Gen 2인지 3.0인지 안 써 놨고, 윈도우에서는 USB 3.2 Gen 2x2 디바이스도 USB 3.0이라고 떠서 말도 안됨.
   내 컴퓨터에서는 오직 데스크탑에서만 10Gbps를 쓸 수 있었음. 다른 컴퓨터에서는 6-7Gbps밖에 못 씀. 
   # 맥북 테스트
  맥북에서는 이 어댑터를 플러그인하면 바로 인식하고, 드라이버 설치 없이 바로 쓸 수 있었음. 근데 네트워크 설정에서_speed가 2500Base-T로 나오더라 ㅋ.
   # 윈도우 테스트
   윈도우에서는 이 어댑터를 플러그인하면 인식은 하지만, 인터넷에 연결 안 됨. 그래서 Realtek 드라이버 최신 버전을 다운받아 설치했음. 
   # USB 포트 밴드윗드 테스트
   * 데스크탑: 9.5Gbps down, 5Gbps up 
   결론은 이 어댑터는 USB 3.2 Gen 2x2 포트가 있어야만 제대로 된 속도를 낼 수 있다 이말임. 근데 USB 명칭이 너무複雜해서, 컴퓨터에 USB 3.2 Gen 2x2 포트가 있는지 확인하기가 개 힘들더라. 
   * 5G, 2.5G 어댑터가 더 좋은 가치인가?
   *WisPi 5G 어댑터 테스트
   이 에이드터는 30달러였음. 내가 тест해 본 결과, 이 어댑터는 4.6Gbps를 내줌. 10G 어댑터는 1.4배 더 빠르지만, 가격은 2배 더 비쌈. 
   근데 이 어댑터는 RJ45만 지원하고, SFP+는 지원 안 해서, 썬더볼트 어댑터가 아직도 더 낫다 이말임. 
   # 결론
   요약하자면, 이 어댑터는 10기가 네트워킹을 쓰고, RJ45를 쓰고, 컴팩트한 어댑터를 원하면 괜찮은 선택이라고 할 수 있음. 근데 10기가 속도가 필수 아니라면, 2.5Gbps나 5Gbps 어댑터가 더 비용 대비 효율이 좋다고 할 수 있음.
