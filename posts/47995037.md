### 애플 SHARP 브라우저 구동 프로젝트公開!
이 프로젝트는 애플의 [SHARP](https://apple.github.io/ml-sharp/) 모델을 브라우저에서 구동할 수 있게 해줌. [ONNX Runtime Web](https://onnxruntime.ai/docs/web/)을 사용해서 구현했으니, 브라우저에서 모델을 돌려볼 수 있다.

#### 프로젝트 기능
*   이미지를 업로드해서 Gaussian Splat을 생성할 수 있음
*   생성된 결과를预览하고, `.ply` 파일을 다운로드할 수 있음

#### 프로젝트 구성
*   Bun으로 프로젝트를 설치하고 구동할 수 있음
*   React와 TypeScript를 사용해서 UI를 구성했음
*   ONNX Runtime Web worker를 사용해서 모델 추론을 수행함

#### 사용법
1.  프로젝트를 Clone하고, `bun install`를 수행해서 의존성을 설치해줌
2.  `bun dev`를 수행해서 프로젝트를 구동시킴
3.  브라우저에서 업로드한 이미지를 선택하고, Generate Splat 버튼을 클릭해서 결과를预览하고, `.ply` 파일을 다운로드할 수 있음

#### 중요 사항
*   애플 SHARP 모델의 라이선스에 주의해야 함. 연구 목적으로만 사용할 수 있음
*   브라우저에서 모델을 구동할 때, 충분한 디스크 공간과 RAM이 필요함
*   Chrome이나 Edge 브라우저에서 사용을 권장함

#### 기술 스택
*   Bun
*   React
*   TypeScript
*   Vite
*   ONNX Runtime Web
*   GaussianSplats3D viewer

### 프로젝트 상태
현재 프로젝트는 작동하는 프로토 타입 단계에 있으며, 브라우저와 기기의 성능에 따라서 구동 속도가 다를 수 있음.
