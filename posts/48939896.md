# 데이터 사이언스 수학의 정석

데이터 사이언스에 쓰이는 수학을 전체적으로 설명하는 책이라고 보면 됨. 데이터 사이언스에서 수학이 어떻게 사용되고 있는지를 알려주는데, 일단은 이런 장들이 있음:
- 곱등이, 복등이 그리고 놀람 în 고차원
- 특이값 분해와 주성분 분석
- 선형 회귀와 정규화
- 그래프, 네트워크, 클러스터링
- 비선형 차원 감소와 확산 맵
- 랜덤 투영을 이용한 선형 차원 감소
- 데이터 사이언스 최적화
- 분류
- 딥러닝에 대한 수학적 소개
- 그래프 라플라시안의 대 표본 한계
- 커뮤니티
- 측정의 집중에 대한 분석과 가우시안 분석
- 행렬 집중 부등식
- 압축 감지와 희소성
- 저차원 행렬 회복

이 책은 데이터 사이언스에서 사용되는 수학에 대한 기초를 다루고 있기 때문에 데이터 사이언스와 관련된 사람들은 이 책을 참고하면 좋을 거 같음. 데이터 사이언스에 대해 더 알고 싶은 사람들은 꼭 참고해 볼 것임.
