# Krea 2 기술 리포트公開 
   Krea 2 기술 리포트를 공개했다. 이 기술 보고서에서는 Krea 2라고 하는Foundation Model을 소개한다. Krea 2는 아예 새로운 컨셉의 Image Generation MODEL임. 
   **데이터 커리이션 원칙** 
   Krea 2의 데이터 커리이션 원칙은 기존의 IQA 모델과 같은 점수를 기반으로 데이터를 필터링하는 접근 방식과는 다르다. Krea 2의 개발자들은 이러한 접근 방식이 데이터에 암시적인 편향을 유도할 수 있다고 주장한다. 대신, Krea 2는 데이터 커리이션을 위해 다음과 같은 원칙을 따름 
   * 중복된 샘플과 과도하게 temsil되는 개념을 제거한다. 
   * 이미지의 중요한 측면을 일관되게 捕获하지 못하는 샘플을 제거한다. 
   * 편향이나 아티팩트를 유도하는 샘플을 제거한다. 
   * 낮은 해상도에서 신뢰할 수 있는 모델을 구축하기 위해 시각적 복잡도가 너무 높은 샘플을 제거한다. 
   * AI生成 이미지를 제거한다. 
   **모델 아키텍처와 훈련 파이프라인** 
   Krea 2의 모델 아키텍처는 Diffusion Transformer(DiT)라 불리는 새로운 아키텍처를 기반으로 한다. 이 아키텍처는 여러 가지 구성 요소가 함께 작동하여 모델의 출력 분포를 점진적으로efined하게 만든다. 
   훈련 파이프라인은 다단계로 구성되어 있다. 
   1. **pre-training**: 낮은 해상도(256px, 512px, 1024px)에서 데이터를 학습한다. 
   2. **mid-training**: 중간 해상도에서 데이터를 학습한다. 
   3. **supervised fine-tuning(SFT)**: 모델을 더 정교한 태스크를 위해 미세 조정한다. 
   4. **preference optimization**: 사용자의 선호도를 기반으로 모델을 최적화한다. 
   5. **reinforcement learning(RL)**: 강화 학습을 통해 모델을최적화한다. 
   # 결과 
   Krea 2는 이미지 생성 모델 중에서 가장 우수한 성능을 보인다. 사용자 인터페이스도 매우 직관적이고 사용하기 쉽다. 개쩌는 기술임 ㄹㅇ
