# 윈도우 인사이더 프로그램 개편 공지
     마이크로소프트측에서 윈도우 인사이더 프로그램 관련하여 몇 가지 개선 사항을 공지했음.
     먼저 2개월간 윈도우 인사이더 프로그램을 리드하며 유저들과 대화해 본 결과, 두 가지 주제가 두드러졌다고 함.
     첫째는 투명성이 부족하다는 거임. 유저들이 마이크로소프트가 하는 일에 대해 더 잘 समझ고, 진행 상황을_shipping_을 통해 볼 수 있도록 하고 싶다는 뜻임.
     두번 째는 개발자와 유저 간의 자부심이 부족하다는 거야. 유저들이 자신들이 만든 걸 자랑스럽게 생각하고, 인사이더 유저들은 최초로 사용해보고 개발자들에게 피드백을 주는 데 자부심을 느끼고 싶다는 거지.
     근데 마이크로소프트는 그냥 그러라고만 하지 말고 실제로 그럴 수 있게끔 시스템을 만들어나가고 있음.
     예를 들어 윈도우 인사이더 프로그램을 더 쉽게 사용할 수 있도록 설정 페이지를 업데이트했어.
     이제 실험용과 베타 채널이 따로 있기 때문에 유저들이 원하는 기능을 더 쉽게 테스트해 볼 수 있게 됨.
     그리고 피드백 허브도 업데이트했는데, 이제는 피드백을 더 쉽게 남길 수 있음.
     
     # 윈도우 업데이트 개선 사항
     윈도우 업데이트도 개선됐는데, 이번에는 업데이트 중단 시간을 줄이려고함.
     이제는 업데이트 시점을 유저가 결정할 수 있음. 유저가 업데이트를 언제 할지 선택할 수 있게 됐고, 파워 메뉴에서도 업데이트를 쉽게 할 수 있게 바뀜.
     
     # AI 경험 개선
     또한 AI 관련해서도 몇 가지 개선 사항이 있음.
     예를 들어 스니핑 툴이나 포토스 어플에서 AI 관련 버튼을 없애 버림.
     노트패드에서는 그냥 '글쓰기 도구' 이렇게 표시하도록 바꿨음.
     이렇게 AI 기능을 더 의도적으로 사용하도록 바꾸고, 유저들에게 가치 있는 기능들만 제공하겠다는 뜻임.
     
     # 파일 탐색기 개선
     파일 탐색기도 개선 사항이 몇 가지 있음.
     이제 파일 탐색기가 더 빠르고 안정적임. 몇 가지 버그들도 고쳐 줬고, 유저들의 피드백을 반영해 줌.
     
     # 위젯 및 피드 경험 개선
     위젯이나 피드 경험도 개선됐는데, 기본적으로 daha 조용한 설정으로 바뀜.
     유저가 원할 때만 위젯이나 피드가 나타나도록 바뀜. 그리고 유저가 위젯이나 피드를 더 쉽게 커스터마이즈할 수 있게 됨.
     
     # 시스템 성능 개선
     시스템 성능도 개선 사항이 몇 가지 있음.
     예를 들어 메모리 사용량을 줄일 수 있는 방안을 찾아봄. 위젯이나 다른 어플리케이션들이 더 효율적으로 메모리를 사용하도록 바꿨음.
     
     # 앞으로의 계획
     앞으로도 계속해서 윈도우를 개선해 나갈 계획임. 유저들의 피드백을 더 많이 받아서, 더 좋은 운영체제를 만들고 싶다는 거야.
     近日, 마이크로소프트는 윈도우 인사이더 유저들과 만나서 피드백을 받아왔고, 앞으로도 계속해서 만날 계획임.
     유저들이 더 가치 있는 피드백을 줄 수 있도록 시스템을 만들어나갈 계획임.
