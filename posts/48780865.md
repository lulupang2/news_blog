### 스팀 컨트롤러磁鐵 충전 자동화 프로젝트!

이 프로젝트는 개빡친 점이 많음. 먼저 **스팀 컨트롤러 자동 충전**이라는 게 가능해? ㄹㅇ 신기하긴 하더라. 카메라로 컨트롤러와 충전기를 감지하고, 웹 기술로 컨트롤러를 자동으로 충전기에 돌격시킴. 개멋있음.

#### 주요 특징

* **광학 유동 추적**: 컨트롤러와 충전기를 카메라로 추적. 오픈씨브이js 이용.
* **웹하이디 텔레메트리 & 햅틱 내비게이션**: 웹하이디로 컨트롤러와 연동. 내부 듀얼 리니어 레조넌스 액추에이터로 햅틱 신호 전송.
* **근접 크립 모드**: 컨트롤러가 충전기 가까이 오면 햅틱 신호 빈도수 50% 절감. 충전기에 부드럽게 도킹되게 하는 기능임.
* **배터리 상태 폴링**: 콘트롤러 배터리 상태 실시간 확인.

#### 설치 방법

* **니스크 패키지 매니저**로 설치 가능.
* 크로미움 기반 브라우저와 웹캠 필요.
* 프로젝트 시작은 단 한 줄이면 됨: `nix-shell --run "npm install && npm run dev"`

#### 사용 방법

1. 웹캠과 컨트롤러, 충전기를 준비하고, 웹 인터페이스에서 컨트롤러와連結.
2. 자동 추적 버튼 클릭.
3. 컨트롤러가 자동으로 충전기에 돌격!
