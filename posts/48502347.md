### 개요
Kimi K2.7-Code가 공개됨. 얘는 기존 Kimi K2.6를 기반으로 코드 모델링能力을 개선한 거임. 실세계 장시간 코드 작업 능력 개선되고 token 효율성도 개선됨. 

### 모델 특징
- Model Introduction: Kimi K2.7 Code는 Kimi K2.6를 바탕으로 코드 모델링에 집중해서 개발한 거임. 코드 작업 능력 개선되고 복잡한 소프트웨어 엔지니어링 워크플로우에서 최종 작업 완료율 개선됨. 또한 token 효율성 개선되어서 token 사용량 30% 정도 감소했음.
- Model Summary Architecture: 
  - Mixture-of-Experts (MoE) 
  - Total Parameters: 1조 
  - Activated Parameters: 32억 
  - Number of Layers (Dense layer 포함): 61 
  - Number of Dense Layers: 1 
  - Attention Hidden Dimension: 7168 
  - MoE Hidden Dimension (per Expert): 2048 
  - Number of Attention Heads: 64 
  - Number of Experts: 384 
  - Selected Experts per Token: 8 
  - Number of Shared Experts: 1 
  - Vocabulary Size: 160,000 
  - Context Length: 256,000 
  - Attention Mechanism: MLA 
  - Activation Function: SwiGLU 
  - Vision Encoder: MoonViT 
  - Vision Encoder Parameters: 400百만

### 평가 결과
- Benchmark 결과: 
  - Coding Kimi Code Bench v2: 50.9 -> 62.0 
  - Program Bench: 48.3 -> 53.6

  - Agentic Kimi Claw 24/7 Bench: 42.9 -> 46.9 
  - MCP Atlas: 69.4 -> 76.0 
  - MCP Mark Verified: 72.8 -> 81.1
