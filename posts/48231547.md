### 파이어폭스에서 애드프루트 프로젝트 바로 시작하기
 애드프루트 프로젝트를 파이어폭스에서 바로 시작할 수 있게 됨. 웹 시리얼을 이용해 하드웨어 디바이스와 바로 연결하고 코드 작성하고 제어할 수 있음. 파이어폭스 다운받아보기
 파이어폭스는 윈도우 8.1 이전 버전에서 더 이상 지원하지 않음. 파이어폭스 ESR(장기 지원 버전)를 다운받아 사용할 수 있음. 파이어폭스 ESR 64비트 다운받기
 파이어폭스 ESR 32비트 다운받기
 다른 버전 다운받기
 파이어폭스는 맥OS 10.14 이전 버전에서 더 이상 지원하지 않음. 파이어폭스 ESR(장기 지원 버전)를 다운받아 사용할 수 있음. 파이어폭스 ESR 다운받기
 웹 시리얼 프로그래밍에 딱인 파이어폭스
 애드프루트 사용자들에게 아이디어에서 바로 연결된 하드웨어 프로젝트까지의 직접적인 경로 제공. 추가 도구나 복잡한 설정 없이 사용 가능함. 보드와 직접 연결하기
 웹 시리얼로 파이어폭스에서 지원되는 웹 툴에서 바로 호환 가능한 디바이스와 통신할 수 있음. 시도해보기
 설정은 덜고 만들기는 더 많이~
 파이어폭스를 열고 디바이스를 연결하고 별도 데스크톱 앱 없이 코드 작성 시작할 수 있음. 오픈 웹을 위한 파이어폭스
