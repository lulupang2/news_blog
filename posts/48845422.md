# FableCut: 브라우저 비디오 에디터 
     FableCut은 크로미움 기반 브라우저에서 돌아가는 프리미어급 비디오 에디터임. 이거 왜 특별한 걸음? 
     프로젝트 파일이 JSON 문서라서 직접 손도 가능하고, AI 에이전트도 손댈 수 있는 거임 ㅋㅋㅋ. 
     ## 사용법 
     FableCut 사용법은 간단함. 리포지토리를 클론하고, 노드로 서버를 키면 끝임. 
     ```bash
     git clone https://github.com/ronak-create/FableCut.git
     cd FableCut
     node server.js
     ```
     ## 기능 
     FableCut의 기능은 프리미어급임. 비디오 트랙 4개, 오디오 트랙 3개까지 지원되고, Undo/Redo도 가능함. 
     ### 비디오 편집 
     * 다중 선택支持: 마우스 드래그로 영역 선택 가능, Ctrl/Cmd+클릭으로 추가/제외 가능 
     * 클립 이동: Ctrl+A로 전체 선택, Esc로 전체 취소 가능 
     * 클립 삭제: Delete버튼으로 클립 삭제 가능 
     * 클립 분할: S키로 클립 분할 가능 
     * 시간 표시: beat & cue 마커 지원, 엣지 스냅 지원 
     ### 오디오 ویرایش 
     * 실시간 오디오 波形 표시 지원 
     *Canvas 화면비율 프리셋 지원: 16:9, 9:16, 4:5, 1:1 지원 
     ### 필터 및 이펙트 
     * 12개의 필터 프리셋 지원: 시네마틱, 틸-오렌지, 노ワ르, 빈티지 등 
     * 어저스터層 지원: 클립 그레이드 지원, 프리미어급 지원 
     ### 캡션 및 텍스트 
     * 키네틱 캡션 지원: 타이핑, 단어 팝업 등 
     * 네온 글로우 지원: 틱톡 캡션 지원 
     * 폰트 편집기 지원: 시스템 폰트, 커스텀 폰트 지원 
     ### AI 지원 
     FableCut은 AI 에이전트도 지원함. 
     CLAUDE.mdファイル에 따라 에이전트가 직접 프로젝트 파일을 수정할 수 있음.
