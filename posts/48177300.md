### 요약
- 1979년에 작성된 운영체제 보안의 근간을 다룬 논문이公开됨
- 보안성이 수학적으로 증명된 운영체제(OS)를 만드는 것이 가능한지에 관해 연구함
- 현재 운영체제보안 기술의 근간이 되는 논문으로 평가됨

### 운영체제 보안
- 운영체제(OS)는 컴퓨터 하드웨어와 소프트웨어를管理하는 소프트웨어임
- 운영체제는 컴퓨터 하드웨어와 사용자 간의 인터페이스 역할을 함
- 운영체제의 보안은 컴퓨터 시스템 전체의 보안을 좌우함

### 논문의 내용
- 운영체제의 보안성을 수학적으로 증명하는 방법을 제시함
- 운영체제의 보안성을 증명하기 위해 필요한 조건과 가정 등을 정의함
- 운영체제의 보안성을 증명하는 예시를 보여줌

### 운영체제 보안 기술의 중요성
- 운영체제 보안 기술은 컴퓨터 시스템의 보안을 좌우함
- 운영체제 보안 기술의 발전은 컴퓨터 시스템의 보안성을 강화할 수 있음
- 운영체제 보안 기술은 현재에도 활발하게 연구되고 있음
