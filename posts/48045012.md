### 디스크리스 리눅스 부팅하기
리눅스를 디스크 없이 부팅하는 방법에 대해서 알아보자. 
대부분의 리눅스 사용자들은 리눅스를 디스크에 설치해 사용한다. 하지만 디스크를 사용하지 않고도 리눅스를 부팅할 수 있다. 이 방법을 디스크리스 부팅이라고 한다.

#### 왜 디스크리스 부팅을 사용할까?
디스크리스 부팅을 사용하는 이유는 여러 가지가 있다. 
- 디스크를 사용하지 않아도 되기 때문에 디스크 용량을 늘리지 않아도 된다.
- 디스크가 고장나더라도 리눅스를 부팅할 수 있다.
- 여러 대의 기기를 같은 리눅스를 사용할 수 있다.

#### 디스크리스 부팅을 위한 준비
디스크리스 부팅을 하기 위해서는 몇 가지 준비가 필요하다.
- ZFS를 사용하는 서버가 필요하다.
- iSCSI 타겟이 필요하다.
- PXE 부팅을 지원하는 네트워크 카드가 필요하다.

#### ZFS 설정
ZFS를 설정하는 방법은 다음과 같다.
```bash
zpool create tank /dev/disk/by-id/${DISK_ID}
zfs create -V 32G tank/debian-disk-12700k
```
#### iSCSI 설정
iSCSI를 설정하는 방법은 다음과 같다.
```bash
targetcli
cd /backstores/block
create debian-disk-12700k /dev/zvol/tank/debian-disk-12700k
cd /iscsi
create iqn.2026-05.xyz.716697.pve-vt:tank-debian-disk-12700k
```
#### PXE 설정
PXE를 설정하는 방법은 다음과 같다.
```bash
apt install apache2 git ansible tftpd-hpa targetcli-fb
git clone https://github.com/netbootxyz/netboot.xyz.git
cd netboot.xyz
ansible-playbook -i inventory site.yml
```
#### 결론
디스크리스 부팅을 사용하면 디스크를 사용하지 않아도 리눅스를 부팅할 수 있다. ZFS, iSCSI, PXE를 사용하면 디스크리스 부팅을 쉽게 설정할 수 있다. 이 방법은 디스크를 사용하지 않아도 되기 때문에 디스크 용량을 늘리지 않아도 된다. 또한 디스크가 고장나더라도 리눅스를 부팅할 수 있다. 여러 대의 기기를 같은 리눅스를 사용할 수 있다.
