# 오니스 1.0이 뭐냐면?
   
   자가 개선되는 오픈 소스 모델이래. 요즘 코딩 모델들이 다 이런 식이라서, 이 모델도 자가 개선된다는 거임.
   
   # 성능은 어때?
   
   오니스는 다양한 벤치마크를 수행했어. 예를 들어, Terminal-Bench 2.1, SWE-Bench, NL2Repo 등에서 최고의 성능을 나타냈음.
   
   # 사용 방법은?
   
   오니스를 사용하고 싶다면, 최근 버전의 라이브러리들을 설치해야 할거래. 예를 들어, Transformers 5.8.1 이상, vLLM 0.19.1 이상, SGLang 0.5.9 이상이 필요함.
   
   그리고, 오니스는 다양한 크기의 모델을 제공해 줌. 예를 들어, 9B, 35B, 397B 등이 있어. 따라서, 사용하는 사람의 필요에 따라 모델 크기를 선택할 수 있음.
   
   # 결론은?
   
   오니스는 자가 개선되는 오픈 소스 모델로, 다양한 벤치마크에서 최고의 성능을 나타냈어. 따라서, 코딩 모델을 필요로 하는 사람들에게는 유용한 도구가 될 수 있을 것 같음.
   
   그리고, 오니스를 사용하고 싶다면, 라이브러리 설치부터 사용 방법을 잘 알아보는 것이 좋을 것 같아. 그리고, 다양한 크기의 모델을 제공해 주니까, 사용하는 사람의 필요에 따라 모델 크기를 선택해서 사용하면 좋을 것 같음.
