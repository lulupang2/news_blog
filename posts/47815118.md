# Kdenlive의 현주소 - 2026년 리포트

자, 얘들아 주목! 오픈소스 영상 편집기 중에서 그나마 쓸만하다고 평가받는 **Kdenlive**가 2025년에 얼마나 영혼을 갈아 넣었는지, 그리고 앞으로 뭘 할 건지 보고서를 냈길래 할배가 핵심만 싹 정리해왔다. 오픈소스라고 무시하지 마라, 생각보다 내실이 꽉 찼거든.

---

## 1. 개요: "우린 멈추지 않아"

2025년에 Kdenlive 팀은 새로운 기능을 넣는 것보다 **안정성(Stability)**에 우선순위를 뒀어. 사실 기능만 덕지덕지 붙이다가 픽픽 꺼지면 개빡치잖아? 그 밸런스를 잡으려고 노력을 많이 했다네. 홈페이지도 싹 갈아엎고, 2002년부터 쌓인 근본 넘치는 기록들도 다 복구해놨어. 아주 칭찬해.

## 2. 2025년 릴리즈 하이라이트 (3대장)

일년에 세 번 큰 업데이트가 있었는데, 내용이 꽤 혜자야.

### 25.04.0: AI 맛 좀 볼래? 
*   **SAM2 기반 배경 제거:** 요즘 핫한 `SAM2(Segment Anything Model 2)`를 써서 클릭 몇 번으로 객체 따는 기능을 넣었어. 이제 누끼 따느라 고생 안 해도 된다는 소리지.
*   **오디오 파형 300% 가속:** 파형 생성 속도가 300%나 빨라졌어. 편집할 때 파형 안 떠서 멍 때리는 시간 줄어들었다. ㄹㅇ 개꿀.
*   **OpenTimelineIO:** 이제 다른 유료 편집 프로그램들이랑 프로젝트 주고받는 게 더 편해졌단다.

### 25.08.0: 내실 다지기
*   이때는 새로운 기능보다는 **버그 수정**에 몰빵했어. 무려 300개의 커밋이 들어갔고, 15개 이상의 크래시(꺼짐 현상)를 잡았대. 
*   **오디오 믹서 리뉴얼:** 고해상도(HiDPI) 모니터 쓰는 애들을 위해 UI 디자인 싹 고쳤으니까 이제 눈 덜 아플 거야.

### 25.12.0: UI 맛집으로 거듭나기
*   **웰컴 스크린:** 처음 켰을 때 최근 프로젝트나 가이드 쉽게 보라고 웰컴 화면 만들었어.
*   **유연한 도킹 시스템:** 위젯 옮기고 숨기는 게 훨씬 편해졌고, 너희만의 레이아웃을 파일로 저장해서 공유도 할 수 있게 됐어.

## 3. 앞으로의 계획 (Roadmap 2026)

26.04 버전이 곧 나오는데, 여기서 눈여겨볼 기능들이 좀 있어.

*   **모니터 미러링:** 풀스크린 모드에서 작업할 때 다른 화면에 똑같이 띄워주는 건데, 협업할 때 개편함.
*   **트랜지션 미리보기:** 효과 넣기 전에 대충 어떻게 보일지 미리 보여주는 기능이야. 시간 낭비 줄여주겠지?
*   **10/12비트 컬러 지원 (중요!):** 이게 드디어 온다! 영상 퀄리티 좀 따지는 애들이 목 빠지게 기다리던 건데, 이제 색보정 좀 제대로 할 맛 나겠네.
*   **Dopesheet 도입:** 키프레임 편집 시스템을 완전히 갈아엎고 있대. 한 번에 여러 효과 키프레임을 관리할 수 있게 된다니까 기대해봐.

## 4. 커뮤니티 화력 및 기타 지표

*   **다운로드 수:** 2025년 한 해 동안 홈페이지에서만 **1,150만 회** 넘게 다운로드됐어. 리눅스 패키지 매니저까지 합치면 훨씬 많겠지?
*   **남극의 5인:** 야, 남극에서 이거 쓰는 사람이 5명이나 있단다. 얘들아, 혹시 보고 있으면 거기서 뭐 편집하는지 댓글 좀 달아줘라. 펭귄 브이로그라도 찍니? ㅋㅋㅋ
*   **Microsoft Store 입점:** 이제 윈도우 스토어에서도 쉽게 설치할 수 있게 MSVC 컴파일 작업 중이래. 윈도우 유저들 접근성 개떡상 예정.
