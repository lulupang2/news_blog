### 블렌더 5.2 LTS 개요
   - Long Roar Ahead ノ드 기반 물리 엔진, 온라인 자산 라이브러리, 다양한 신규 기능들 포함
   - 2년간의 LTS 업데이트 지원함
   -_download_Blender 5.2 LTS [다운받기](https://www.blender.org/download/)
   
   ### 신규 기능들
   - **ジオメトリ ノード**
     - 음악 파일을 노드로 불러와서 애니메이션과 시뮬레이션에 활용할 수 있는 Sample Sound Frequencies ノード 추가됨
     - Mesh Bevel ノード 추가되어 노드 기반으로 메시의 에지나 버텍스를 세부적으로 컨트롤할 수 있음
     - Hair Dynamics 등에서의 노드 기반 물리 엔진 구현
   - **클로쥬**
     - 클로즈를 쉽게 추가할 수 있는 노드 기반의 클로즈 모디파이어 지원
     - 노드를 활용한 노드 기반 물리 엔진 구현
   - **노드 툴**
     - 노드 툴의 입력 사항이 기억되어 노드를 반복해서 사용할 때 편리함
     - 노드 툴의 입력 사항이 파이썬으로도 할당될 수 있음
   
   ### 노드 기반 물리 엔진
   - 노드 기반 물리 엔진이 도입되어 사용자 정의ノードGroupId를 활용한 노드 기반 물리 시뮬레이션 구현이 가능해짐
   - XPBD ソルバー 노드를 통해 노드 기반 물리 엔진을 커스텀할 수 있음
   
   ### 기타
   - **데이터 블록**
     - 데이터 블록의 이름을 뷰어 노드에 표시할 수 있음
     - 데이터 블록을 소켓에 比較하고 None과 비교할 수 있음
   - **사이클스**
     - 텍스처 캐시 기능이 추가되어 대용량 텍스처를 로딩할 때 메모리 사용량과 로딩 시간을 크게 줄일 수 있음
     - 쉽게 사용할 수 있는 Auto Generate 기능도 지원됨
   - **이피**
     - 이피도 많은 신규 기능을 지원하며, 노드 기반 물리 엔진, 노드 기반 클로즈 등 블렌더의 주요 기능들과도 잘 호환됨
