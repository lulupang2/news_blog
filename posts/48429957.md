# 키밴치(키밴치)
   키밴치는 키밸류 스토어(KVS)라는 것을 테스트 해주는 도구임. 
   쉽게 말해서 데이터를 빠르게 저장하고 불러오는 것을 테스트 해주는 툴이라고 생각하면 됨.
   이 툴에는 여러 가지功能이 있다고 하는데, 
   *   **스크립트 작성**: 이 툴에서 사용되는 스크립트는 루아(Lua)로 작성할 수 있음. 쉽게 말해서, 사용자가 직접 테스트 스크립트를 작성할 수 있는 거임.
   *   **성능 측정**: 이 툴을 통해 사용자는 KVS의 성능을 측정할 수 있음. 성능을 측정할 수 있는 항목은 쓰루풋(throughput)과 지연 시간(latency)임. 
   *   **병렬성**: 이 툴에서는 사용자가 테스트를 병렬로 진행할 수 있음. 쉽게 말해서, 여러 개의 테스트를同时 진행할 수 있는 거임.
   *   **확장성**: 키밴치 자체가 확장 가능한 툴임. 사용자는 다양한 KVS 엔진들과 함께 사용할 수 있음. 쉽게 말해서, 여러 가지 DB를 테스트해볼 수 있는 거임.
   # 사용법
   사용하는 방법은 간단함. 
   1.  키밴치를 빌드한다.
   2.  워크로드를 설정한다.
   3.  키밴치를 실행한다.
   # 빌드
   키밴치를 빌드하는 방법은 간단함. 
   *   まず, 사용자는 키밴치의 소스 코드를 다운받는다.
   *  그다음, 사용자는 make 명령어를 사용하여 키밴치를 빌드한다. 
   # 워크로드 설정
   워크로드를 설정하는 방법도 간단함. 
   *   사용자는 루아 스크립트를 작성하여 워크로드를 설정할 수 있음.
   # 실행
   키밴치를 실행하는 방법은 간단함. 
   *   사용자는 키밴치의 명령어를 사용하여 키밴치를 실행할 수 있음.
