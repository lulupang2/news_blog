# 로마 제국의 인구 지도 작업 중임 ㅋㅋㅋ
   ## 왜 이렇게 하는 거야?
   로마 제국 인구 지도작업중임. 
   데이터 출처는 EDCS(에피그라피 데이타베이스 클라우스-슬라비)인데, 여기에 AI 기술을 적용해서 
   이름을 추출한 거임.
   ## 데이터 특징
   * 총 25만개의 인슐레이션 데이터
   * 개인 이름, 성별, 지위 등 정보 추출
   * 지도에 마커 표시하고, 클릭해서 정보 볼 수 있음
   ## 사용 방법
   * 지도에 Zoom_in_해서 지역별 정보 볼 수 있음.
   * 마커 클릭하면 개인 이름, 인슐레이션 텍스트, 번역본等 볼 수 있음.
   * 검색창과 필터로 데이터 필터링하고 CSV나 JSON으로 내릴 수 있음.
   ## 주의 사항
   * 이름 추출 정확도는 80~85% 수준임.
