### 닌텐도 스위치, 네트워크 스위치로 바꾸기 ㅋㅋㅋ
어제 생각난 건데, 스위치를 실제 네트워크 스위치로 바꾸면 재밌을 것 같다는 거임. 좀 생각해 봤다가 하드웨어로는 가능할 것 같고, usb 포트도 있으니까. 문제는 소프트웨어인데, 전에 스위치 해킹 좀 했지만 이건 거의 Scratch부터 구축해야 해서 귀찮았음. 그런데 스위치 루트가 있길래, 이걸로 안드로이드랑 리눅스를 깔면 쉽게 네트워크 스위치 구축할 수 있을 거임. 
먼저 우분투 스위치 루트 깔고, usb 이더넷 다ングル 연결하고, 터미널 열면 이더넷 카드 인식 안 된다고함. 그럴 줄 알았어. 결국에는 드라이버 문제라서, 스위치 루트 업데이트 파일 다운받고,_boot 폴더랑.boot.scr 삭제하고, 업데이트한 이후에야 이더넷 카드 인식하고, 브릿지 인터페이스 설정하고, ip 링크 설정하고, 결국엔 정상 작동함. 테스트 컴퓨터에선 브라우저랑 스피드 테스트도 정상임. 
브릿지 설정하는법은
```bash
brctl addbr br0 # 브릿지 인터페이스 추가
brctl addif br0 enx8416f90eae20 enx9cebe816215d # 브릿지에 이더넷 카드 추가
ip link set br0 up # 브릿지 인터페이스 업
```
이다 이중으로 네트워크 구성해서 속도는 90mbps 라는 결론 thôi. 100mbps 다ングル라서 그 이상은 안나오는 거 같음.
