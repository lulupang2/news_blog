# 포스트그레스 19 미리보기
   포스트그레스 19에 대한 이야기는 이제부터 시작입니다. 
   포스트그레스 버전마다 특징이 있습니다. 어떤 버전은 하나의 큰 기능이 중심이되는 것이고, 어떤 버전은 오랜 시간 지속된 문제를 해결하는 것이고, 어떤 버전은 작은 개선 사항들이 가득찬 것입니다. 
   포스트그레스 19는 모든 것들이 결합된 버전 같습니다. REPACK CONCURRENTLY는 큰生产 データ베이스에서 품질을 개선하는 업그레이드입니다.
   또한 SQL 프로퍼티 그래프 쿼리도 주목할 만한 기능입니다. 논리적 복제가 더 완전해졌습니다.
   VACUUM, EXPLAIN, COPY, 파티셔닝, 모니터링, 성능 및 플래너 동작이 개선되었습니다. 
   이것들은 큰 업데이트는 아니지만 실제로 데이터베이스를 운영할 때 도움이 됩니다.
   ### REPACK 
   REPACK은 포스트그레스의 핵심 기능으로 추가되었습니다. 
   REPACK CONCURRENTLY는 큰 생산 데이터베이스에서locks 없이 테이블을 재조직 할 수 있는 중요한 기능입니다.
   ### 파티셔닝 
   파티셔닝은 포스트그레스에서 데이터를 효율적으로 관리할 수 있게 해줍니다. 
   포스트그레스 19에서는 파티션을 합치거나 분할할 수 있습니다.
   ### 논리적 복제 
   논리적 복제는 데이터를 복제하고 동기화를 유지할 수 있게 해줍니다. 
   포스트그레스 19에서는 시퀸스 값이동을 가능하게 해주어 데이터 일관성을 유지할 수 있습니다. 
   ### Autovacuum 
   Autovacuum은 포스트그레스의 자동적인 유지 보수 기능입니다. 
   포스트그레스 19에서는 Autovacuum이 더智能해 졌습니다.
   ### SQL 프로퍼티 그래프 쿼리 
   SQL 프로퍼티 그래프 쿼리는 그래프 형식의 데이터를 관리할 수 있게 해줍니다. 
   포스트그레스 19에서는 SQL 프로퍼티 그래프 쿼리를 지원합니다. 
   ### COPY 
   COPY는 데이터를 복사하고 이동하는 기능입니다. 
   포스트그레스 19에서는 COPY 기능이 개선되었습니다. 
   COPY FROM에서 헤더 라인을 건너뛸 수 있고, COPY TO에서 JSON 형식으로 출력할 수 있습니다. 
   이와 같은 기능들은 포스트그레스 19의 특징을 보여줍니다.
