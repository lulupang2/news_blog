### TanStack 보안 사고
TanStack npm이 보안 사고로 악성 코드가 심어졌대. 
#### 영향 받은 패키지
42개의 패키지에 악성 코드가 심어졌고, 84개의 버전이 영향을 받았대. 
#### 공격 방법
GitHub Actions 캐시 중독과 OIDC 토큰 추출을 통해 공격이 일어났대. 
#### 사고 처리
사고가 발생한 직후에 외부 연구자에 의해 공격 코드가 발견되었고, npm 보안팀에報告되었다고 함. 
#### 피해 규모
まだ 피해 규모가 정확하게 계산되지 않았지만, 공격이成功했다면 많은 개발자들이 영향을 받았을 거야. 
#### 대응책
npm에서 제공하는 보안 가이드를 따라야하고, 영향 받은 패키지의 버전을 확인해야대. 
#### 결론
개발자들은 항상 보안을 생각해야하고, npm 패키지의 버전을 확인하는 습관을 가져야해.
