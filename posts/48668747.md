### 구형컴 살리기 리눅스 사용법

리눅스(dist)로 구식컴을 살려내는 방법에 대한 정리 글임. 
일단 리눅스로 구형컴을 살리려면 제일 먼저 **기존 컴퓨터 사양을 확인**해야함. 사양을 확인하는 명령어는 주로 `free -h`, `lscpu`, `lsblk`임.
이 명령어를 사용하면 사용 가능한 RAM과 CPU 아키텍처, 저장장치 용량 정보를 얻을 수 있음.
 RAM이 2GB 미만이면 antiX나 Puppy Linux 같은 최대한 가벼운 리눅스를 써야함. 
RAM이 2GB ~ 4GB 사이면 Lubuntu나 Linux Lite를 쓰면 되고, 
4GB 이상이면 Xubuntu나 Linux Mint 같은 좀 무거운 리눅스도 문제없이 돌아감.

### 리눅스 구형컴 최적화 방법

리눅스를 구형컴에 설치했다면 최적화를 통해 성능을 높일 수 있음.
주로 `zram`을 설정하고 `swappiness` 값을 조정하는 것이 있음. 
`zram`은 RAM에 압축된 swap 공간을 생성하여 디스크 I/O를 줄이는 기술임. 
`swappiness` 값은 리눅스가swap을 사용하는 빈도를 조절하는 값임.
 구형컴에서 `swappiness` 값을 낮출 경우, 디스크에 쓸 때 발생하는 성능 저하를 최소화할 수 있음.
 
### 리눅스 구형컴 데스크탑 환경

리눅스 데스크탑 환경은 사용자 경험에 큰 영향을 줌.
 LXQt, Xfce, MATE 등 가벼운 데스크탑 환경을 사용하면 구형컴에서도 부드러운 성능을 기대할 수 있음.
 LXQt는 RAM 사용량이 가장 적게는 480MB 정도이고, 
Xfce는 약 650MB 정도 사용하고 MATE는 중간 정도의 RAM 사용량을 가짐.
 각각의 데스크탑 환경은 특징과 사용감이 다르므로, 직접 사용해보고 적절한 걸 선정하는게 좋을듯.
