### git 커밋들을 폴더로 마운트해서 NFS로 사용할 수 있다?
요즘에 git을 좀 더 직관적으로 관리하고 싶어서 찾아봤는데, git 커밋들을 폴더로 마운트해서 NFS로 사용할 수 있다고? ㄹㅇ 신기하다.
예를 들어, [git-commit-folders](https://github.com/borkdude/git-commit-folders)라는 프로젝트를 사용하면 git 커밋들을 폴더로 마운트해서 NFS로 사용할 수 있단다. 
이 프로젝트는 git 커밋들을 폴더로 마운트해서 NFS로 사용할 수 있도록 해준다고 하는데, 어떻게 작동하는지 보자.
그냥 커밋들을 폴더로 만들어주면 돼서 간단하다고? 그럼 만들어 보자. 
-commits 폴더 안에 커밋들을 만들어주면 돼. 
-커밋 폴더 안에 커밋의 내용을 담아주면 돼. 
-그리고 커밋 폴더를 NFS로 마운트해서 사용하면 돼.

### 사용 예시
-commits 폴더 안에 만들어진 커밋 폴더를 listar해서 볼 수 있단다. 
-commits 폴더 안에 만들어진 커밋 폴더의 내용을 볼 수 있단다. 
-commits 폴더 안에 만들어진 커밋 폴더의 파일을 수정할 수 있단다.

### 문제점
-파일 시스템을 마운트하는 건 어렵단다. 
-파일 시스템을 마운트하는 데 필요한 코드를 작성하는 건 어려울 수 있단다. 
-파일 시스템을 마운트하는 데 필요한 코드를 작성하는 데 시간이 걸릴 수 있단다.

### 해결책
-파일 시스템을 마운트하는 코드를 작성하는 건 어려울 수 있지만, [git-commit-folders](https://github.com/borkdude/git-commit-folders)라는 프로젝트를 사용하면 쉽게 할 수 있다고 하네. 
-파일 시스템을 마운트하는 코드를 작성하는 데 시간이 걸릴 수 있지만, [git-commit-folders](https://github.com/borkdude/git-commit-folders)라는 프로젝트를 사용하면 빠르게 할 수 있다고 하네.
