### 라즈베리 파이 제로에서 디스크 없이 웹 사이트 호스팅하는 방법

라즈베리 파이 제로에서 Alpine Linux를 이용해 디스크 없이 웹 사이트를 호스팅하는 방법을 알려줌. 

### 하드웨어 준비

* 라즈베리 파이 제로 v1.3
* 512MB 이상의 마이크로 SD 카드
* 이더넷 해드(Waveshare Ethernet HAT) 또는 OTG 어댑터
* 이더넷 케이블
* 마이크로 USB 전원 코드/전원 어댑터
* 쿨한 케이스(선택 사항)

### 마이크로 SD 카드 준비

마이크로 SD 카드를wipe하고 FAT32 포맷으로 변경해 줌.
```bash
diskutil eraseDisk FAT32 ALPINE MBRFormat /dev/disk4
```
그리고 Alpine Linux 이미지를 마이크로 SD 카드에 복사해 줌.
```bash
tar xzf alpine-rpi-*.tar.gz -C /Volumes/ALPINE
```
마이크로 SD 카드를 라즈베리 파이 제로에 꽂고, 키보드와 모니터를 연결해 줌. 

### Alpine Linux 설치

Alpine Linux가 부팅되면, 루트 계정으로 로그인해 줌.
```bash
setup-lbu mmcblk0p1
mkdir -p /media/mmcblk0p1/cache
setup-apkcache /media/mmcblk0p1/cache
lbu commit -d
```
그리고 `setup-alpine`를 실행해 줌.
```bash
setup-alpine
```
키보드, 호스트 네임, 네트워크, DNS, 시간대를 설정해 줌. 

### 소프트웨어 설치

* darkhttpd
```bash
doas apk add darkhttpd
```
* nginx
```bash
doas apk add nginx
```
* rsync
```bash
doas apk add rsync
```
### 포트 열기

라즈베리 파이 제로의 IP 주소와 포트를 열어줌. 

### TierHive VPS 설정

TierHive VPS를 설정해 줌. HAProxy Edge 서비스를 이용해 라즈베리 파이 제로에 트래픽을 전달해 줌. 

### TLS 설정

TierHive의 자동 SSL 갱신 기능을 이용해 TLS를 설정해 줌. 

### 백업

라즈베리 파이 제로의 마이크로 SD 카드를 백업해 줌.
```bash
ssh root@YOUR-PI-ZERO-IP "dd if=/dev/mmcblk0 bs=4M" > zero-backup.img
```
