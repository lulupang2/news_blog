### Qwen 3.6 27B: лучший выбор для локальной разработки

기존 로컬 모델들은 마음에 들지 않았는데, Qwen 3.6 27B가 다름. Qwen 3.6는 두 가지 버전으로 나눠져 있는데, mixture-of-experts 모델인 Qwen 3.6 35B A3B와 dense 모델인 Qwen 3.6 27B가 있음. 하지만 더 느리긴 하지만, 27B 모델이 더 강력하다고 생각함.

이 모델은 매우 빠르긴 한데, 내 맥북에서 테스트해 본 결과, 매우 뜨겁다는걸 알게 되었습니다. ㅋㅋㅋ 그래서 폰에 달린 열감지기를 잡고 사진을 찍었음. Qwen 3.6는 vraiment 강력하긴 한데, 내 컴퓨터를 매우 뜨겁게 해서, 맥북이 녹아버릴 거 같았음. 근데, 성능은 정말 좋음. 정말로 Qwen 3.6 27B를 추천함.

Simon Willison이 penguins on a bicycle라는 테스트를 했는데, Qwen 3.6 35B A3B와 Qwen 3.6 27B를 비교했는데, 결과는 매우 놀라웠음. Qwen 3.6 27B는 제한된 문장 작성과 같은 테스트에서 뛰어난 성능을 보임. 기존 GPT-4.5 모델들이 할 수 있는 일을, Qwen 3.6 27B도 할 수 있었음.

### 실제 테스트

OpenCode에서 hexagonal minesweeper를 생성하라고 부탁했는데, 바로 성공했음. 그런데, mixture-of-experts 모델은 빠르긴 하지만, 제대로 된 패키지를 만들지 않고, 단일 index.html 파일을 생성했음.  Qwen 3.6 27B는 실제 작업에도 매우 유용했음. Maciej Cielecki의 요청으로 테스트를 했는데, 매우 만족스러운 결과를 얻었음.

### 실제 코드 실행

로컬에서 Qwen 3.6를 실행하는 방법은 sangat 쉽게 설명할 수 있음. llama.cpp를 통해 모델을 실행할 수 있음. llama.cpp는 직접 모델을 로컬에서 실행할 수 있는 오픈소스 툴임. llama.cpp를 사용하면 Ollama를 사용할 필요가 없음. 그리고, Ollama를 사용하는 것을 추천하지 않음. Hugging Face에서 적절한 모델을 다운로드하고, llama.cpp를 사용하면 모델을 실행할 수 있음.

 llama-server -hf unsloth/Qwen3.6-27B-MTP-GGUF:Q8_0 \ 
  --spec-type draft-mtp -ngl 999 -fa on -c 65536 --port 8080

성능은 매우 중요함. 몇 가지 테스트를 실행했는데, 결과는 매우 놀라웠음. Qwen 3.6 27B는 very 빠른 모델임. Macbook Max M5 128 GB에서 테스트를 했는데, 매우 빠른 성능을 보임. 하지만, 35B A3B 모델은 3배 더 빠르긴 한데, Quality는 27B 모델이 더 높음.
