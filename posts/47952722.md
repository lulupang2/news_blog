# HERMES.md 라고 적으면 EXTRA 사용량으로 과금 당하는 버그 존나 웃기네 ㅋㅋㅋㅋ
   - 커밋 메시지에 'HERMES.md' 가 있으면 EXTRA 사용량으로 과금 당하는 버그가 있음.
   - 이 버그는 Claude Code 에서만 발생하고, 다른 코드 에디터에서는 발생하지 않음.
   - 'HERMES.md' 라고 적으면 EXTRA 사용량으로 과금 당하지만, 'hermes.md' 라고 적으면 정상적으로(plan quota) 과금 됨.
   - 이 버그를 발견한 사람은 Systematic binary search 라는 방식을 사용해서 발견했음.
   - 이 버그는 macOS 에서만 발생하고, 다른 OS 에서는 발생하지 않음.
   - 이 버그를 해결하려면 커밋 메시지에 'HERMES.md' 라는 문자열을 제거해야 함.
   - 개빡치는 점은 이 버그가 EXTRA 사용량으로 과금이 된 후에야 알 수 있고, ERROR 메시지도 없다는 거임 ㅋㅋㅋ
