# 러스트言語를 리스프風으로 바꿔보자 ㅋㅋㅋㅋ

러스트言語를 리스프風으로 바꿔보는 프로젝트에요. 이 프로젝트에서는 러스트의 세맨틱을 유지하면서 리스프风의 문법을 적용했습니다.ownership, borrowing, lifetimes, generics, traits, pattern matching 등 러스트의 모든 기능을 s-expr로 표현할 수 있어요.

## 설치 방법

`git clone https://github.com/ThatXliner/rlisp.git`로 저장소를 클론하고, `cd rlisp`로 디렉토리로 이동한 뒤 `cargo install --path .`로 설치할 수 있어요.

## 사용 방법

`rlisp compile file.lisp`로 파일을 러스트 코드로 컴파일하고, `rlisp build file.lisp`로 컴파일과 빌드를 동시에 할 수 있어요. `rlisp run file.lisp`로 컴파일, 빌드, 실행을 모두 한번에 할 수 있어요.

## 문법

리스프风의 문법을 사용해서 러스트 코드를 작성할 수 있어요. 예를 들어, `(fn add ((x i32) (y i32)) i32 (+ x y))`는 `fn add(x: i32, y: i32) -> i32 { x + y }`와 동일해요.

`(struct Point (x f64) (y f64))`는 `struct Point { x: f64, y: f64 }`와 동일해요.

`(enum Option (T) (Some T) None)`는 `enum Option<T> { Some(T), None }`와 동일해요.

이 프로젝트는 그냥 재미로作った 거라고는 하지만, 실제로 도움이 될 수도 있어요.예를 들어, 매크로를 작성할 때 도움이 될 수 있어요. 매크로는 컴파일 타임에 실행되는 s-expr를 반환하는 함수에요.

## 매크로

매크로는 컴파일 타임에 실행되는 s-expr를 반환하는 함수에요.예를 들어, `(defmacro double (x) (quasiquote (+ (unquote x) (unquote x))))`는 `(double 21)`를 `(+ 21 21)`로 확장해요.

## 결론

요약하자면, 러스트言語를 리스프風으로 바꿔보는 프로젝트에요. 이 프로젝트에서는 러스트의 세맨틱을 유지하면서 리스프風의 문법을 적용했습니다.ownership, borrowing, lifetimes, generics, traits, pattern matching 등 러스트의 모든 기능을 s-expr로 표현할 수 있어요. 매크로를 작성할 때 도움이 될 수 있어요.
