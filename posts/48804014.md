# 클라우드플레어 워커스 캐시登場!
     클라우드플레어가 새로운 캐시 기술인 워커스 캐시를 출시했음. 
     이 기술은 워커에 캐시를 두는 기술로, CPU 시간을 절약하고, 페이지 로딩 시간을 단축시키는 효과가 있음.
     ## 워커스 캐시란?
     워커스 캐시는 클라우드플레어의 워커 기능을 이용해서 캐시를 저장하고, 관리하는 기술임. 
     워커는 클라이언트의 요청을 받았을 때, 클라우드플레어의 캐시를 먼저 확인하고, 캐시에 히트가 없는 경우에만 서버 렌더링을 수행하게 됨.
     이렇게 하면 CPU 시간을 절약할 수 있고, 페이지 로딩 시간을 단축시킬 수 있음. 
     ### 워커스 캐시의 특징
     * 캐시를 저장하고 관리하는 기술
     * CPU 시간을 절약하고, 페이지 로딩 시간을 단축시킴
     * 워커를 이용해서 캐시를 관리
     * 클라우드플레어의 캐시를 이용해서 캐시 히트率을 높임
     *캐시 미스의 경우 워커가 서버 렌더링을 수행
