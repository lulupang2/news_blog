# 소개
     나사에서 찍은 태양과 달 영상을 실시간으로 보여주는 대시보드가 만들어졌 븅신임. 이 대시보드는 태양과 달의 최근 영상을 보여주고, 태양의 활동을 실시간으로 업데이트 해줌. 
     # 기능
     * 태양의 활성 영역과 코로나를 anzeigen 해줌
     * 달의 위상과 высота를 보여줌
     * 태양과 달의 실시간 영상을 보여줌
     * 태양의 플레어와 코로나 질량 방출을 감시할 수 있음
     # 사용 방법
     * 나사에서 제공하는 데이터를 이용해 실시간으로 태양과 달의 정보를 보여줌
     * 태양과 달의 영상을 4K로 볼 수 있음
     * 태양의 活動 영역과 코로나를 查看할 수 있음
     # 업데이트
     * 안드로이드 버전은 이미 출시되어 있으며, 애플 버전은 곧 출시될 예정임. 
     # 개인 정보 보호
     * 이 앱은 사용자의 위치를 저장하지 않으며, 사용자의 데이터를 수집하지 않음.
     * 사용자의 위치는 사용자 기기에만 저장되어 있다고 함. 
     # 결론
     이 대시보드는 나사에서 제공하는 데이터를 이용해 태양과 달의 정보를 실시간으로 보여주는 유용한 도구임. 개인 정보 보호에 신경을 쓴 점도 좋고, 안드로이드 버전은 이미 출시되어 있으니 관심 있는 사람들은 꼭 다운로드 받아보길 추천함 ㅋㅋㅋ
