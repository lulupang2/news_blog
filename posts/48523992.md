###무료 DB 설계 툴

ブラ우저에서 돌아가는 무료 ER 다이어그램 툴이 나왔다는 소식이 임. SQL을 ER 다이어그램으로 변환해주는 툴이라고 하는데, 브라우저에서 돌아가서 편하다고 함. 100% 로컬에서 돌아가니까 업로드는 걱정하지 않아도 된대.

###特色

* 무료임
* 브라우저에서 돌아감
* 100% 로컬에서 돌아가서 업로드는 안함
* PostgreSQL, MySQL, SQLite, SQL Server 지원
* 드래그 앤 드롭으로 테이블을 이동할 수 있음
* PNG나 SVG로 내보내기 가능

###사용법

1. 브라우저에서 접속
2. SQL을 작성하거나 붙여넣기
3. 자동으로 ER 다이어그램이 생성됨
4. 드래그 앤 드롭으로 테이블을 이동
5. PNG나 SVG로 내보내기
