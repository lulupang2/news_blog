오픈 소스 동기화 툴 오픈 rsync가 출시되었습니다! 리눅스용 rsync랑 완전히 호환되니까 걱정 말아요. 
   오픈 rsync는 BSD 라이선스로 출시돼서 자유롭게 사용하실 수 있습니다. 
   오픈 rsync는 오픈BSD تیم에서 개발했으며, 오픈BSD의 공식 rsync 구현체로 채택되었습니다. 
   오픈 rsync는 리눅스용 rsync랑 완전히 호환되므로 기존 rsync 사용자들도 쉽게 전환할 수 있습니다. 
   오픈 rsync의 주요 기능으로는 다음과 같은 것들이 있습니다: 
   * 리눅스용 rsync랑 완전히 호환됩니다. 
   * BSD 라이선스로 출시되어 자유롭게 사용하실 수 있습니다. 
   * 오픈BSD의 공식 rsync 구현체로 채택되었습니다. 
   * 쉽게 설치하고 사용할 수 있습니다. 
   오픈 rsync를 사용하려면 다음과 같이 설치하시면 됩니다: 
   ```bash
   % ./configure
   % make
   # make install
   ```
   그뒤에 오픈 rsync를 사용하려면 다음과 같이 명령어를 입력하시면 됩니다: 
   ```bash
   % openrsync -lrtp remote:foo/bar ~/baz/xyzzy
   ```
   이 명령어는 foo/bar 디렉토리를 ~/baz/xyzzy 디렉토리와 동기화합니다. 
   오픈 rsync는 매우 강력한 동기화 툴이므로 다양한 용도로 사용할 수 있습니다. 
   예를 들어, 파일을 백업하거나 동기화할 때 사용할 수 있습니다. 
   또한 오픈 rsync는 오픈 소스라서 개발자들이 직접 수정하고 개선할 수 있습니다. 
   따라서 오픈 rsync는 매우 유용한 툴입니다.
