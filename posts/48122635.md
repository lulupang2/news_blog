# 무료 도메인 얻는 법
미국에서 지역 도메인이라는게 있음. 예를 들어 seattle.wa.us 이런식으로 지역명.주명.us 형식으로 된 도메인.
이 도메인은 1992년에 처음 만들어졌는데, 정부에서 계약한 업체가 관리하고 있어.
이 도메인을 등록하려면 미국 시민권자거나 미국에서 업무를 하는 단체여야 함.

## 지역 도메인 등록 절차
1. **도메인 선택**: 등록하고 싶은 지역 도메인을 먼저 찾아야 함. 이미 등록된 도메인은 [이 리스트](링크)에서 확인할 수 있음.
2. **네임서버 구하기**: 무료 네임서버를 제공하는 업체가 필요함. 아마존 라이트세일에서 무료 네임서버를 받을 수 있음.
3. **도메인 등록 폼 작성**: Interim .US Domain Template v2.0 폼을 작성해야 함. 폼을 작성하면서 네임서버 주소를 입력해야 함.
4. **도메인 등록**: 폼을 작성하고 도메인 등록 대행 업체에 전송하면 됨. 업체에서 확인 后에 도메인이 등록됨.
5. **DNS 설정**: 도메인 등록 후 DNS 설정을 해야 함. 라이트세일에서 DNS 설정을 할 수 있음.

### FAQ
* 도메인 등록시 주소가 필요한데, 실제로 그 주소에 살고 있어야 하나요?
 + 글쎄요. 등록 폼에 주소를 넣는다고 하더라도, 실제로 확인하는 사람은 없을듯 함.
* WHOIS 요청시 주소가 노출되나요?
 + 아니요. WHOIS 요청시에는 등록 대행사 정보만 노출되고, 실제 주소는 노출되지 않음.
