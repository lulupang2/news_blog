# 님 프레스토란?
님 프레스토는 님 언어용 효율적인 REST API 구현 라이브러리야. 이 프레임워크는 님 언어의 공식 패키지 매니저인 님블을 사용해서 설치할 수 있음. 
## 설치 방법
 `$ nimble install https://github.com/status-im/nim-presto.git` 라고 입력하면 쉽게 설치할 수 있다는 점이 꿀잼 ㅋㅋ

## 라이센스
MIT 라이센스와 아파치 라이센스 2.0 중에 하나를 선택해서 사용할 수 있으니 참고해. ㄹㅇ 편하네

## 기능
님 프레스토는 REST API를 구현하기 위한 다양한 기능을 제공하고 있다. 개빡치는 점은 문서와 예제 코드가 충분히 제공되어 있단 점임. 그래서 개발자들이 쉽게 사용할 수 있고, 자신의 프로젝트에 바로 적용할 수 있다는데 meaningful함. 

## 결론
님 프레스토는 님 언어 개발자들을 위한 강력한 도구인 거 같음. REST API를 손쉽게 구현할 수 있다는 점이 매력적이고, 개발자들의生产성을 향상시킬 수 있는 잠재력을 가진 라이브러리라고 본다. 님 프레스토를 사용하면 보다 효율적이고 쉽게 REST API를 구현할 수 있을 거라 생각함. 그래서 님 언어 개발자라면 반드시 경험해봐야 할 라이브러리라고 생각한다는 거임.
