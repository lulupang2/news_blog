# 겜인 API 파일 검색 updates ㄹㅇ개빡치네
   - 이제 멀티모달 지원됨. 커스텀 메타데이터도 가능하고 페이지 단위의 인용도 가능해졌음.
   - 이런 기능들이 개발자들이 구조화되지 않은 데이터를 더 효율적으로 관리할 수 있게 해줄 거임.

   ## 겜인 API 파일 검색 개요
   - 겜인 API 파일 검색은 개발자들이 구조화되지 않은 데이터를 더 효율적으로 관리할 수 있도록 도와주는 툴임.
   - 이제 멀티모달 지원을 추가했음. 이 말은 텍스트와 이미지 데이터를 함께 처리할 수 있다는 거야.
   - 커스텀 메타데이터도 지원됨. 이건 개발자들이 데이터에 원하는 속성을 추가할 수 있게 해주는 거임. 예를들면 '부서: 법務팀' 같은 속성을 추가할 수 있음.

   ## 멀티모달 지원
   - 멀티모달 지원이란 텍스트와 이미지 데이터를 함께 처리할 수 있는 거임.
   - 이 기능은 개발자들이 더 풍부한 정보를 검색할 수 있도록 해줌.
   - 예를들면 광고 대행사에서 특정 이미지를 찾고 싶은데 이미지 파일명이 기억나지 않을 수도 있음. 이럴 때 자연어로 된 이미지 검색을 할 수 있어 더 효율적으로 이미지를 찾을 수 있음.

   ## 커스텀 메타데이터
   - 커스텀 메타데이터는 개발자들이 데이터에 원하는 속성을 추가할 수 있게 해주는 거임.
   - 이 기능은 개발자들이 데이터를 더 효율적으로 관리할 수 있도록 해줌.
   - 예를들면 '부서: 법務팀' 같은 속성을 추가할 수 있음. 이러면 특정 부서의 데이터를 쉽게 찾을 수 있음.

   ## 페이지 단위의 인용
   - 페이지 단위의 인용은 검색된 결과를 원래 출처와 연결할 수 있게 해주는 거임.
   - 이 기능은 개발자들이 더 투명하고 신뢰할 수 있는 검색 결과를 제공할 수 있도록 해줌.
   - 예를들면 특정 PDF 파일에서 특정 문장을 검색했을 때 원래 출처가 어디인지 정확하게 알 수 있음.
