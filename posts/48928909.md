# TTT3R: 3D Reconstruction as Test-Time Training
 TTTR은 CUT3R를 개량해서 3D 재건 기술을 테스트하면서 바로 연습할 수 있는 방법이야. 이 기술은 간단한 상태 업데이트를 통해서 längeneralization을 업그레이드함.
 
 ### 시작하는 방법
 1. TTTR 저장소를 클론하고, 환경을 설정해봐. `git clone https://github.com/Inception3D/TTT3R.git` 하고, `conda create -n ttt3r python=3.11 cmake=3.14.0`로 환경을 설정하고, `conda activate ttt3r`로环境을 활성화 해주면 됨.
 2. 필요한 라이브러리를 설치해. `conda install pytorch torchvision pytorch-cuda=12.1 -c pytorch -c nvidia`하고, `pip install -r requirements.txt`로 필요한 라이브러리를 설치하고, `pip install evo pip install open3d`로 평가에 필요한 라이브러리를 설치해야 해.
 
 ### 모델을 사용하는 방법
 모델을 사용하는 방법은 간단해. `CUDA_VISIBLE_DEVICES=6 python demo.py --model_path MODEL_PATH --size 512 \ --seq_path SEQ_PATH --output_dir OUT_DIR --port 8080 \ --model_update_type ttt3r --frame_interval 1 --reset_interval 100 \ --downsample_factor 100 --vis_threshold 5.0`으로 실행하면 되고, 모델 경로와 크기, 시퀸스 경로, 출력 디렉토리, 포트 등 몇 가지 파라미터만 정해주면 되니까 참고해.
 
 ### 평가하는 방법
 모델을 평가하는 방법은 eval.md 문서를 참고하면 돼. 문서에는 평가에 필요한 모든 정보가 담겨 있으니까.
 
 ### 감사의 말
 이 저장소는 CUT3R, Easi3R, DUSt3R, MonST3R, Spann3R, Viser 등 여러 저장소를 참고해서 만들었어. 해당 저장소의 기여자들에게 감사를 표하노.
 
 ### 인용
 이 기술을 사용해서 연구를 하면, 논문의 인용을 부탁해. `@article{chen2025ttt3r, title={TTT3R: 3D Reconstruction as Test-Time Training}, author={Chen, Xingyu and Chen, Yue and Xiu, Yuliang and Geiger, Andreas and Chen, Anpei}, journal={arXiv preprint arXiv:2509.26645}, year={2025} }`로 인용하면 되고.
 
 # 마무리
 TTTR은 3D 재건 기술을 테스트하면서 바로 연습할 수 있는 방법이고, CUT3R를 개량해서 längeneralization을 업그레이드했어. 이 기술을 사용하면, 3D 재건 기술을 더 쉽고 효율적으로 사용할 수 있을 거야. 참고해서 활용해 봐.
