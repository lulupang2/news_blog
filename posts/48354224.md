### 케피르 C 컴파일러 개망한듯
오늘부터 케피르 C 컴파일러 개발을 개별 개발자 혼자서 진행할 거임. 무슨 일이죠? 먼저 말씀드리자면, 난 이 결정에 매우 마음이 아프지만, 프로젝트를 지속시키기 위해 필요하다고 생각함. 현재까지 공개된 코드는 그대로 유지될 건데, 새로운 개발 업데이트는 이제부터 개인 개발자 혼자서 진행할 거임. 기존 코드에서 발견된 버그는 여전히 修复할 건데, 개선할 내용이 있으면 계속 업데이트 할 예정이임. 

개발자 혼자서 개발하면 좋을 거 없나? 그럼 안타깝지만, 개발을 하면서 너무 힘들었음. 이제 케피르 C 컴파일러 개발은 개발자 혼자서 하는 프로젝트가 된 거임. 이 결정이 맞다고 생각해. 개발자 혼자서 개발하면, 개발에 더 집중할 수 있고, 더 좋은 개발을 할 수 있을 거라 생각함.
이 프로젝트를 시작했을 때, 난 그냥 재미로 시작한 거였어. 하지만 이제는 개발이 너무 복잡해서 혼자서 개발할 수 없을 것 같았어. 그래서 결정한 거임.
 난이제부터 케피르 C 컴파일러 개발을 혼자서 진행할 건데, 더 좋은 개발을 할 수 있을 거라 믿음. 관심 있는 사람들은 계속 지켜봐줬으면 좋겠음. 
케피르 C 컴파일러 개발을 지지해 주신 모든 분들께 감사하고, 앞으로도 개발을 지속해나갈 거임. 케피르 C 컴파일러가 더 gute 개발이 되도록 노력할 거임. 감사합니다.
