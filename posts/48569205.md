### 극혐급 미니미니 UI 라이브러리 MicroUI
- MicroUI는 cực히 작은Immediate-mode UI 라이브러리야. 
#### 특징
- 작다: 1100줄밖에 안되는데ㅋㅋㅋㅋ
- 메모리 관리도 직접할 수 있어. 그냥 지정한 사이즈 내에서만 돌아감. 
- 기본 控件(window, scrollable panel, button, slider, textbox, label, checkbox, wordwrapped text)가 다 있지. 
- 렌더링 시스템은 별개로 되어있어. 사각형, 텍스트 그리기만 가능하면 됨. 
#### 레이아웃 시스템
- 쉽게 커스텀 컨트롤을 추가할 수 있게 설계되었음. 
```c
if (mu_begin_window(ctx, "My Window", mu_rect(10, 10, 140, 86))) {
  mu_layout_row(ctx, 2, (int[]) { 60, -1 }, 0);
  mu_label(ctx, "First:");
  if (mu_button(ctx, "Button1")) {
    printf("Button1 pressed\n");
  }
  mu_label(ctx, "Second:");
  if (mu_button(ctx, "Button2")) {
    mu_open_popup(ctx, "My Popup");
  }
  if (mu_begin_popup(ctx, "My Popup")) {
    mu_label(ctx, "Hello world!");
    mu_end_popup(ctx);
  }
  mu_end_window(ctx);
}
```
