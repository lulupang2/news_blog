### Claude/GPT 대체 로컬 모델 써보신 분?
Has anyone here fully swapped Claude/GPT for a local model as their main coding tool, not just for side experiments? <- 이 글의 핵심이니까 기억해 두셈. 
요약하면 Claude/GPT 말고 로컬 모델로 코딩할 때 쓸만한 모델이 있는지 물어보는 글이당. 
만약 쓰신 분들이 있으면 설정과 성능(예를 들어 토큰당 시간 등)을 공유해 줬으면 좋겠다는 거임.
만약 로컬 모델로 코딩할 때 쓸만한 게 있으면 개빡치는 점은 없겠지? ㅋㅋㅋ
그럼 어떤 로컬 모델을 쓸 수 있는지 알려줘야겠다. 
근데 로컬 모델은 데이터를 로컬에 저장해야 하니까 개인 데이터가 노출될 걱정이 있긴 한데, 그럼에도 불구하고 쓸만한 로컬 모델이 있는지를 묻는 글이당. 
로컬 모델을 쓸 때 성능이 얼마나 좋은지도 궁금하당. 그러니까 로컬 모델을 이용해 본 분들은 성능 공유좀 해주셈. 
어떤 분이 로컬 모델을 이용해 본 걸 공유해 주시면 좋겠당. 
만약 로컬 모델을 이용해 본 분들이 계시면 공유해 주세요. 
로컬 모델은 개인情報 유출에 취약하다는 점은 있지만, Claude/GPT를 대체할 만한 가치가 있을 수도 있겠다는 거임. 
그럼 로컬 모델을 사용할 때 설정과 성능에 대해 자세히 설명해 주시면 좋을 것 같당. 
그럼 Claude/GPT를 대체할 만한 로컬 모델을 찾아보겠다. 
로컬 모델을 대체할 만한 다른 모델이 있으면 그건 어떨까? 
그러면 로컬 모델을 쓰는 사람들은 설정이나 성능 같은 걸 공유해도 괜찮겠지? 
클라우드 모델말고 로컬 모델을 쓸만한 이유가 있을까? 
만약 로컬 모델을 쓸만한 이유가 있으면 그것도 공유해 줬으면 좋겠당.
