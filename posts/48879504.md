# 엠에잠 터미널 에뮬레이터 Ghostel.el에 관하여
엠에잠 사용자라면 알겠지만, 터미널 에뮬레이터가 필요할 때가 있음. 그런데 최근에 나온 Ghostel.el이란걸 사용해 본 결과, 진짜 편하더라 ㅋㅋ. Ghostel.el은 libghostty-vt를 사용해서 만든 터미널 에뮬레이터로, native dynamic module과 엘리스프(Elisp)로 구성되어 있음.

## 주요 기능
- 터미널 에뮬레이션
- 프로세스 모델
- 북마크
- 링크 및 파일 감지
- 클립보드
- 입력
- 패스워드 프롬프트 감지
- 쉘 통합 기능
- 렌더링
- 인라인 이미지(Kitty 그래픽스 프로토콜)
- 엘리스프에서 쉘 호출
- 알림 및 진행 상태
- 컬러 팔레트

## 설치 방법
MELPA에서 `ghostel`을 설치하거나, `use-package`를 사용해서 설치할 수 있음.

## 사용법
- `M-x ghostel`로 터미널 에뮬레이터를 열 수 있음.
- 기본 입력 모드는 세미 차 모드(semi-char mode)로, 대부분의 키가 터미널로 전송되지만 일부 키는 엘리스프로 전송됨.
- 차 모드(char mode)에서는 모든 키가 터미널로 전송됨.
- 라인 모드(line mode)에서는 엘리스프 버퍼처럼 사용할 수 있음.

## 확장 기능
- 에블 모드(Evil-mode) 확장
- 컴파일 모드(Compilation mode) 확장
- 이쉘(Eshell) 통합
-คอม인트(Comint) 통합
- 엘리스프 입력 방법
