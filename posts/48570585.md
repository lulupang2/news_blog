### 컴퓨터 과학에서 디스패치 테이블 작성하기
디스패치 테이블 작성은 컴퓨터 과학에서 매우 중요한 개념임. 특히 bytecode 인터프리터를 작성할 때 많이 사용됨.
본격적으로 시작하기 전에, 본 글은 Eli Bendersky의 웹사이트의 일부임을 알림.
본인이 파이썬 소스 코드를閲覧하다가 발견한 컴퓨티드 götu_extension에 대해介绍함.
컴퓨티드 götu는 GCC 확장 기능 중 하나로, 코드를 더 효율적으로 작성할 수 있게 해줌.
본인은 간단한 예제로 이를 테스트해 봄.

### 간단한 바이트코드 인터프리터 작성
먼저, 간단한 바이트코드 인터프리터를 작성할 것임.
이 인터프리터는 단순히 bytecode를 실행하는 것임.
본人が 작성한 인터프리터는 다음과 같은 코드임:

```c
#define OP_HALT 0x0
#define OP_INC 0x1
#define OP_DEC 0x2
#define OP_MUL2 0x3
#define OP_DIV2 0x4
#define OP_ADD7 0x5
#define OP_NEG 0x6

int interp_switch(unsigned char* code, int initval) {
    int pc = 0;
    int val = initval;
    while (1) {
        switch (code[pc++]) {
            case OP_HALT: return val;
            case OP_INC: val++; break;
            case OP_DEC: val--; break;
            case OP_MUL2: val *= 2; break;
            case OP_DIV2: val /= 2; break;
            case OP_ADD7: val += 7; break;
            case OP_NEG: val = -val; break;
            default: return val;
        }
    }
}
```
이 코드는 단순한 bytecode 인터프리터임.
switch 문을 사용하여 bytecode를 실행함.

### 컴퓨티드 götu 사용
그러나, 본인은 컴퓨티드 götu를 사용하여 코드를 더 효율적으로 작성할 수 있음을 발견함.
컴퓨티드 götu는 GCC 확장 기능 중 하나로, 코드를 더 효율적으로 작성할 수 있게 해줌.
본人が 작성한 컴퓨티드 götu 사용 예제는 다음과 같음:

```c
int interp_cgoto(unsigned char* code, int initval) {
    /* The indices of labels in the dispatch_table are the relevant opcodes */
    static void* dispatch_table[] = {
        &&do_halt,
        &&do_inc,
        &&do_dec,
        &&do_mul2,
        &&do_div2,
        &&do_add7,
        &&do_neg
    };

    #define DISPATCH() goto *dispatch_table[code[pc++]]

    int pc = 0;
    int val = initval;
    DISPATCH();
    while (1) {
        do_halt:
            return val;
        do_inc:
            val++;
            DISPATCH();
        do_dec:
            val--;
            DISPATCH();
        do_mul2:
            val *= 2;
            DISPATCH();
        do_div2:
            val /= 2;
            DISPATCH();
        do_add7:
            val += 7;
            DISPATCH();
        do_neg:
            val = -val;
            DISPATCH();
    }
}
```
이 코드는 컴퓨티드 götu를 사용하여 bytecode를 실행함.
goto 문을 사용하여 코드를 더 효율적으로 작성할 수 있게 해줌.
