## 레드햇 npm 패키지에 악성 코드 도는 중 ㅋㄷㅋㄷ
레드햇에서 배포하는 npm 패키지들이 뚫렸다고 합니다. [SECURITY]: Malicious npm releases detected across @redhat-cloud-services/ scope #492 이슈를 보면, 레드햇의 여러 패키지가 악성 코드에 감염됐다는 것이 밝혀졌습니다.

Affected Packages Package Compromised Version 이런 거임.
* @redhat-cloud-services/chrome 2.3.1
* @redhat-cloud-services/compliance-client 4.0.3
* @redhat-cloud-services/config-manager-client 5.0.4
* ... 이거슨 패키지 이름과 버전임.
개빡치는 점은 레드햇에서 배포하는 패키지들이라고 모든게 다 뚫렸다는 거임.

https://www.stepsecurity.io/blog/multiple-redhat-cloud-services-npm-packages-compromised 이런 링크도 있는데,
거기 가서 보시면 자세한 내용을 알 수 있습니다.
ㅇㅇ, 레드햇에 이런 일이 생긴 건 상당히 민감한 문제라서, 빠른 시일 내에 수정할 계획임.
ㄹㅇ, npm 패키지들을 최신 버전으로 유지하고, 주기적으로 업데이트하는 습관을 가지세요.
개짜증나는 거지만, 이렇게 해서 보안을 강화해가는 거임.
