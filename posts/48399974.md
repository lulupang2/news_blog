### KVarN은 뭐냐면
화웨이가 개발한 KV 캐시를 더 잘 쓰라고 만든 기술임. KV 캐시는 어떤 모델이든 잘 돌아가게 해주는 중요한 역할을함. 근데 KV 캐시를 더 잘 쓰게 해줘야 모델의 성능이 더 좋아짐. 
그런데 기존의 KV 캐시 기술은 모델의 성능을 더 좋게 할 수는 있어도 KV 캐시의 용량을 더 늘려주지는 못함. 그러다가 화웨이가 KVarN이라는 기술을 개발해서 KV 캐시의 용량을 3~5배까지 늘려줄 수 있게 됨. 
이 기술은 KV 캐시의 용량을 늘려주고 KV 캐시의 성능도 더 좋게 해줌. 그러니까 모델의 성능도 더 좋아지고 KV 캐시의 용량도 더 많이 쓸 수 있으면서 성능도 좋은 거임.

### KVarN을 사용하는 방법
KVarN은 사용하는 방법도 간단함. 기존의 KV 캐시 기술을 사용하는 방법과 거의 같음. 그냥 KV 캐시의 dtype을 kvarn_k4v2_g128로 바꿔주면 됨. 그러면 KVarN이 자동으로 KV 캐시의 용량을 늘려주고 성능을 더 좋게 해줌.

### KVarN의 장점
KVarN은 KV 캐시의 용량을 3~5배까지 늘려줄 수 있음. KV 캐시의 성능도 더 좋게 해줌. 그러니까 모델의 성능도 더 좋아지고 KV 캐시의 용량도 더 많이 쓸 수 있음. 
또한 KVarN은 사용하는 방법도 간단하고 KV 캐시의 dtype만 바꿔주면 자동으로 KV 캐시의 용량을 늘려주고 성능을 더 좋게 해줌. 
그러니까 모델의 성능을 더 좋게 하고 싶은 사람, KV 캐시의 용량을 더 많이 쓸 수 있게 하고 싶은 사람은 KVarN을 사용하는게 좋을 것 같음.
