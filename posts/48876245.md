# RISCBoy는 개념딴 열려있는 소스 개인용 게임 콘솔 임
  * RISC-V 호환 CPU, 레스터 그래픽스 pipeline, 디스플레이 컨트롤러 등으로 구성됨
  * 다른 칩 인프라스트럭처도 포함되어 있음 (버스 패브릭, 메모리 컨트롤러, UART, GPIO 등)
  * PCB 레이아웃은 KiCad로 설계됨
  * 게임보이라고 생각하면됨. 근데 RISC-V를 사용해 만든 것이라서 차세대 게임보이라고 보면 되지 않을까? ㅋㅋ
  * 설계는 합성 가능한 Verilog 2005로 작성되었음
  * iCE40-HX8k FPGA에 적합하도록 설계되었음. 이 칩은 LUT4 기반으로 7680개의 논리 요소를 가지고 있음
  * RV32IMC 명령어 세트를 지원하며, RISC-V 컴플라이언스 스위트에 통과함
  * M-모드 CSRs, 예외 처리, 벡터 외부 인터럽트를 지원함
  * HDL/HDL 테스트를 위한 Git 서브모듈 사용
  * 클로닝은 Git에서 지원하는 리커시브 서브모듈 업데이트를 사용할 수 있음
  * 빌딩은 RV32IMC 툴체인을 사용함
  * 시뮬레이션은 Xilinx ISIM 14.x에서 진행됨
  * 테스트는 HDL 및 소프트웨어 레벨에서 진행됨
  * PCB는 iTead의 4층 5x5cm 프로토타이핑 서비스와 호환됨
  * 현재 Rev A PCB를 사용하고 있으며, Rev B는 게이트웨어와 부트로더가 준비되면 진행할 예정임
  * 개발용 하드웨어는 Snowflake FPGA 보드와 비슷한 형태임
