# MCP서버 인증 고단해졌다? 이제 간단해진대요!
     ## 문제점부터 말하자면
     기존 MCP인증 시스템은 사용자마다 인증서 발급해주고, 서버도 하나하나 모두 연결해줘야 했음. 이거야말로真的 복잡하고 귀찮은 과정인건 둘째치고 보안 취약점도 엄청 많았지. 각 서버마다 인증 정보를 저장해야하고, 보안팀도 한눈에 관리하기 어려웠음.
     ## 인증 과정 개선된 거
     이제는 EMA라는 시스템 도입해서 아이디 하나로 모든 MCP서버에 자동으로 연결이 됨. 아이디 제공업체에서 일괄적으로 인증서 발급해주면, 사용자는 로그인 한번으로 모든 MCP서버에 접속할 수 있게 된대요.
     # EMA 시스템 어떻게 작동하냐면
     * 사용자가 아이디 하나만 로그인하면, EMA 시스템이 자동으로 MCP서버에 연결해줌.
     * 보안팀은 인증서 발급 및 관리를 중앙에서 손쉽게 관리할 수 있음.
     # 이제는 MCP서버도 더 쉽게 사용할 수 있잖아?
     ## 기존 시스템의 한계점을 보완한 대안
     * 이제는 인증 정보 여러개 저장할 필요가 없고, 보안팀도 중앙에서 관리하기 쉽도록 바뀐대요.
