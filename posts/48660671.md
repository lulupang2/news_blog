# 와인 설치하기
요약하면 와인이 꼭 필요하댔어. 개인용 OS에 와인이 없으면 윈도우 게임들을 돌릴 수가 없다고 함.
 cogmind 이런 게임도 와인이 없으면 돌아가지를 않는데, 이 와인을 설치하기 위해선 MinGW를 다운받았음. MinGW를 다운받은 뒤에야 와인이 제대로 돌아가기 시작했어.
 notch의 게임도 돌아감 ㅋㅋ

 # 와인 설치하면서 겪은 문제점
 우선 와인은 OpenGL을 지원해야 돌아가는데, 이 OpenGL은 Astral OS의 Mesa 포트가 지원하지 않더라고. 그래서 Mesa를 패치한 뒤 DRI를 이용해서 GPU랑 직접 통신하게 했어. 그리고 EGL도 연결해서 와인이 초기화 될때 문제가 발생하지 않도록 했음.

 # 32비트 윈도우 프로그램 구동
 cogmind는 32비트 프로그램이라서, 64비트 OS에서 돌아가게 하려면 WoW64 모드를 지원해야 함. WoW64는 32비트 윈도우 프로그램을 64비트 프로세스에서 돌아가게 해줌. 이거 구현하기 위해선 LDT를 지원해야하고, 이를 위해서는 커널을 수정해야했음.

 # 와인으로 돌린 게임들
 FTL, 스팀, iexplore.exe, 팩토리오, 스푸키의 점프 스카레 맨션 등등 여러 게임들을 돌려봤음. 이중 일부 게임은 제대로 돌아가는데, 다른 게임은 안 돌아가더라고. 아마도 와인 구현이 아직 부족한 것 같아.

 # 이제 와인 구현 잘 되면 스팀도 돌아가겠지?
 와인을 이제 거의 다 구현했으니, 앞으로는 스팀 구현하고, 커널 최적화, 버그 픽싱에 집중할 계획임. 그리고 다른 사람들도 와인 구현에 참여해줄 거라고 기대하고 있음. 그리고 나도 개인용 OS에 와인을 설치해서 윈도우 게임들을 돌려 볼 계획인디?
