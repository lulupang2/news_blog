# AdBuster 2.0 PRO란?

   AdBuster 2.0 PRO는 오프라인 음성 안정화 엔진과 IR 제어를 사용하여 티비 볼륨을 자동으로 조절해주는 프로그램 임. CEPA Logic을 사용하면 인공지능과 같은 결정 체계를 통해 티비 음량을 조절함.

   # 사용 방법

   1. 최신 릴리즈를 다운로드해서 압축을 풀고, Start.bat를 실행하면 자동으로 모든 모듈이 실행됨.

   - [ Broadlink IR 디바이스를 설정해서 вол륨을 제어해줘야 함 ]

   2. 티비를 보다가突然 광고音량이크면, AdBuster가 자동으로 볼륨을 낮춤. 정상적인 방송이 다시 나오면 원래대로 복구됨.

   - [ 다 클라우드나 서버 같은 거 쓰지 않고, 모든 처리를 로컬에서만 진행함 ]

   - [ irl 볼륨 자동 조절 프로그램 중에, 유일하게 CEPA Logic + 머신러닝 + IR 제어를 사용하는 프로그램 임 ]

   # 기술적인 부분

   - AdBuster는 마이크에서 오디오 신호를 받아서, RMS값과 피크값을 계산하고, 동态스피크 검출 등을 함.

   - 머신러닝 엔진을 사용해서, بلند한 광고와 정상적인 방송을 구분해냄.

   - CEPA Logic을 사용해서, 인공지능과 같은 결정 체계를 통해 티비 음량을 조절함.
