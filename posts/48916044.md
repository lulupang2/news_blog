# 개빡치는 뉴스
   고프로사가 거의 끝장 난 거 같음. 
   요새 카메라ตลาด은 너무 경쟁 심화된 것 같고 고프로는 옛날 명성이 الآن은 없는 거 보니 echt 가슴이 아프네.
   ## 요약
   - 고프로 사가 사실상 파산 직전임.
   - 사장은 자기가 자기 회사에 2000만 달러를 대출해줌. 
   - 하지만 이건 사실상 죽음의 키스인 거임.
   - 고프로사에 지금까지 나온 신제품은 사실상 망한 것 같고, 
   - 앞으로 어떻게 될지 가늠하기 어려움.
   ### 추가 정보
   - 고프로사에 경쟁사들이太 많음. 
   - 예를 들어 인스타360은 고프로사보다 훨씬 좋다고 소문남. 
   - 고프로사에 지금까지 개발한 기술은 사실상 쓰레기인 거 같고, 
   - 앞으로 어떻게 될지 가늠하기 어려움
