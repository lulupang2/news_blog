# 스페이스X AI 코딩 에이전트 '그록 빌드'란?

   스페이스X에서 만든 AI 코딩 에이전트인 '그록 빌드'가 오픈소스로 공개 됨. 이家伙는 터미널 기반으로 작동하며, 코드 분석, 파일 편집, 웹 검색, 실행 및 관리 등을 담당함. 진짜 사람처럼 코드를 수정하고, 웹을 검색하고, 터미널 명령어를 실행시키고, 작업을 관리함. ㄹㅇ 신기방기한 기능이다.

   

   # 그록 빌드 설치하기

   그록 빌드를 설치하는 방법은 간단함. macOS, 리눅스, 윈도우즈용 미리 빌드된 바이너리를 제공하며, Rust를 사용해 소스코드부터 빌드할 수도 있음. curl이나 PowerShell로 단 몇 줄의 명령어로 쉽게 설치할 수 있음. 예를 들어, macOS나 리눅스에서安装하려면 단순히 `curl -fsSL https://x.ai/cli/install.sh | bash`만 치면 됨. ㅇㄴ.

   

   # 그록 빌드의 기능

   그록 빌드는 정말 많은 기능을 지원함. 코드 분석, 파일 편집, 웹 검색, 터미널 명령어 실행, 작업 관리 등 다양한 기능이 있으며, 머쓱하게도 코드를 편집하고, 터미널 명령어를 실행시키며, 웹을 검색하는 등 기본적인 기능 외에도 확장 기능을 지원하기 때문에 진짜 사람이 개발한 것처럼 동작함. ㄹㅇ 개신기한 기능인 것 같음.

   

   # 그록 빌드의 장점

   그록 빌드의 가장 큰 장점은 오픈소스로 공개된다는 점임. 즉, 누구나 그록 빌드를 사용할 수 있고, 개발에 참여할 수 있으며, 새로운 기능을 추가할 수도 있음. 또, 스페이스X에서 개발한만큼 정말 쓸모있는 기능을 지원할 것 같은 느낌이 듬. 개빡치는 점은 없고, 그냥 좋눔 ㅋㅋㅋㅋ
