# macOS 바irtual machine 성능 테스트 
   
   최근 애플 실리콘 맥에 macOS 가상 머신(Virtual machine)을 설치했을 때의 성능에 대해 테스트를 해봤다고?
   
   테스트 장비는 맥 미니 M4 Pro고, 여기에 14개의 코어와 48GB 램을 달고, 2TB의 내부 SSD도 달았음.
   
   그럼 성능은 어떻게 되는겨?
   
   ### 성능 테스트
   
   테스트 결과, VM은 거의 호스트와 비슷한 성능을 내놓았는데, 
   싱글 코어 성능은 VM에서 98%의 성능을 기록했고, 
   멀티 코어 성능은 VM에서 13,222점을 기록해서 호스트와 비슷한 수준의 성능을 내놓음. 
   
   GPU 성능은 호스트 대비 95% 수준이었는데, 
   이는 맥북 네오와 같은 저전력 장치에서도 원활한 성능을 기대할 수 있다는 뜻임. 
   
   다만 신경 쓸 점은 VM에서 나은 엔진 성능이 호스트에 비해 떨어지는 것인데, 
   이는 macOS에서 AI 관련 작업을 수행할 때 CPU와 GPU를 사용하는 것이 더 나을지도 모른다는 것을 의미함. 
   
   ### 최소 사양
   
   맥북 네오와 같은 저전력 장치에서도 macOS VM을 설치할 수 있을까?
   
   테스트 결과, 맥북 네오에서도 macOS VM을 설치할 수 있었고, 
   2개의 가상 코어와 4GB의 램으로도 원활한 성능을 기대할 수 있음. 
   
   다만, VM의 크기가 50GB 미만으로 줄이면 macOS 업데이트가 불가능해지므로, 
   최소 60GB의 VM 크기를 권장함. 
   
   이는 APFS의 스PARSE 파일 기능을 사용하면 실제 디스크 용량을 줄일 수 있으니, 
   이를 考慮해서 macOS VM의 크기를 설정하는 것이 좋을듯. 
   
   결론은, 맥북 네오와 같은 저전력 장치에서도 macOS VM을 설치할 수 있고, 
   적절한 사양의 설정으로 원활한 성능을 기대할 수 있단 것이고, 
   이는 macOS를 사용하는 사람들이 더 많은 선택지를 가진다는 것을 의미함
