### zed편집기 v1.0 떳다!!!!!!!!! 

zed편집기가 Finally v1.0 나왔다!ㅋㅋㅋㅋㅋㅋㅋㅋㅋ원래 에디터라고 하면 atom이랑 비슷하게 생긴것들만 생각했는데 이건 완전 다름ㅇㅇ

zed편집기는 원래 atom을 만들었던 팀에서 만들었는데 atom은 크로미움 기반으로 만든다고 하니까 Electron 프레임워크도 같이 개발하게됨ㅠㅠ근데 Electron 기반으로 만들어서 VS Code 같은거도 만들었는데 이게 요즘에는 AI 코드 편집기로 변신한다고함ㅋㅋㅋㅋ

zed편집기 팀은 이거를 보고 '웹 기술이 편하긴 하지만 한계가 있어서 더 좋은 에디터를 만들려면 새로운 접근방식이 필요하다'고 생각했고, 그냥 웹페이지처럼 만들지 말고 비디오 게임처럼 만들었다고ㅋㅋㅋㅋ 그래서 UI 프레임워크를 직접 만들었는데 GPUI라고 하니까 Rust로 만들었대ㅋㅋㅋㅋ

결과는? zed편집기는 개발자들한테 엄청 인기임ㅋㅋㅋㅋ 지금 hundreds of thousands의 개발자들이 zed편집기를 사용해서 소프트웨어를 만든다고함ㅋㅋㅋㅋ

그리고 zed v1.0는 modern editor가 지원해야 하는 기능들 다 지원하는판이라서, languages와 ecosystems 지원, Git integration, SSH remoting, Debugger, rainbow brackets 등등 다 있음ㅋㅋㅋㅋ

그리고 zed편집기는 AI-native editor라서 여러 개의 에이전트를 동시에 돌아가게 할 수 있고, 에이전트가.predictions해서 개발자들이 다음에 어떤 코드를 쓸지 알려준다는건데 이거 개신기하다ㅋㅋ키보드 칠때마다 에이전트가 예측해주니까 코드짜는속도가 엄청 빠르다고함😂

zed for Business도 같이 출시했다고해서, 중앙결제, 역할기반 접근제어, 팀관리等 다양한 기능들이 있다함😊

zed v1.0라고해서 끝난것 같지 않고, zed팀은 '이제开发자가 더 쉽게 zed를 사용할 수 있게되었다'는 의미로 1.0을 اعلام했다고함😂

zed팀의 비전은 최상급의 개발 환경을 만들어서 개발자들이 인간과 AI에이전트가 협업하여 더 멋진 소프트웨어를 개발할 수 있게 도와주는 것이라고함😇

그리고 DeltaDB라는 싱크로니제이션 엔진도 개발중이라는데, CRDTs를 기반으로 만들었다고해서 karakter-level의粒度로 코드베이스의 변화를 추적할 수 있다고함😲
