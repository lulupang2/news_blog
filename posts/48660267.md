# 노드js 툴킷 '넙' 소개
   넙은 노드js의 기능을 대체하는 것이 아니라, 노드js를 보완해주는 툴킷임. 
   바운(Bun) 처럼 노드js의 생태계를 강화하는 도구를 의미함. 
   ## 주요 기능
   *   노드js 파일 실행 (js, ts, mjs, cjs, mts, cts, jsx, tsx 지원)
   *  스크립트 런너 (npm run, pnpm run 대체)
   *   패키지 러너 (npx, pnpm dlx 대체)
   *   패키지 매니저 (npm, pnpm 대체)
   *   노드 버전 매니저
   *   코어팩 스타일의 글로벌 심
   노드js의 실험적인 API를 기본으로 사용할 수 있고, 일부는 폴리필 형태로 제공됨.
   # 사용 예시
   *   노드js 파일 실행: `nub index.ts`
   *   스크립트 실행: `nub run build`
   *   패키지 실행: `nubx eslint . --fix`
   *   패키지 설치: `nub install`
   *   노드 버전 설치: `nub node install 26`
