# MATRIX 곱셈 최적화하기
스위프트로 MATRIX 곱셈을 빠르게 하는 방법에 대한 글임. MATRIX 곱셈은 언어 모델训練에서 중요한 연산임.

## 시작하기
Author는 MATRIX 곱셈을 최적화하기 위해 시작했고, 그 결과를 공유함.

### Swift vs C
Author는 C言語 버전과 Swift 버전의 MATRIX 곱셈 연산 속도를 비교하고 있음.
Swift 버전은 C言語 버전보다 느려서 Author는 최적화하기로 결정함.

### 최적화 과정
Author는 Swift 버전을 최적화하기 위해 여러 가지 방법을 시도하고 있음.
가장 큰 성과를 낸 방법은 MutableSpan을 사용하는 것임.
MutableSpan은 Swift 6.2에서 도입된 기능으로 Array의 성능을 개선하는 데 도움이 됨.

### 결과
Author의 최적화 결과 MATRIX 곱셈 연산의 속도가 2.8GFLOP/s에서 1TFLOP/s로 증가함.

이 결과는 MATRIX 곱셈 연산이 언어 모델训練에서 중요한 연산이라는的事實과 Swift의 퍼포먼스를 개선하는 데 도움이 됨.
