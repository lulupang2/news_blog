   # 애플 학습 플랫폼 Canvas 뚫렸음 ㄷㄷ
   Canvas라는 애플 학습 플랫폼이 다운됐는데, ShinyHunters라는 해킹 집団이 떡밥을 던지면서 학습 플랫폼 Canvas 다운됐다고 함. 
   학생들 정보 털렸는데, 학생 이름, 이메일 주소, 아이디 번호, 메시지 등등 다 털렸다고 하는건데 5월 12일까지 협상 안하면 다 공개한다는건데 ㄹㅇ 개막장
   ShinyHunters라는 해킹 집단은 예전에도 Ticketmaster, AT&T, Rockstar Games, ADT, Vercel 등등 회사들의 정보를 털어먹은 전과가 있다고 하는데, 
   이번에는 Canvas 정보까지 털어서 학습 플랫폼에 문제가 생긴 모양임. 
   Canvas는 9천 개 학교의 정보가 털렸고, 2억7천5백만명의 학생, 교사, 직원 등의 정보가 털렸다고 하는데 ㄹㅇ 개 빡치네. 
   ShinyHunters는 데이터 유출 사이트에 학교 이름이나 정보를 올리고 협상을 제안했는데,Canvas 측은 아직 아무 대응도 안했음. 
   Canvas 측은 '잠시만요'라고만 하고 있음
