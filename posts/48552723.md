> **🔗 원문 보러가기:** [Trinket.io shutting down, so we saved it and hosted it a trinket.strivemath.org](https://trinket.strivemath.org/)


트린켓이오가 폐쇄 위기에 놓이자 스트라이브매스가 호스팅해 무료로 영원히 제공 중임. [트린켓](https://trinket.strivemath.org/)은 브라우저 기반 코딩 플랫폼으로 파이썬, HTML, 자바 등을 작성하고 실행할 수 있음. 
코드를 작성하고 실행 버튼 누르기만 해도 바로 결과가 나오고 로그인이나 다운로드도 필요없음. 
스트라이브매스는 이 서비스를 무료로 영원히 제공하기로 결정했고, 기존 트린켓이오 사용자들은 데이터를 가져와 사용할 수 있음. 
이 서비스는 교육자들에게도 유용해 Interactive 코딩 수업을 만들 수 있고 학생들의 진행상황을 추적할 수 있 음. 
스트라이브매스는 트린켓 커뮤니티를 위해 최선을 다할 것이니 많은 이용해 주시기를 바람. 
트린켓이오 사용자들은 데이터를 가져와 사용할 수 있고, 신규 사용자들은 간단한 회원가입만으로免费로 사용 가능함. 
개빡치는 점은 이제 트린켓 사용하기 위해 돈 내거나 기다리지 않아도 되다는 거임. 
스트라이브매스에서 호스팅하는 트린켓은 다양한 언어를 지원하며, 브라우저 기반으로 작동하므로どこ에서든지 사용 가능함. 
트린켓은 교육과 코딩 학습에 있어 매우 유용한 도구임. 
스트라이브매스에서 제공하는 트린켓은 무료로 영원히 사용 가능하니 많은 이용해 주시기를 바람.
