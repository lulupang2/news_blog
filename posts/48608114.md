# IBM Z와 리눅스원, 최신 오픈소스 소프트웨어 지원 업데이트
     IBM Z와 리눅스원은 s390x 하드웨어 아키텍처를 사용해서 다양한 리눅스 배포판을 구동할 수 있음. 
     예를 들면, SUSE 리눅스 엔터프라이즈 서버(SLES), 레드헷 엔터프라이즈 리눅스(RHEL), 우분투 등이 있음. 
     IBM의 일 팀은 이런 오픈소스 프로젝트와 협력해서 일부 어플리케이션이 예상대로 컴파일되고 실행되는지 확인하는 일을 맡음. 
     이 작업은 지속적으로 진행되고 있어서, 소프트웨어마다 새로운 릴리즈가 나올 때마다 검증해야 함 ㅋㅋㅋ.
     
     # 최신 업데이트
     2026년 5월에 IBM팀은 다음과 같은 소프트웨어들의 최신 버전을 검토함:
     * 아파치 카산드라
     * 아파치 HBASE
     * 아파치 HTTP 서버
     * 아파치 이그나이트
     * 비츠
     * cAdvisor
     * 칼리코
     * 닥스지니
     * 팔코
     * 그라파나
     * HAPROXY
     * HTOP
     * MongoDB 드라이버 (자바, PHP, 루비)
     * 네티-TC네이티브
     * 네오4지
     * OPA
     * 오픈레스트리
     * PHP
     * PM2
     * 포스트그레스QL
     * 파이썬
     * 래빗MQ
     * 루비
     * 테라폼
     
     이렇게 검증된 소프트웨어의 전체 목록은 [이 링크](https://www.ibm.com/community/z/open-source-software/)에서 확인할 수 있음.
     
     # 새로운 소프트웨어 지원
     이번달에 s390x 지원을 추가한 프로젝트는 모두 10개가 넘음. 
     * Kuadrant dns-operator CI와 v0.17.0 릴리즈 바이너리
     * go-simdjson CI 업데이트
     * aba CI 업데이트
     * poof 0.6.0 릴리즈에 s390x 지원 추가
     * go-bin의 첫 s390x 유희들 릴리즈 (버전 1.26.3)
     * nginx-opentracing에 s390x CI 추가
     * OpenSCM 클라이언트에 s390x 지원 추가 (버전 0.2.8)
     * 테라폼 1.15.4 릴리즈에 s390x 도커허브 이미지와 바이너리 다운로드 추가
     
     # 개발자们에게
     IBM Z와 리눅스원에서 구동되는 어플리케이션을 개발하고 있다면, 이 리포트에서 소개하는 소프트웨어들을 참고하면 유용할거임. 
     개발 중인 프로젝트를IBM 리눅스원 커뮤니티 클라우드에서 테스트해 볼 수 있음. 
     이 클라우드에서 120일 동안 무료로 가상머신을 사용할 수 있어서, 어플리케이션이 IBM Z와 리눅스원에서 잘 구동되는지 확인할 수 있음.
