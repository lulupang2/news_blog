# 파이인프라 3.8.0 버전 나왔어!
   파이인프라 개발자들이 열심히 일해서 새로운 버전을 내놓았음. 
   이전 버전에 있던 문제점들을 수십 개나 고쳐서 더 안정적이고 쓸만하게 만든 듯.
   ### 기능 추가
   * api.command에서 fmt 문자열 사이에 공백이 생기는 버그를 고침
   * api에서 클릭에서 분리하여 출력 함수를 플러그인 형식으로 만들었어
   * 사용자 입력을 명령어로 쓸 때 injection 취약점을 막아주었음
   *.selinux 파일 컨텍스트가 없으면 에러를 내지 않고 넘어가도록했어
   * 시스템 팩트들을 더 많이 수집할 수 있게 개선했음 
   * 깃헙 연동 기능이 더 좋아졌고, 도커도 그렇고各种 기능 개선했어 ㅋㅋ
   * 보안도 많이 강화했음. 예를 들어 SSH 아이디에이전트 설정을 지원하기도하고, 
   명령어를 만들 때 신뢰할 수 없는 값들을 다 따옴표로 감싸주기도 함.
   ### 커널러들이 열심히 일해서 좋긴한데...
   커미터들 이름을 보니까 대부분이 외국인 같네. 한국 개발자들도 참여하면 좋겠다.
   ### 마무리
   파이인프라 3.8.0 버전은 이전 버전에 비해 많이 개선되었고, 앞으로도 더 나아질 것 같아. 
   개발자 여러분들도 많이 사용하시고, 기여해 주세요! ㅋㅋ
