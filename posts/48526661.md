자바스크립트의 역사부터 2035년까지의 미래까지이야기하는 거임. 
  _pycon 2014에서 가져온 개쩌는 강의임. 
   개빡치는 점은 자바스크립트가 1995년부터 시작된 거 다. 
   그리고 2035년까지 어떻게 바뀌게 될지 이야기해 줌. 
   자바스크립트의 장단점을 이야기하면서도 ultimate impact가 개적으로 긍정적인 면이 많다는 거임. 
   이 강의는 자바스크립트를 까는 건 아니고, 오히려 자바스크립트의 장단점을 다각도로 분석해 줌. 
   Destroy All Software라는 사이트에서 더 심한 강의도 볼 수 있음. 
   Execute Program이라는 곳에서도 TypeScript, Modern JavaScript, SQL, 정규식 등Interactive하게 공부할 수 있음. 
   각 코스는 브라우저에서 바로 실행할 수 있는 Hundreds of interactive code examples로 구성되어 있음. 
   이 강의를 보면서 자바스크립트의 미래를 한번 상상해 볼만 함. 
   자바스크립트의 역사와 미래를 알면 앞으로 더 좋은 개발자가 될 수 있을 것 같음.
