### 엔터프라이즈급 NAS 시대 개판남! ZFS 기반의 ENAS 등장

엔터프라이즈급 저장공간이야 항상 비싼 라이센스 비용, 专用 하드웨어, 복잡한 관리 때문에 기업들이 고민임 ㅋㅋㅋ. 그런데 ENAS 등장해서 모든 걸 바꿨음. 
private하고 lokal한 저장공간 플랫폼으로 설계되었고, 성능, 확장성, 심플함을 모두 제공하기 때문에 기업들이 엔터프라이즈급 솔루션에서 기대하는 모든 걸 제공함. 엔터프라이즈급 성능을 위해 설계된 모든 게ใหม로 바뀐 ENAS는 가장 мощ한 하드웨어와 현대적인 저장 기술을 결합해서 엔터프라이즈 솔루션에서 기대하는 성능과 신뢰성을 제공함. 
ENAS에는 검증된 ZFS 아키텍처, 선택 가능한 M.2 NVMe L2ARC 캐싱, 그리고 강력한 ARM Neoverse N2 플랫폼이 결합되어 있기 때문에 highest performance 저장 솔루션이라고 할 수 있음. 
- 8개의 Arm Neoverse N2 코어로要求되는 워크로드 처리 가능
- 64GB ECC 메모리로 데이터 정합성 향상
- 듀얼 NVMe 캐시로 성능 가속
- 네이티브 ZFS로 저장 보안 강화
- 현대적인 비즈니스 워크로드를 위해 설계된 엔터프라이즈급 성능

### 대규모 확장성과 완전한 유연성

ENAS는 기업이 성장하면 함께 성장할 수 있도록 설계됨. 확장 가능한 저장 능력과 하드웨어 제약 없이 드라이브 모델에 대한 제한이 없기 때문에 기업은 비용 없이 저장 용량을 확장할 수 있음.
- 16개의 드라이브 베이를 통해 대용량 배포 가능
- 원하는 용량까지 확장 가능
- 오픈 드라이브 호환성을 통해 최대 유연성 달성
- 듀얼 25 Gigabit SFP28 포트와 중복된 전원 공급으로 인해 내구성이 강화됨. 
.vendor lock-in 없이 저장 공간을 확장할 수 있음.

### 유니파이(UniFi)로 라이센스 걱정 없는 관리

하드웨어는 이야기가 아니야. ENAS는 유니파이 플랫폼과 직접 통합되어 파일, 백업, 공유 된 조직 데이터를 위한 중앙 집중식 소스라고 할 수 있음. 그리고 모두를위한 소프트웨어 라이센스 비용 없이 가능함.
- 라이센스 비용이나 기능 잠금 없이 사용 가능
- 중앙 집중식 파일 및 백업 관리
- 팀 전체에 걸쳐 데이터 가시성 통일
- 유니파이를 통해 단순화된 관리
- 라이센스 비용 걱정 없이 강력한 저장 관리 가능

### 모든 직원 위한 보안 접근

현대적인 조직은 비즈니스 데이터에 대한 보안이고 무결한 접근이 필요함. ENAS는 기존의 아이디 제공 업체와 통합되어 사용자 관리를 단순화하는 역할 기반 접근 권한을 제공함.
- 기존 아이디 제공 업체와 통합
- 폴더 권한 기반 역할
- 유니파이 엔드포인트를 통해 네이티브 보안 접근
- 모든 기기에서 일관된 접근 제공
- 아이디 기반 접근 제어로 협업 보안 시작

### 중앙 집중식 백업 오케스트레이션

유니파이 패브릭과 사이트 매니저를 통해서는 백업 관리가 쉽게 가능해 짐. 중앙 집중식 소스로부터 모든 사이트를 관리하면서 중요한 데이터를 보호할 수 잇음. 
- 모든 사이트에 걸친 백업 관리 통일 
- 단일 플랫폼에서 중앙 집중식 오케스트레이션
- 통합 스냅샷 기능
- 재해 복구 준비 향상
- 모든 위치에서 데이터 보호를 위한 단일 플랫폼

### 최신 가상화 위한 공유 저장공간

ENAS는 전통적인 파일 저장을 넘어서 유니파이 드라이브에 네이티브 iSCSI 지원을 통해 가상화 환경과 고가용성 인프라를 위한 공유 블록 저장공간을 제공함.
- 네이티브 iSCSI 공유 블록 저장공간
- 여러 컴퓨팅 서버를 위한 중앙 집중식 저장공간
- 라이브 마이그레이션 워크플로우 지원
- Proxmox, VMware, Hyper-V 클러스터와 호환
- 가상화와 고가용성을 위한 공유 블록 저장공간

ENAS는 엔터프라이즈급 성능, 대규모 확장성, 중앙화된 관리, 보안 접근 제어, 멀티사이트 백업 오케스트레이션, 가상화 준비된 저장공간을 모두 하나의 _license-free_ 플랫폼으로 제공함. 专用 제한과 반복적인 소프트웨어 비용을 없애서 더 간단하고 확장성이 뛰어나고 비용 효율적인 엔터프라이즈급 저장 솔루션을 제공함.
