### DOM-DOCX: HTML을 네이티브 워드 파일로 변환하는 도구
 DOM-DOCX 는 HTML 을 네이티브 워드 파일 (.docx) 로 변환해주는 도구야. 라이브 데모도 지원하고, 브라우저나 노드에서 사용할 수 있어. 
 ### 지원 기능
 * HTML 과 CSS 지원
 * 워드 파일 생성 지원 (.docx)
 * 라이브 데모 지원
 * 브라우저와 노드에서 사용 가능
 * 지원되는 기능:
   + 제목, 단락, 리스트, 표, 이미지 등
   + 워드 파일 생성 옵션
   + 라이브 데모
   + 브라우저와 노드에서 사용 가능
 ### 사용 방법
#### 브라우저에서 사용하기
```javascript
import { convertHtmlToDocx } from "dom-docx/browser";

const html = `<h1>제목</h1><p>내용</p>`;
const blob = await convertHtmlToDocx(html);
const a = document.createElement("a");
a.href = URL.createObjectURL(blob);
a.download = "output.docx";
a.click();
```
#### 노드에서 사용하기
```javascript
import { convertHtmlToDocx } from "dom-docx";
const html = `<h1>제목</h1><p>내용</p>`;
const docx = await convertHtmlToDocx(html);
// 워드 파일 저장
```
## 결론
 DOM-DOCX 는 HTML 을 워드 파일로 쉽게 변환할 수 있는 도구야. 지원되는 기능이 많고, 브라우저와 노드에서 사용할 수 있어 개발자에게 유용한 도구가 될 거야.
