# 소개
     GLM-5.2라는 최신 대형 언어 모델을 느린 컴퓨터에서 구동시키는 프로젝트임. 
     모델 용량은 약 370GB인데, 램은 25GB 밖에 안 되는 컴퓨터에서 돌아감.
     # 모델 특징
     * 모델의 전체 크기는 약 744B 파라미터임
     * 모델은 Mixure of Experts(MoE) 방식으로 구성되어 있어서, 한번에 활성화되는 파라미터는 40B 정도임
     * 모델의 구조는 75개의 MoE 레이어와 256개의 专가(expert)로 구성됨
     *専가는 디스크에 저장되어 있고, 필요할 때 스트리밍되는 방식임
     # 성능
     * Cold Start 시 성능은 매우 낮은데, 약 0.05 tok/s 임
     * 하지만 캐시 히트율이 올라가면 성능이 많이 개선됨
     * 최적의 환경에서는 약 2-4 tok/s 까지 성능이 올라감
     # 사용 방법
     * Linux에서 사용해야 하며, gcc 컴파일러와 OpenMP, AVX2 확장 명령어를 지원하는 CPU가 필요함
     * NVMe 디스크에 모델을 저장해야 하며, 램은 16GB 이상이 필요함
     # 벤치마크 결과
     * Intel Core Ultra 7 270K Plus에서 약 0.07 tok/s 의 성능을 보임
     * Apple M5 Max에서 약 1.06 tok/s 의 성능을 보임
     # 결론
     GLM-5.2를慢 컴퓨터에서 구동시키는 것은 충분히 가능함. 하지만 캐시 히트율을 높이고, 적절한 하드웨어를 사용하는 것이 성능을 올리는 데重要함
