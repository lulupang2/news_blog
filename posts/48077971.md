# 프리BSD 보안 취약점: 루트권한으로 탈출 가능 ㅋㅋㅋ
프리BSD에서 코드 실행 취약점이 발견됐다고 함. 그냥 루트권한 가져올 수 있다고는 하지만 ㅋㅋㅋ

## 취약점 정보
프리BSD 프로젝트는 최신 보안 취약점을 公開했음. execve(2) 시스템 콜에서 발생하는 문제라고 함. 그냥 운영체제에서 코드 실행할 때 생기는 문제인 것 같음.

### 발생 원인
운영체제의 커널에서 버퍼 오버플로우가 발생해서 공격자들이 데이터를 조작할 수 있다고 함. 이게 가능하면.superuser 권한까지 가져올 수 있다고는 하지만.. ㄹㅇ 개빡치는 점

## 대책
프리BSD 프로젝트는 패치 파일을 공개했음. 그냥 다운로드 받아서 업그레이드하면 될 것 같음. 그리고 시스템 재부팅하면 문제 해결됨. 재부팅 안 하면 문제 解決 안 되는 건 아님.ㅋㅋㅋ

### 패치 방법
1. 패치 파일 다운로드: https://security.FreeBSD.org/patches/SA-26:13/exec.patch
2. 패치 파일 적용: `patch < /path/to/patch`
3. 커널 재 컴파일: `cd /usr/src && make kernel`

그리고 시스템 재부팅하면 문제 解決됨. 문제 해결 안 된다고 우는 사람들은 그냥システム 재부팅 안 한 거임.ㅋㅋㅋ
