# Jelly UI 소개
Jelly UI는 네이티브 HTML 폼 컨트롤에 소프트바디 물리학을 적용한 웹 컴포넌트 라이브러리임. 0개의 의존성, 40개의 커스텀 엘리먼트, 1개의 스크립트 태그로 구성되어 있고 WCAG AA 컬러 토큰, 다크 모드, 오른쪽에서 왼쪽 언어 지원 등을 기본으로 포함하고 있음. 

# 주요 특징
* 0개의 의존성으로 프로젝트를 더욱 간단하고軽量으로 유지할 수 있음
* 40개의 커스텀 엘리먼트로 사용자_experience를 다양하게 커스텀 할 수 있음
* WCAG AA 컬러 토큰으로 색상_contrast를 자동으로 관리할 수 있음
* 다크 모드와 오른쪽에서 왼쪽 언어 지원으로 사용자 편의성 개선

# 사용 방법
Jelly UI를 사용하기 위해서는 단순히 스크립트 태그 하나를 추가하면 됨. 
```html
<script type="module" src="https://jelly-ui.com/package.js"></script>
<jelly-theme mode="auto">
  <jelly-button variant="mint">Publish</jelly-button>
</jelly-theme>
```
