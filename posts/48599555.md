# 프록시모스 VE와 마이크로 가상 머신

프록시모스 VE는 가상화 플랫폼이고 마이크로 가상 머신은 가벼운 가상 머신인데 둘을 조합해서 쓸 수 있음. 개빡치는 점은 쉽게 설정할 수 없다는 거임. 하지만 방법은 있음 ㅋㅋㅋ

## 쉽게 설정하는 방법

먼저 프록시모스 VE를 설치하고 마이크로 가상 머신을 다운받아야 함. 다운받은 마이크로 가상 머신을 프록시모스 VE에 올리고 설정 파일을 수정하면 됨. 설정 파일 수정이 가장 핵심인데 잘못 수정하면 안됨. 수정할 때 주의해야 할 점은 여러 가지 있지만 일단 올리면 됨. 그리고 몇 가지 설정을 변경하면 마이크로 가상 머신이 실행됨. ㄹㅇ 쉽네

## 마이크로 가상 머신의 장점

마이크로 가상 머신은 가벼워서 빠르게 실행되고 리소스도 적게 씀. 그러니까 가볍고 빠르네. 그리고 보안이 좋음. 보안이 좋은 이유는 가상화 기술을 사용해서 격리되었기에 마이크로 가상 머신끼리 통신하지도 못하고 호스트 시스템도 못 건드림. 그러니까 보안 걱정 없이 사용할 수 있음. 이래서 마이크로 가상 머신을 사용하는 거임. 그리고 프록시모스 VE에서 마이크로 가상 머신을 사용하면 여러 가지 이점이 있음. 예를 들어 여러 가지 운영 체제를 실행해 볼 수 있고 테스트 환경을 쉽게 구성할 수 있음. 이래서 프록시모스 VE와 마이크로 가상 머신을 조합해서 사용하는 거임. ㄹㅇ 좋네
