# 애플이 새로 만든 시리 AI는 진심으로 도와줄거임ㅇㅇ
     시리는애플 인텔리전스와 함께 돌아가고, 개인 중심의 인공지능이라서 사용자에게 진짜 도움을 주는 게 목표임.
    _apps_와 결합되어 있고, 사용자 컨텍스트를 고려하며, 모든 단계에서 민감한 정보를 보호할 수 있음.
     ### 주요 기능:
     * 시리 AI와 자연스럽게 대화해서 필요한 걸 찾고, 더 많은 일을 처리할 수 있게 해줌.
     * 시리 AI는 기존 인공지능보다 더 똑똑하고, 사용자가 원하는 정보를 찾아올 수 있음.
     * **시리 앱**: 대화 기록을 한 곳에 모아두고, 사용자가 원하는 대로 맞춤 설정할 수 있음.
     * **시리와 카플레이**: 운전 중에 시리를 이용해서 길 안내나 음악 등을 제어할 수 있음.
     * **시리와 카메라**: 시리와 함께 카메라로 물체를 인식하고, 정보를 얻을 수 있음.
     ### 시리 AI의 기타功能:
     * **사진 편집**: Spatial Reframing, Extend, Clean Up 등을 이용해서 사진 편집할 수 있음.
     * **이미지 플레이그라운드**: 고퀄리티 이미지 생성, 이미지 수정 및 변형 등 가능함.
     * **라이브 번역**: 메시지, 페이스타임 등에서 자동 번역 가능.
     * **사파리**: 탭 자동 그룹화, 변경 사항 경고 등으로 사용자 경험이 개선됨.
     * **사용자 인증**: 비밀번호 자동 변경, 보안 강화 등이 지원됨.
     # 애플 인텔리전스는 사용자의 개인 정보를 보호하며, 온디바이스 프로세싱을 통해 민감한 정보를 안전하게 유지시킴.
     # 결론: 새로 나온 애플 시리 AI는 사용자 친화적이고, 개인 중심의 인공지능이면서도, 사용자 개인 정보를 철저하게 보호하는 기능을 제공할거임. 진짜 사용자에게 도움을 주는 인공지능이 될 예정임ㅋㅋ
