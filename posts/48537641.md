### 오ープ런처 퓨전.api
   오ープ런처: 퓨전은(prompt를 받았을 때) 하나의 모델로는 부족해서 여러 개의 모델을 동시에 사용해서 분석하는 기술 임.
   예전에는 모델 하나만 썼었지만 이제는 여러 개의 모델을 동시에 사용해서 더 좋은 결과를 낼 수 있음.
   이 기술은 퓨전(compare playground fusion)이라고 부름.
   퓨전은 여러 개의 모델을 사용해서 분석을 해주는데, 이 모델들 중에 판단하는 모델도 있음.
   이 판단하는 모델이 여러 개의 모델이 분석한 결과를 종합해서 최종적인 분석 결과를 내줌.
   그리고 이 최종적인 분석 결과를 사용자가 이해하기 쉽도록 작성해 줌.
   퓨전을 사용하면 모델 하나만 사용하는 것보다 더 좋은 결과를 얻을 수 있다고 함.
   하지만 모델 하나만 사용하는 것보다 더 비싸다니까 주의해야 할 점임.
   예를 들어, 퓨전을 사용하면 여러 개의 모델을 사용해서 분석하는 데 비용이 더 많이 드는 거야.
   그래서 모델 하나만 사용하는 것보다 더 비싸지만, 더 좋은 결과를 얻을 수 있으니까 값어치 있는 기술 임.
