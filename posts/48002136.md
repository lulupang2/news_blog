# 딥클라우드 소개
딥클라우드는 클라우드 코드의 에이전트 루프를 더 싸고 좋은 디씨클라우드에서 사용할 수 있게 해주는 툴임. 클라우드 코드는 진짜 좋은 자동 코딩 에이전트지만 200美元/월에 사용량 제한이 있어서 아쉬웠음. 
## 딥클라우드 사용법
1. 디씨클라우드 API 키를 받아서 환경변수로 설정해 줘야 함. 
2. 디씨클라우드의 API 키는 0.87美元/M 출력 토큰에 불과해서 너무 좋음 ㅋㅋㅋ 
3. 딥클라우드는 클라우드 코드의 에이전트 루프를 더 싸게 사용할 수 있게 해주는데 클라우드 코드의 UX와 기능은 그대로임. 
4. 딥클라우드는 다양한 백엔드를 지원해서 오픈라우터나 파이어워크 에이를 사용할 수도 있음. 
## 기능
* 파일 읽기/쓰기/편집
* 배시/파워쉘 실행
*글로브와 그랩 검색
* 멀티스텝 자율 도구 루프
* 서브에이전트 생성
* 깃 연산
* 프로젝트 초기화(/init)
* 싱킹 모드(기본적으로 활성화됨)
 ## 비용 비교
요약하자면 디씨클라우드는 클라우드 코드의 에이전트 루프를 17배 더 싸게 사용할 수 있게 해주는 툴임. 10일간의 사용량으로는 200美元/월인 클라우드 코드보다 90% 더 싸고 25일간의 사용량으로는 75% 더 쌈. 
## 지원 언어
파워쉘, 쉘, 자바스크립트가 지원됨.
