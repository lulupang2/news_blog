# COBOL로 만든 1인칭 슈터게임 FPS.cob

딱 보니까 게임 개발이 요즘 너무 쉬운가 보면 COBOL로 게임을 만들어 보는 건 어때? ㅋㅋㅋ
이 게임은 Wolf3D 스타일과 둠 스타일 두 가지로 플레이가 가능해
- map/level1.map:-original wolf3d style 
- map/doom_sectors.map:둠 스타일임

**설치 방법**
 cobc ffplay bash 가 필요해
 repo_root에서 bash build.sh 를 실행시키면 게임을 시작할 수 있어 
 특정 맵을 불러오려면 bash build.sh ./map/level1.map 이나 bash build.sh ./map/doom_sectors.map 를 쓰면 돼

**컨트롤 방법**
  - W/S:앞뒤 이동
  - A/D:좌우 회전
  - 스페이스:총 발사
  - Q:게임 종료

**자원**
 텍스처, 스프라이트는 res 폴더에 있고 맵은 map 폴더에 있어

**정보**
설명이나 웹사이트, 주제를 제공하지 않았네 ㅋㅋ 
Apache-2.0 라이센스를 갖고 있어

**오류**
현재 페이지를 불러오는 중에 오류가 발생했어 
새로고침을 해봐 
**기여자**
 기여자는 없네... ㅋㅋ 
**언어**
 COBOL 99.7%
 Shell 0.3%
