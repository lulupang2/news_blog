### 루비 배들러 쿨다운 지원 기능
루비 배들러에 쿨다운 지원 기능이 도입되어 새로 등록된 гем에 대하여 일정 기간 동안 설치를 제한하는 기능을 제공하게 됨.
이 기능은 신규 업데이트가 있을 때마다 바로 설치하지 말고, 몇 일 정도 검토 기간을 갖는 것을 목표로 함.
이 기능은 옵션으로 제공되고,mandatory 2요인 인증과 신뢰할 수 있는 배포 외에 하나의 보안 기능으로 작용하는 거야.
쿨다운 기능은 루비겜즈 오ργ에 등록된 гем의 created_at 타임스탬프를 읽어서, 얼마나 된 것인지 확인하고, 설정된 일数만큼 대기하게 됨.
만약 등록되지 않은 소스거나, 구 버전의 젬 서버거나, 비공개 레지스트리이면 쿨다운 기능을 우회할 수 있어.
쿨다운 기능이 적용된 버전의 루비 배들러를 사용하려면, `$ gem update --system` 명령어를 통해 최신 버전의 루비 배들러를 설치하고, `$ bundle update --bundler=4.0.13` 명령어로 버전을 고정하면 됨.
그 다음에는 Gemfile에서 쿨다운 옵션을 설정하면 되는데, 보통은 7일 정도를 기본값으로 사용하는 듯.
쿨다운 설정은 `$ bundle config set cooldown 7` 명령어로 toàn局에 설정할 수도 있고, 프로젝트별로 설정할 수도 있어.
만약 일회성으로 쿨다운을 설정하고 싶다면, `$ bundle install --cooldown 7`와 같은 명령어를 사용하면 된다기록한다.
### 쿨다운 기능 사용법
쿨다운 기능은 매우 간단하게 사용가능하고, 프로젝트마다 설정이 가능하여 유연하게 사용할 수 있어.
쿨다운 기능을 사용하면, 루비 배들러를 사용하는 프로젝트의 보안을 개선할 수 있으며, 악의적인 업데이트를 사전에 방지할 수 있어.
쿨다운 기능 외에도 다양한 보안 기능을 루비 배들러에서 제공하고 있고, 이 기능을 사용하면 더욱 안정적인 개발 환경을 구축할 수 있기 때문에, 루비 개발자라면 반드시 사용해봐야할 기능중 하나야
