# 개요 
고랭 1.24에서 http/2 클리어텍스트를 사용해 서버를 설정하는 방법을 설명할게요. 

# 기존 설정 방법 
전에 설정하려면 golang.org/x/net/http 패키지를 사용해서 설정해야 했는데, 너무 복잡했던거 같음 ㅋㅋㅋㅋ 
```go 
import (
	"net/http"
	"golang.org/x/net/http2"
	"golang.org/x/net/http2/h2c"
)

handler := ... // 기존 핸들러
h2s := &http2.Server{}
handler = h2c.NewHandler(handler, h2s)
srv := &http.Server{
	Addr:       fmt.Sprintf("%s:%d", "", 9888),
	Handler:    handler,
	ReadHeaderTimeout: 5 * time.Second,
	ReadTimeout:      10 * time.Second,
	WriteTimeout:     35 * time.Second,
	IdleTimeout:      620 * time.Second,
}
err = http2.ConfigureServer(srv, h2s)
if err != nil {
	// 오류 처리
}
```

# 새로운 설정 방법 
고랭 1.24부터는 더 간단해졌어요. 설정이 훨씬 읽기 쉬움 ㅋㅋ 
```go 
handler := ... // 기존 핸들러
srv := &http.Server{
	Addr:       fmt.Sprintf("%s:%d", "", 9888),
	Handler:    handler,
	ReadHeaderTimeout: 5 * time.Second,
	ReadTimeout:      10 * time.Second,
	WriteTimeout:     35 * time.Second,
	IdleTimeout:      620 * time.Second,
}
srv.Protocols = new(http.Protocols)
srv.Protocols.SetHTTP1(true)
srv.Protocols.SetUnencryptedHTTP2(true)
```

# 테스트 
로컬에서 테스트하기전에는 항상 검사해야함 ㅋㅋ
```bash 
curl -i --http2-prior-knowledge http://localhost:9888
```
