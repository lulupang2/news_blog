# 포스 언어란?
포스 언어는 Forth라는 언어에서 영감을 받아 웹 사이트를 작성할 때 사용할 수 있는 언어야. 이 언어는 스택 기반 언어라서 좀 더 효율적이고 재미있게 작성할 수 있음. 

# 사용 예시
예를 들어, HTML을 작성할 때 다음과 같이 작성할 수 있어.
```h1 ( s -- ) "<h1>" emit . "</h1>" emit ; "Hello, World!" h1```
이 언어는 스택 기반 언어라서 Wort 정의를 쉽게 추가할 수 있어. 

#Forge 언어의 특징
포스 언어의 특징은 서버 사이드 렌더링과 클라이언트 사이드 렌더링을 모두 지원해. 서버 사이드 렌더링은 크롤러와 WebMentions를 위해 사용되고, 클라이언트 사이드 렌더링은 SPA 경험을 위해 사용돼. 

이 언어는 좀 어색하지만 재미있게 작성할 수 있는 언어야. 포스 언어를 사용하면, 좀 더 효율적이고 재미있게 웹 사이트를 만들 수 있을 거로 보임.  ㄹㅇ
