### Chatto 오픈소스 떴다 ㅋㅋㅋㅋ
Chatto 오픈소스 떴다 ㅋㅋㅋㅋ 이거 진짜 큰 뉴스임 ㅋㅋㅋ Chatto라는 팀 채팅 앱을 거의 1년 동안 작업해 왔는데 finally 오픈소스로 공개가 됨 ㅋㅋ 
먼저 사용해 볼 수 있는 가장 빠른 방법은 Homebrew를 쓰는 거임 
```bash
brew install chattocorp/tap/chatto
chatto init
chatto run
```
자세한 사용 방법은 [Chatto 시작가이드](링크) 참고해줘.
Chatto는 진짜 잘 만든 팀 채팅 앱임. 다른 채팅 앱들과 다르게 정말 가볍고 반응성이 좋고, 오픈소스로 제공돼서 마음껏 수정해 쓸 수 있음 ㅋㅋ 그리고 무료로 쓸 수 있잖아? 이거 정말 혜자임 ㅋㅋㅋㅋ 

### Chatto의 특징
- **오픈소스**: 마음껏 수정하고 배포할 수 있음
- **자체 호스팅**: 직접 서버를 열어 사용할 수 있음
- **가볍고 반응성 좋음**: 채팅 앱으로는 가장 हलがい서 사용하기 쉽게 만들어짐
- **데이터 보호와 프라이버시**: 모든 개인 및 채팅 데이터를 사용자별로 암호화해서 저장하고, 사용자가 계정을 삭제하면 데이터가 즉시 삭제됨

그리고 [Chatto Cloud](링크)도 곧 공개될 예정임 ㅋㅋㅋㅋ Chatto Cloud는 Chatto 서버의 호스팅을 제공하는 서비스고, 유료 서비스지만 정말 혜자임 ㅋㅋㅋㅋ 
- **자동 스케일링**: 사용자가 많아져도 자동으로 서버를 늘려줌
- **夜间 백업**: 밤에 자동으로 데이터 백업을 해줌
- **Zero-downtime 업그레이드**: 서버를 업그레이드할 때 사용자에게 영향을 주지 않음

### Chatto의 미래
Chatto는 지금 버전 0.4 임 ㅋㅋ 아직 몇 가지 중요한 기능이欠如하지만, 이미 프로덕션에서 사용할 수 있을 정도로 안정성이 좋음 ㅋㅋㅋㅋ 
향후 업데이트에서는 안전한 기능과 클라이언트를 다듬어주는 것에 중점을 둘 예정임 ㅋㅋㅋㅋ 
6-12개월 내에 버전 1.0을 출시할 계획인데, 그 때까지 업데이트가 조금 있으면서도 최소한의 breaking change만 발생할 것임 ㅋㅋㅋㅋ 

### 연락처
만약 Chatto를 자체 호스팅해 쓴다면, 경험談이나 불러일으킬 수 있는 버그를 [Chatto HQ 커뮤니티](링크)에서 공유해줘 ㅋㅋㅋㅋ 
또한 Chatto를 회사나 오픈소스 프로젝트에서 사용하길 원한다면, 연락해줘 ㅋㅋㅋㅋ
