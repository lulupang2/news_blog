# 도구와 방법
손으로 데이터 시각화를 그리기 위해서는 
 여러 가지 도구와 방법이 필요해. 
일단紙, T스퀘어, 자, 드로잉 보드, 페인터 테이프, 연필, 펜, 지우개 등이 있어야 해. 
그리고 원을 그리고, 선을 그리고, 글씨를 쓸 수 있어야 하지. 
가장 중요한 건 간결하고, 정확하고, 예쁘게 그리는 거야. 
그리고 그 과정을 즐기면서, 데이터가 무슨 뜻인지 생각해 보는 거야.

# 데이터 시각화를 시작하다
먼저 격자를 그리면 편해. 
격자를 그리다 보니, 데이터가 어떤 모양인지 보인다니까. 
그리고 데이터를 하나씩 그리다 보면, 
어느새 데이터 시각화가 완성된 거야. 
그리고 완성된 걸 보니까, 
뭔가 뿌듯하고, 기분이 좋아지는 걸 느꼈어.

# 데이터 시각화의미
데이터 시각화는 숫자로 표시된 데이터를 그림으로 나타내는 거야. 
그림이니까, 더 쉽게 데이터의 의미를 이해할 수 있어. 
예를 들어, 어떤 회사에서 판매 실적을 수집해놨는데, 
그 데이터를 시각화하면, 
어떤 달에 판매가 가장 많았는 지, 
어떤 달에 가장 적었는 지 알 수 있어. 
이런 식으로, 데이터 시각화는 데이터를 더 쉽게 이해하게 해 주는 거야.

# 데이터 시각화의 예
예를 들어, 애플의 주가 변화를 시각화하면, 
주가가 시간의 경과에 따라 어떻게 변했는지 알 수 있어. 
또는, 어떤 제품의 판매 실적을 시각화하면, 
어떤 제품이 가장 많이 팔리는 지 알 수 있어. 
이런 식으로, 데이터 시각화는 데이터를 더 쉽게 이해하게 해 주는 거야.
