# Dirac 에이전트 개요
Dirac은 코드를 자동으로 작성하고 수정할 수 있는 오픈소스 에이전트임 ㅋㅋㅋ 이 에이전트는 gemini-3-flash-preview 터미널 벤치마크에서 1등을 했고, 코드 수정 비용을 64.8% giảm少해 줌 ㄹㅇ

## 주요 특징
* 해시 앵커 편집: Dirac은 코드 수정을 위해 해시 앵커를 사용해서 코드를 매우 정확하게 수정할 수 있음 ㅇㅇ
* AST 네이티브 精度: Dirac은 언어의 문법을 이해해서 코드 구조를 sangat genau하게 수정할 수 있음 
* 멀티 파일 배칭: Dirac은 여러 파일을同时 수정할 수 있음 
* 높은 대역폭 문脈: Dirac은 코드 수정을 위해 필요한 정보를 lean하게 유지해서 LLM이 Always 관련 정보만 사용할 수 있도록 해 줌 

## 설치 방법
_dirac 에이전트는 VS Code 확장으로 설치할 수 있음  그리고 npm로 CLI를 설치할 수도 있음 

## 시작 방법
_dirac auth로 인증을 하고, dirac "prompt"로 시작할 수 있음 

## 환경 변수
_dirac 에이전트는 환경 변수를 통해 API 키를 제공할 수 있음 

## 공통 명령어
_dirac "prompt": 대화형 작업을 시작할 수 있음 
_dirac -p "prompt": 작업 전략을 확인하고 작업을 시작할 수 있음 
_dirac -y "prompt": 작업을 자동으로 승인할 수 있음 

## 시작하기 
_dirac 사이드바를 여는 방법 
1. Dirac 사이드바를 열기 
2. AI 제공자를 선택하기 
3. 작업을 시작하기
