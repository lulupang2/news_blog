# 컨덴서 알람 시계
컨덴서 알람 시계는 ESP32 칩을 이용해서 만든 자작 프로젝트임. 이 프로젝트는 ElectroBOOM의 컨덴서 알람 비디오에서 영감을 얻었다고 함. 
### 특징
- ESP32 칩을 이용함
- 웹서버로 설정을 변경할 수 있음
- NTP를 이용해서 시간을 자동으로 불러옴
- 128x64 SSD1315 디스플레이 사용
- 3개의 별도 컨덴서 슬롯
- 컨덴서당 최대 3A의 전류
- 10옴의 저항을 이용해서短絡시 브라운아웃을 방지
- USB-C 또는 배럴 ジャック(12-15V)로 전원 공급
- 작고 комп팩트한 사이즈(72x74x36mm)
### 사용 방법
1. 플래시 후 와이파이에 연결하면 디스플레이에 현재 시간이 표시됨
2. '셀렉트' 버튼(두 개의 버튼을 동시에 누름)을 눌러 설정으로 들어감
3. 왼쪽과 오른쪽 버튼을 이용해서 위아래로 이동하면서 알람 시간, 일정, 다음 컨덴서 슬롯 등을 설정할 수 있음
4. '어바웃' 페이지에서 시계의 IP를 확인할 수 있음
5. 웹 인터페이스에 접근할 때는 `http://<IP>/`를 입력해야 함(파이어폭스 gibi 브라우저는 HTTPS만 지원하기 때문에 주의할 것)
### 주의 사항
- 전압 조정기들이 사용 중에 약간 뜨겁게 될 수 있음(정상적임)
- 부카 컨버터 회로를 이용해서 전압을 낮추는 게 더 좋을 수도 있음(열 관리를 위해)
- 컨덴서를 선택할 때는 폭발音을 줄일 수 있는 압력 해제 슬롯이 없는 컨덴서를 선택해야 함
- 가능하면 가장 큰 컨덴서를 선택하고 전압 등급을 낮추는 게 좋음(안정성 향상)
