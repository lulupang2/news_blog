> **🔗 원문 보러가기:** [Show HN: Pu.sh – a full coding-agent harness in 400 lines of shell](https://pu.dev/)


 pu.sh는 400줄짜리 쉘 스크립트로 만든 코딩 에이전트임. 
 발음하면 '푸시'처럼 하라고 함. 
 > 작아서 포켓에도 들어갈 정도임. 
 사용 방법은 다음과 같음:
 ```bash
curl -sL pu.dev/pu.sh -o pu.sh && chmod +x pu.sh
./pu.sh
```
아, 그리고 npm이나 pip, 도커 필요없다고 함. 
그냥 커넥션 맺고 api 키받으면 됨.
