### 최적의 LLT 모델 찾기
whichllm 이라는 툴은 내 하드웨어에 설치할 수 있는 모델 중에서 벤치마크 결과가 가장 좋은 모델을 찾아 줌. 이 툴은 GPU/CPU/RAM을 자동으로 감지하고, HuggingFace에서 제공하는 모델 중에서 내 하드웨어에 맞는 모델을 순위대로 찾아 줌. 
 WHICHLLM의 주요 특징은 하드웨어를 감지하여 자동으로 최적의 LLT(Large Language Model)를 찾고 벤치마크 결과를 기반으로 한 순위 제공, Evidence-based ranking 등이 있음. 
 WHICHLLM이 제공하는 주요기능은 
1. 하드웨어 감지 
2. 벤치마크 결과 기반 순위 제공 
3. Evidence-based ranking 
4. 자동 LLT 모델 다운로드 및 설치 
5. 모델을 기반으로 한 챗봇 구동 기능 
6. JSON 형식의 출력 
 WHICHLLM의 작동 원리는 
1. 하드웨어 감지: NVIDIA, AMD, Apple Silicon, CPU-only 
2. 벤치마크 결과 기반 순위 제공: LiveBench, Artificial Analysis Index, Aider, multimodal/vision, Chatbot Arena ELO, Open LLM Leaderboard 
3. Evidence-based ranking: 직접적인 벤치마크 결과, 변형된 모델,基础 모델, 보간된 결과, 사용자 보고된 결과 등 
 WHICHLLM이 제공하는 명령어는 
1. `whichllm`: 내 하드웨어에 맞는 최적의 모델을 찾아줌 
2. `whichllm --gpu "<그푸 이름>
