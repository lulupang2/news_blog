# 리눅스용 윈도우 9x 서브시스템 떴다!
     리눅스에서 윈도우 9x 프로그램 돌릴 수 있게 해주는 서브시스템 나왔다는 소식이야. ㅇㄴ
     이거 때문에 리눅스 유저들이 또 윈도우로 갈아탈까 봐 걱정도 되지만 ㅋㅋㅋ, 그럴 건 없겠지.
     # 기능
     * 윈도우 9x 프로그램 실행
     * 리눅스와 윈도우 9x 간 파일 공유
     * 윈도우 9x 프로그램에서 리눅스 명령어 실행
     이게 가능한 이유는 윈도우 9x의 커널을 리눅스에서 에뮬레이션하기 때문이야. ㄹㅇ 신기하네.
     # 향후 계획
     아직은 실험판 단계지만, 향후에는 더 많은 기능을 추가할 계획이라고 해. 기대된다!
     아무튼, 이 새로운 서브시스템으로 리눅스 유저들에게 또 하나의 선택지가 생긴 건 좋은 소식이니까, 기대해 볼 만하다.
