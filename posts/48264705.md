### 레트로 미친듯게 구운 소브제트 프로그래밍 언어 인터프리터 'Rapira'가 떴다 ㄷㄷ

원래 [Rapira](https://begoon.github.io/rapira) 라는 언어는 80년대 초반에 구 소련에서 개발한 educational programming language야. 
이 언어는 [Агат](https://ko.wikipedia.org/wiki/Агат_(미니コンピューター))라는 마이크로컴퓨터에 사용됐고, 현재는 TypeScript와 Bun으로 인터프리터가 만들어져서 브라우저에서 테스트할 수 있어.
 
### Rapira 란 무엇인가 
Rapira는 소련의 [G. A. Zvenigorodsky](https://ko.wikipedia.org/wiki/Звенигородский_Георгий_Анатольевич)가 개발해서 [Школьница](https://ko.wikipedia.org/wiki/Школьница_(вычислительная_машина))라는 학교 컴퓨팅 시스템을 위해 만들었음. 
그당시에 Agat 마이크로컴퓨터에서 돌아가는 언어 였대. 
 
### 언어 스펙
* lexer, parser, tree-walking evaluator 
* 러시아어 키워드만 사용 
* 대소문자 구분 안 함 
* *...** 블록 コメント
* ; 문장 구분자
* 3가지 compound 타입 (튜플 <>, 셋 <* *>, 레코드 <¤ ¤>)
* 3가지 화살표 프로시저 파라미터 스키마 (이름 / 이름 => / <=> 이름)
* 트레일링 РЕЗ: 
* 함수 결과에 사용 

### CLI
* rapira FILE.rap : 프로그램 실행
* rapira : 멀티 라인 REPL 모드
* --svg PATH : 터틀 그래픽스 캡처 
 
### 웹 플레이그라운드
* HTML + CodeMirror 6 + Web Worker
* 라이트/다크 테마 
* 예제 선택 로딩 
* 빌드 bun build,(docs/ 경로에 저장, 깃헙 페이지에 호스팅)

### 스니펫 테스트 파이프라인 
* .rap 파일 diffed against sibling .expected.txt / .expected.svg 

### 설치 
rapira 라는 이름으로 npm에发布되어 있어서 
npx rapira FILE.rap 
npx rapira --svg out.svg 
npx rapira 
로 실행 가능. 
또는 글로벌로 설치할 수도 있음 
npm install -g rapira 

### 사용 예 
```bash
npx rapira examples/factorial.rap
npx rapira examples/turtle_square.rap --svg square.svg
```
