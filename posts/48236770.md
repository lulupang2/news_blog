### 개미지옥 탈출하는 코드 에디터 'Superset' 떴다!

요즘 클라우드 에이전트 같은 거쓰면 개미지옥 같은 코드 에디터 많이 만나봤을 거임. 그런데 'Superset'이라는 코드 에디터가 나오서 CLI 에이전트들을 하나로 묶어주고 편리하게 쓸 수 있게 해줌.

#### Superset의 주요 기능

- **병렬 실행**: 10개 이상의 CLI 에이전트를 동시에 실행할 수 있음. 
- **워크트리 분리**: 각 태스크마다 별도의 워크트리를 할당해서 에이전트들이 서로 간섭하지 않도록 설계됨.
- **에이전트 모니터링**: 에이전트들의 상태를 하나의 창에서 모니터링할 수 있음.
- **diff 뷰어**: 에이전트들의 변경된 내용을 즉시 확인하고 수정할 수 있음.
- **워크스페이스 프레스**: 환경setup, 의존성 설치 등 자동화할 수 있음.

#### 지원하는 에이전트들

- Amp Code
- Claude Code
- OpenAI Codex CLI
- Cursor Agent
- Gemini CLI
- GitHub Copilot
- OpenCode
- Pi

 등등... 거의 대부분의 CLI 에이전트들을 지원함.

#### 시작 방법

1. `git clone https://github.com/superset-sh/superset.git`로 소스코드 받고
2. 환경변수 설정하고
3. 의존성 설치하고
4. `bun run dev`하면 실행됨.
