# 소개
파라락스 효과에 대해 들어봤음? 이제는 CSS만으로도 구현할 수 있게 됐어. 이거 되게 중요한점이 몇개 있음. 먼저 이 효과를 쓰면 성능이 개선되고, 코드도 엄청 간결해지노.

# 코드
`.parallax` 클래스를 사용하면 되고, 이 클래스 안에 들어가는 코드가 이렇음:
```css
.parallax {
  view-timeline-name: --parallax-tl;
  view-timeline-axis: block;
  overflow: hidden;
  & > * {
    scale: calc(1 + var(--parallax-offset, 20) * 2 / 100);
    animation: parallax auto linear both;
    animation-timeline: --parallax-tl;
    animation-range: cover;
    will-change: translate;
  }
}
```
이거를 사용하면 알아서 파라락스 효과가 적용되는거임. 이 클래스를 사용하는 방법은 이렇음:
```html
<div class="parallax">
  <img src="..." />
</div>
```

# 동작 원리
이 효과는 어떻게 동작하는 걸까? `view-timeline-name` 속성이 핵심임. 이 속성을 사용하면 뷰의 진행 상황에 따라 애니메이션을 실행할 수 있게 됨. 뷰의 진행 상황은 0부터 100까지임. 0은 뷰가 시작하는 순간, 100은 뷰가 끝나는 순간임.

이 속성을 사용하면 애니메이션의 클록을 시간에서 뷰의 진행 상황으로 변경할 수 있게 됨. 따라서 애니메이션의 지속 시간도 변경될 수 있음.

# 개선점
이 코드를 사용하면 빈 공간이 생기는 문제를 피할 수 있음. 이는 자식 요소를 확대해서 빈 공간을 채울 수 있기 때문임.

이 효과를 사용하면 사용자에게 더 나은 경험을 제공할 수 있음. 하지만 일부 사용자는 애니메이션 효과가 싫어할 수 있으므로, `prefers-reduced-motion` 미디어 쿼리를 사용해서 애니메이션 효과를 끄는 것이 좋음.
